一括登録では、デバイス識別子を使用し、複数のAndroidデバイスを登録できます。 CSVファイルをアップロードし、デバイスを一括で追加できます。

**手順**

1. **[デバイス]** ページで **[一括登録]** タブをクリックします。 **[一括登録]** ページが表示されます。
2. **[追加]** をクリックします。
3. **[プロファイル名]** テキストフィールドで、プロファイルの名前を入力します。 任意で **[+説明を追加]** をクリックし、CSVファイルの説明を入力します。
4. **[CSVをアップロード]** セクションで、**[CSVテンプレートをダウンロード]** をクリックし、CSVテンプレートをダウンロードします。 既存の形式を使用して、ファイルを編集し、デバイスを追加できます。

一括登録 CSV では一度に最大 200,000 行まで許可されます。
5. CSVファイルを編集し保存したら、**[CSVをアップロード]** をクリックし、CSVファイルをアップロードします。 アップロードが成功したことを確認するメッセージが表示されます。

情報が不十分な行では、CSV アップロードが失敗する可能性があります。 各レコードには、少なくともシリアル番号と製造元情報、または IMEI 値が含まれている必要があります。

追加された CSV ファイルを削除するには、「マイナス」アイコンをクリックします。 別のCSVファイルを選択してアップロードするには、**[別のファイルを選択]** リンクをクリックします。
6. オプション：**[トークンなしの一括アップロードとカスタム属性の割り当て]** を選択すると、トークンの生成なしにすべてのタイプのデバイスを一括登録できます。 トークンなしの一括登録は、IMEIまたはシリアル番号とメーカー名の組み合わせ（カスタム属性の有無に関係なく）がアップロードされたCSVファイルに記載されている場合にも適用可能です。 しかし、デバイスの登録は、CSVファイルでアップロードされた属性値の正しさによって決まります。 以下の表は、一括登録に入力された属性値の組み合わせに基づく結果のシナリオを説明したものです。


|  **シナリオ**<br />   |  **入力した属性値**               |                 |                |  デバイス登録ステータス <br />             |
|-------------------|----------------------------|-----------------|----------------|---------------------------------|
||**IMEI**|**シリアル番号**|**メーカー**||
|1。|正|誤|誤|デバイスが登録される|
|2|誤|正|正|デバイスが登録される|
|3|誤|誤|正|デバイスが登録されていません|
|4|誤|正|誤|デバイスが登録されていません|

6. メーカー名は大文字と小文字を区別しません。
7. ユ**ーザーの選択 **フィールドでは、オプションでユーザーを選択できます。
登録トークンは、「登録トークン」列に表示されます。 登録トークンを更新するには、「 **更新**」をクリックします。
トークンの有効期限は、「 **トークン有効期限** 」セクションに表示されます。 トークン有効期限を延長するには、**[延長]** をクリックします。 「 **延長最大** 」フィールドに、トークンを延長する日数を入力します。
指定する日数は 7 ～ 999 の範囲内である必要があります。 ****[トークンなしの一括アップロードとカスタム属性の割り当て] のオプションを選択した場合、このページは表示されません。
8. **[完了]** をクリックします。


アップロード後、アップロードしたCSVファイルの以下の詳細が **[プロファイルの一括登録]** ページの表に表示されます。

|  設定               |  説明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|**プロファイル名**|プロファイルの名前。|
|**説明**|プロファイルの説明。|
|**前回の変更**|CSVファイルの前回の変更日。|
|**タイプ**|プロフィールに関する情報。 デフォルトでは、「Self Maintenance」に設定されています。|
|**デバイスの数**|一括登録対象のデバイスの数。|
|**関連ユーザ**|関連ユーザの名前。 ユーザーを変更するには、「 **ユーザーの変更** 」リンクをクリックします。|
|**アクション**|以下のアクションのいずれかを実行できます。<br />**既存のインベントリをダウンロード** - このボタンをクリックすると、プロファイルで使用可能なすべてのデバイスの詳細情報がダウンロードされます。<br />**表示** - このリンクをクリックすると、一括登録のためにアップロードしたプロファイルの詳細が表示されます。<br />**編集** - プロファイル詳細情報を編集します。 1つのデバイスオプションが選択されているときにのみ使用できます。 <br />**削除** - このリンクをクリックすると、アップロードしたプロファイルが削除されます。 確認ウィンドウで、**[はい]** をクリックし、アップロードしたプロファイルの削除を確定します。|

CSVファイルのアップロード中に生成されたトークンを登録に使用する必要があります。 誤ったトークンを入力すると、通常のIRegフローにリダイレクトされ、IDとパスワードの入力が必要となります。

### アクション

[プロファイル詳細の表示] セクションで一括登録プロファイルを表示するときに、[プロファイル詳細の表示] ページにある [アクション] タブから他のタスクを実行できます。
- **さらにデバイスを追加** - このオプションを使用して、その他のデバイスをプロファイルに追加できます。 **[IMEI番号]** か **[シリアル番号]** のいずれか、またはその両方と、**[メーカー]**、そして任意で **[カスタム属性]** 情報を入力した後、**[保存]** をクリックする必要があります。
- **構成の修正** - このオプションを使用して、既存の構成を修正できます。 **Ivanti 固有の鍵**を追加し、**[定義済みの Android System Extras]** または **[カスタム Android システム キー]** に変更して、**[更新]** をクリックします。
- **QR コードを生成** - このオプションを使用して、プロファイルに一括登録で使用する QR コードを生成します。
- **トークンを更新** - トークンを更新するか、トークンの有効期間を延長します。 有効期間は、7～999日の範囲で設定することも、**[無期限]** に設定することもできます。 
- **削除** - 選択したプロファイルからデバイスを削除します。 デバイスを選択し、[削除] ボタンをクリックすると、確認ポップアップが画面に表示されます。 **[削除]** をクリックします。
- **編集** - 選択したプロファイルからデバイスを編集します。 デバイスを選択して、**[編集]** ボタンをクリックする必要があります。

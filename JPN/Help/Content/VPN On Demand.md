**対象：**iOSデバイス

VPNオンデマンド構成では、ドメイン、ホスト名などに基づきVPNサーバーへのアクセスを設定します。

## VPNオンデマンド設定

| **設定** | 対処方法 |
 | ---------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | 名前 | この構成を識別する名前を入力します。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      この構成の目的を明示する説明を入力します。|

 | 接続タイプ | 構成する VPN のタイプを選択します。<br />残りの設定は、この選択に依存します。オンデマンドVPNを確立するドメインおよびホスト名にこの構成を使用する場合に選択します。iOS規則を有効化<br />(オンデマンド VPN を有効にするが選択されている場合に適用) | iOS および macOS では、以下を設定できます。<br />trueと判断されたネットワークへの接続の許可または禁止、および許可または無視を設定するネットワークルール。
* trueと判断されたネットワークへの接続を必要時に許可する、あるいはすべて許可しない接続ルール。<br />ネットワークルールの場合、次のタイプのパラメーターを指定できます。<br />DNSのドメインが一致
* DNSのサーバーアドレスが一致
* SSIDが一致
* URL文字列のプローブ
* インターフェイスのタイプが一致<br />接続ルールの場合、次のタイプのパラメーターを指定できます。<br />DNSのドメインが一致
* DNSのサーバーアドレスが一致
* SSIDが一致
* URL文字列のプローブ
* インターフェイスのタイプが一致
* ドメイン名
* DNSサーバー
* URL プローブ |

 | プロバイダー タイプ (iOS 9) | 次のトンネル プロバイダーのいずれかを選択します。<br />アプリプロキシ - アプリレイヤでトラフィックをトンネリングします。 [****](https://developer.apple.com/documentation/networkextension/app_proxy_provider)アプリプロキシプロバイダーの詳細は、Appleドキュメンテーションをご覧ください。
* パケットトンネル - IPレイヤでトラフィックをトンネリングします。 [****](https://developer.apple.com/documentation/networkextension/packet_tunnel_provider)パケットトンネルプロバイダーの詳細は、Appleドキュメンテーションをご覧ください。有効にすると、VPNのデフォルト以外のルートによって、ローカルに定義されているルートが上書きされます。<br />[すべてのネットワークを含める] が有効になっている場合、[ルートを強制] 設定は無視されます。
* iOS 14.2以降で使用可能です。[すべてのネットワークを含める] とともに有効にすると、ローカルネットワークのすべてのトラフィックの経路がVPNの外側になります。|

 | IncludeAllNetworks | 有効にすると、すべてのトラフィックが VPN 経由でルーティングされます。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |


プロトコルと設定は以下のとおりです。

### [****](#)IPsec（Cisco）

| **設定** | 対処方法 |
 | ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | サーバー | VPN サーバーの IP アドレスまたはホスト名を入力します。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            接続の認証に使用するユーザアカウントを入力します。**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

 | マシン認証 | 証明書認証のみがサポートされています。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    認証情報：使用するID証明書を選択します。 |

 | ユーザー PIN を含める | ユーザーに PIN の入力を求める場合に選択します。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             プロキシの設定<br />********[手動] または [自動] を選択し、プロキシを構成します。<br />**[手動] を選択すると、**以下のフィールドが追加で利用できるようになります。<br />* **[サーバーおよびポート]**：プロキシサーバーのネットワークアドレスとポート番号を入力します。*
* **[認証]**：プロキシへの接続に必要な場合は、有効なユーザ名を入力します。*
* **[パスワード]**：プロキシへの接続に必要な場合は、有効なパスワード名を入力します。<br />**[自動] を選択すると、**以下のフィールドが追加で利用できるようになります。<br />****プロキシサーバーURL：プロキシの完全修飾URLを入力します。|

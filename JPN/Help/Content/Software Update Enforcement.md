対象：
- iOS 17.0以降、がサポートする最新版まで
- macOS 14.0以降、がサポートする最新版まで


**手順**

1. **[構成]** > **[+追加]** を開きます。
2. **[ソフトウェア更新の実施]** 構成を検索して選択します。
3. 次の表に示すように、**[ソフトウェア更新の実施]** 設定を構成します。


| **設定** | **説明** |
 | -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | OS バージョンのアップグレードを選択します | 次のいずれかのオプションを選択します。<br />****最新バージョン：特定の時間帯に自動的に最新バージョンに更新します。
* **特定のバージョン**：特定の時間帯に特定のOSバージョンおよびビルドバージョンに更新します。|


4. **[次へ]** をクリックして配布設定を行います。
5. 配布オプションを1つ選択して、**[ソフトウェア更新の実施]** を設定します。 配布オプションの構成についての詳細は、構成の操作を参照してください。
6. **[完了]** をクリックします。

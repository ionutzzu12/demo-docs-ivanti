管理者は、Windows Update管理を使用して更新するWindowsデバイスによって報告された更新を、表示して承認できます。 この機能では、不要またはテスト済みでない更新がデバイスにインストールされるのを防止します。

******** 更新管理機能を使用するには、デバイスの [ソフトウェア更新] 構成で [更新の承認を要求] オプションを有効化しておく必要があります。 この構成をデバイスに適用して初めて、インストール可能な更新をデバイスが報告し、承認を待つようになります。

## 更新の管理

1. **[管理] > [Windows更新]** を開きます。 The following details of the updates are displayed in the page.
   **Creation Date**: The date when the update was created.
   **Title**: Describes the type of update along with the Knowledge Base article number.
   When clicked on the update, the description is displayed.
   **Classification**: Classifies the type of update into one of the following categories:
   - 重大な更新
   - 定義の更新
   - ドライバーの更新
   - セキュリティの更新
   - 更新
   - アップグレード
   - 更新のロールアップ

2. **配布**：更新の配布対象。 たとえば、アップデートがすべてのデバイスに配布されている場合は、「 **すべて** 」と表示されます。
更新が特定の数のグループに配布された場合、配布数が表示されます。 たとえば、3つのグループにだけ配布する場合は「3」と表示します。
3. また、更新が必要なデバイスに配布されたかどうかを表示できます。 次の列に表示されている数字は、さまざまなカテゴリの更新に該当するデバイスの数を示します。  
   - 資格があるデバイス
   - インストールされたデバイス
   - 失敗したデバイス
   - 再起動が保留中のデバイス

4. これらの番号のいずれかをクリックすると、デバイス ページのフィルターされたビューに移動し、更新のステータスを確認して必要なアクションを実行できます。

更新を確認し、更新のチェックボックスをクリックして、デバイスに配布する更新を選択します。
5. **[アクション]** で **[配布を設定]** をクリックします。
6. W**indows Update の配布 **ウィンドウで、次のいずれかの配布オプションを選択します。
   す**べてのデバイス:** すべてのデバイスに更新を配布します。
   デ**バイスなし:** デバイスに配布される更新を保留します
   カ**スタム:** 指定されたデバイス グループの更新を配布します。
7. **[保存]** をクリックします。


## 更新の検索とフィルタリング

更新は、以下の基準に基づいて検索およびフィルタリングが可能です。
- サポート技術情報ID
- 配布の構成


サポート技術情報IDに基づくフィルタリング：

1. **[Windows更新管理]** ページで、クイック検索フィールドにナレッジベースIDを入力します（[検索] フィールドには数字のみ）。 

   例：KB4056892の場合は、4056892と入力。 検索条件に一致する更新がページに表示されます。
更新に関する追加情報は、[ **サポート** ] および [ **詳細情報** ] リンクをクリックして検索できます。 **[サポート]** は、更新のサポート情報を提供するMicrosoftのWebページにリンクしています。**[詳細]** は、サポート技術情報など更新に関する詳細情報を公開するMicrosoftのWebページにリンクしています。


配布の構成に基づくフィルタリング

**[Windows更新管理]** ページで、構成済みの配布に基づいて、次のフィルタリングオプションを選択します。 
- **すべて**：すべての更新を表示します。
- **構成済み**：デバイスに配布された更新のリストを表示します。
- **未構成**: 配布が指定されていない更新プログラムの一覧を表示します。
構成済みフィルターと未構成フィルターは、実行された配布に基づいており、配布は **なし**になることもできます。


## デバイスの更新表示

各デバイスの詳細な更新情報は以下の手順で確認します。

1. **[デバイス] > [デバイス]** に進みます。
2. デバイス名をクリックして詳細ページを表示します。
3. **[更新]** タブを開きます。 デバイスの更新のうち、保留中 (管理者によって承認されたが、デバイスにインストールされたことが報告されていない更新)、失敗した更新、およびインストール済みの更新が表示されます。

ダッシュボードの通知ページでは、利用可能な新しい Windows 更新プログラムに関する通知も確認できます。 通知には、通知の作成日、取得可能な通知の数、通知の目的が含まれます。 Windows更新通知は、管理ポータルの右上角からも閲覧できます。

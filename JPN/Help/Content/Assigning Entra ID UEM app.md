ユーザ設定の指定を完了するには：

1. Entra ID Windows 10統合エンドポイント管理設定の手順2で作成したMobileIron UEMアプリをクリックします。
2. [MDMユーザスコープ] でカスタマイズしたユーザグループに指定するか、**[すべて]** を選択します。

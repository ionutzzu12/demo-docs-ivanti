 ディスク管理構成の設定により、外部デバイスおよびネットワークストレージデバイス内のデータを効率的に管理するためのポリシーを定義します。

対象：macOS 15.0監視対象以降、がサポートする最新版まで。

**手順**

1. **[構成]** > **[+追加]** を開きます。
2. **[ディスク管理]** 構成を検索して選択します。
3. 次の表に示すように、**[ディスク管理]** 設定を構成します。


| **設定** | **説明** |
 | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | 名前 | この構成を識別する名前を入力します。                                                                                                                                                                                  この構成の目的を明示する説明を入力します。|
 | **ディスクの管理** | |
 | 外部ストレージ | ドロップダウン リストから次のいずれかの値を選択します。<br />****許可：ストレージの読み取りと書き込みを許可します。
* **読み取り専用**：ストレージの読み取りを許可します。
* **禁止**：ストレージへのアクセスを許可しません。ドロップダウンリストから次の値のうち1つを選択します。<br />****許可：ストレージの読み取りと書き込みを許可します。
* **読み取り専用**：ストレージの読み取りを許可します。
* **禁止**：ストレージへのアクセスを許可しません。|


4. **[次へ]** をクリックして配布設定を行います。
5. 配布オプションを1つ選択して **[ディスク管理]** を設定します。 配布オプションの構成についての詳細は、構成の操作を参照してください。
6. **[完了]** をクリックします。

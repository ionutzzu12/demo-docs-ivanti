**ライセンス：**Gold

FileVault 2は、ボリュームのコンテンツに完全なXTS-AES 128ディスク暗号化を実行できます。

FileVault 2を有効化した場合、以下の設定が構成可能となります。

| カテゴリ | 設定 |

 | ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

 | FileVault ユーザー設定 | - 指定されたユーザーがログアウトするまで FileVault の有効化を延期します
  - FileVault有効化のプロンプトを常にユーザに表示
  - ユーザがFileVault有効化を回避できる最大回数
- ユーザーのログアウト時に FileVault の有効化を要求しない |

 | 出力パス | 回復キーとコンピューター情報 plist が保存される場所へのパスを入力します。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |

 | 個人用回復キー | * 個人用回復キーを作成する
* FileVault が有効になった後、ユーザーに個人用回復キーを表示します。
このオプションは、「 **個人用回復キーを作成する** 」が有効になっている場合にのみ表示されます。 デフォルトでは無効化されています。
* 機関リカバリーキーの有効化：キーチェーンの使用 - このペイロードで証明書情報が提供されない場合、/Library/Keychains/FileVaultMaster.keychainで作成済みのキーチェーンを使用します。

ユーザ ロールによって、Ivanti Neurons for MDM で表示できるページや、実行できる操作が決まります。 割り当てることができる役割と、その内容を次の表にまとめます。

| **役割** | **説明** | **スペース固有** |
 | ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------ |
 | システム管理 | 管理者が MDM 証明書、アプリ カタログ設定などのテナント レベルの設定を管理できるようにします。                                                                                                                                                   管理者が、MDM認証やアプリカタログ設定などのテナントレベルの設定を閲覧できます。管理者が、ユーザの追加や削除、役割の指定、ユーザグループへのユーザ追加を実行できます。管理者が、ユーザおよびユーザグループのほか、アプリカタログとコンテンツカタログを閲覧できます。管理者が、デバイスグループ、構成、ポリシーを管理し、すべてのデバイスアクションを実行できます。管理者が、デバイスグループ、構成、ポリシーを閲覧できます。管理者が、アプリやコンテンツの追加、配布、削除を実行できます。| はい |

 | アプリ コンテンツ読み取り専用 | AppConnect タスクを含む、ユーザー、アプリ、コンテンツのデータを表示 | はい |

 | デバイスアクション | 管理者が次のようなデバイスアクションを開始できるようにします。<br />強制チェックイン 
* ロック
* ロックの解除
* メッセージを送信
* 撤去
* ワイプ<br />[デバイスアクション] を選択する前に、[デバイス読み取り専用] を選択してください。 選択しない場合、ユーザに必要な許可が与えられません。| はい |

 | LDAP ユーザーのインポートと招待 | 管理者が LDAP ユーザーを登録し、デバイスを登録するための招待状を送信できるようにします | いいえ |

 | Cisco ISE 操作 | 管理者が Cisco ISE 統合に必要な API を呼び出すことを許可します。                                                                                                                                                                                       管理者がさまざまな管理業務のスケジュールタスクを作成および管理できます。| いいえ |

 | 共通プラットフォーム サービス (CPS) | 管理者が共通プラットフォーム サービスを使用できるようにします。                                                                                                                                                                                                           | いいえ |

 | ユーザーへの影響の少ない移行管理 | 管理者がユーザーへの影響の少ない移行設定を管理できるようにします。                                                                                                                                                                                              管理者がカスタムデバイス登録を使用してデバイスを登録できます。Office 365アプリを保護するMicrosoft GraphのAPI設定を管理者が編集できます。Office 365アプリを保護するMicrosoft GraphのAPI設定を管理者が表示できます。デバイスへのワイプコマンド送信、または発行したワイプコマンドの実行前の取り消しを管理者が実行できます。| いいえ |

 | アクセス統合の管理 | 管理者がアクセス統合を管理できるようにします。                                                                                                                                                                                                              | いいえ |


詳細については、[**ロールの割り当て**](\<./Assigning Roles to Users.md>) を参照してください。

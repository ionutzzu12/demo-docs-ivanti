ユーザのスタートメニューのレイアウトを定義して、使用可能なアプリケーションを定義し、不要なアプリケーションを削除できます。 Windows 10および11バージョンでは、スタートメニューのレイアウトが異なるため、別の機能がサポートされます。 

スタートメニューとタスクバーの構成を設定するには:

1. **[構成] > [+追加]** を開きます。
2. **[Windowsスタートメニュー＆タスクバー]** 構成を選択します。
3. 構成の名前を入力します。
4. Windows 固有のバージョンを選択します。
5. ********[構成設定] セクションで、以下の設定を構成します。


|  設定                    |  操作内容                                                                                   |
|------------------------|-----------------------------------------------------------------------------------------|
|**[名前]**|この構成を識別する名前を入力します。|
|**説明**|この構成の目的を明示する説明を入力します。|
|**ゴールデンデバイスを選択**|アプリをスタートメニューとタスクバーにプロビジョニングする Windows 10デバイスを選択します。 |

6. ****[デバイスからスタートメニューレイアウトをフェッチ] をクリックし、以下のオプションを構成します。


| 設定 | 対処方法 |
 | -------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | **スタート メニューとタスク バーのレイアウト**<br />必要に応じて以下のオプションを選択し、アプリのリストを非表示にします。<br />なし
* すべてのアプリリストを非表示
* すべてのアプリリストを非表示にし、設定アプリで「スタートメニューにアプリを表示」を無効化
* 設定アプリですべてのアプリ リストを非表示にし、すべてのアプリ ボタンを削除し、「スタート メニューにアプリ リストを表示する」を無効にします |
 | **スタート メニュー レイアウト** | |
 | **スタート メニューをカスタマイズします** | スタート メニューとレイアウト パラメータをカスタマイズするには、[ **はい** ] をクリックします。                                                                                                                                                                                                                                  |
 | **タスクバー** | |
 | **タスク バーをカスタマイズします** | タスク バーをカスタマイズするには、[ **はい** ] をクリックします |
 | **カスタム タスクバー ショートカットを追加する前に、既存の固定タスクバー ショートカットを削除します** | カスタム タスクバーを追加する前に、既存の固定タスクバー ショートカットを削除するには、[ **はい** ] をクリックします。                                                                                                                                                                                                |
 | **アプリ タイプ** | アプリのタイプを指定します。                                                                                                                                                                                                                                                                    ****アプリのIDを指定します。********[復元] リンクをクリックすると、元のゴールデンデバイスからタスクバーが復元されます。<br />行の矢印アイコンをドラッグし、特定の行の位置を上下に移動します。<br />ロウを削除する場合は削除アイコンをクリックします。<br />新しいロウを追加するには **[新規追加]** ボタンをクリックします。|


8. ********[構成設定] セクションで、以下の設定を構成します。


| 設定 | 対処方法 |
 | --------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | **名前** | この構成を識別する名前を入力します。                                                                                                                                                                                                                                                      ****この構成の目的を明示する説明を入力します。****アプリをスタートメニューの [ピン留め済み] セクションに設定する Windows 11デバイスを選択します。 |
 | **スタート メニューとタスク バーのレイアウト** | |
 | **スタート メニューとタスク バーのレイアウト** | タスク バーに検索オプションを表示するには、次のいずれかのオプションを選択します。<br />****非表示
* **検索アイコン**
* **検索アイコンとラベル**
* ********タスクビューを非表示にするには、[タスクビューを非表示] チェックボックスを選択します。****次のいずれかのオプションを選択して、スタートメニューに最もよく利用されているアプリを構成します。<br />****最もよく利用されているアプリのリストの [スタート] 内での可視性を強制しない
* **最もよく利用されているアプリのリストを [スタート] に強制的に表示する**
* **最もよく利用されているアプリのリストを [スタート] 内で強制的に非表示にする******[タスクバーへのピン留めを許可] チェックボックスはデフォルトで選択されています。****すべての推奨セクションを非表示にするには、[推奨セクションを非表示] チェックボックスを選択します。<br />ベンダの問題により、**[推奨セクションを非表示]** チェックボックスを選択しても、ユーザがログオフまたはログインした後でさえ、デバイス上での目に見える影響はありません。 現在のところ [推奨] セクションを非表示にすることはできません。********次のいずれかのオプションを選択して、アプリのタイプを指定します。<br />****アプリユーザモデルID
* **デスクトップ アプリケーション リンク パス** |
 | **App** | アプリの ID またはパスを指定します。                                                                                                                                                                                                                                                                  ********[復元] リンクをクリックすると、元のゴールデンデバイスからタスクバーが復元されます。<br />行の矢印アイコンをドラッグし、特定の行の位置を上下に移動します。<br />ロウを削除する場合は削除アイコンをクリックします。<br />新しいロウを追加するには **[新規追加]** ボタンをクリックします。|


9. ****[デバイスからスタートメニューレイアウトをフェッチ] をクリックします。
10. ゴールデン デバイスから設定を取得した後、スタート メニューの PINNED セクションで構成されているすべてのアプリが管理者が確認できるように表示されます。 **次へ** をクリックし、該当するすべてのデバイスおよびユーザー グループに設定を配布します。
ベンダーの問題により、構成が配布されていない場合、ピン留めされたアプリは元の構成のままになります。 ただし、新しい設定が配布されると、前のレイアウトは上書きされ、新しいレイアウトが適用されます。

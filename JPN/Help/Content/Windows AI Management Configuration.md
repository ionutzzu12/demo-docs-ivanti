 では、管理者がWindows AIの設定と構成を管理できます。 Windows AI管理は、パワフルなAIアシスタントの支援により、創造性と生産性を強化します。

**対象**：Windows 10+

**手順**

1. **[構成]** > **[+追加]** を開きます。
2. **[Windows AI管理]** 構成を検索して選択します。
3. 構成の **[名前]** と **[説明]** を入力します。
4. Windows Copilot を無効にしたい場合は、[ **構成セットアップ** ] セクションで、[ **Windows Copilot を無効にする** ] チェックボックスをオンにします。 **Windows Copilot を無効にする** チェックボックスを選択しない場合、構成はデバイス上の以前の構成選択を上書きし、Windows AI 管理を有効にします。
5. **[次へ]** をクリックして配布設定を行います。
6. 配布オプションを1つ選択して **[Windows AI管理]** を設定します。 配布オプションの構成についての詳細は、構成の操作を参照してください。
7. **[完了]** をクリックします。

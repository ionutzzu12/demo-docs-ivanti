## [****](#)一般的な機能と機能拡張

- **Azure の強化されたステータス更新**: Intune デバイスのコンプライアンス ステータスでは、Azure ポータルで、廃止済みまたはワイプ済みの状態のデバイスに対してコンプライアンス ステータスが「いいえ」と表示されるようになりました。
- **Microsoft Intune デバイス コンプライアンス統合の移行をサポート**: Ivanti EPMM の移行の一環として、iOS デバイスと Android デバイスは、への移行後にデバイスのコンプライアンス状態を Microsoft Azure に報告するようになります。 この機能を有効にするには、Ivanti サポートにお問い合わせください。 詳細については、「 *Ivanti EPMM から Ivanti Neurons for MDM への移行ポータル ガイド*」を参照してください。
- **ユーザーおよびユーザー グループの SCIM API の更新**: 管理者は externalDs を使用して SCIM ユーザーおよびユーザー グループを検索できるようになりました。 詳細については、「SCIM を使用したユーザープロビジョニング」を参照してください。
- **強化されたユーザー プロビジョニング**: プロビジョニング プロセス中に、管理者は **カスタム属性** オプションを使用して、カスタム属性を管理対象 Apple ID にマッピングできるようになりました。 詳細については、「Entra ID IdP のユーザー プロビジョニング」を参照してください。


## [****](#)Androidの機能

- ********Threadネットワークのサポート：が仕事用管理対象デバイスモードおよび管理対象デバイスモードでThreadネットワークをサポートするようになりました。 詳細は、ロックダウン＆キオスク：Android Enterpriseを参照してください。

- **ユーザーが社内アプリをアンインストールできないようにするサポート**: オプション **は、ユーザーによるアプリのアンインストールをブロックします。** 社内アプリでも利用できるようになりました。 詳細については、「Android の管理対象構成」を参照してください。


## [****](#)iOS、macOS、watchOS、visionOS、tvOSの機能

- **Apple アクセス管理のサポート**: 監視対象および管理対象デバイスに対して **Apple アクセス管理** をサポートするようになりました。 詳細については、「Apple アクセス管理」を参照してください。
- **iOS 17 デバイスの追加スキップ キー**: **アクション ボタン** と **ロックダウン モードの有効化** スキップ キーが iOS 17 デバイスでサポートされるようになりました。 詳細については、デバイス登録をご参照ください。
- **デバイス登録プロファイルの更新**: 管理者は、監査試験ページからデバイス登録プロファイルに加えられた変更を表示できるようになりました。 詳細については、デバイス登録をご参照ください。
- **iOS 26 デバイス用の追加のスキップ キー**: **スキップ マルチタスク ペイン**、 **OS ショーケース**、および **追加のプライバシー設定** スキップ キーが iOS 26 デバイスでサポートされるようになりました。 詳細については、デバイス登録をご参照ください。
- **Safari 設定**: 管理者は、Safari 設定構成を使用して、Safari の新しい設定のスタート ページを設定できるようになりました。 詳細については、「Safari 拡張機能と設定の構成」を参照してください。
- **Safari の制限**: 管理者は、Safari の制限を使用して履歴を消去し、プライベートに閲覧できるようになりました。 詳細については、iOSの制約を参照してください。
- **オーディオ アクセサリ設定構成のサポート**: 管理者が **オーディオ アクセサリ設定** 構成を作成し、オーディオ アクセサリのペアリングとペアリング解除を一時的に管理できるようにします。 詳細については、「オーディオアクセサリの設定」を参照してください。
- **カメラの制限を許可する**: この設定により、アプリがカメラに自動的にアクセスすることが制限されます。 iOS 26+にのみ適用されます。詳細については、iOSの制約を参照してください。
- **FaceTime および RCS メッセージングのアップデート**: iOS 26 および iPadOS 26 デバイスを搭載したセルラー iPad モデルでは、iMessage、FaceTime、および RCS メッセージング機能に制限が適用されました。 詳細については、iOSの制約を参照してください。

## [****](#)Mobile Threat Defenseの機能

Mobile Threat Defense（MTD）は、デバイス、ネットワーク、アプリケーションに影響を与えるモバイル脅威や脆弱性からマネージドデバイスを保護します。 [****](https://www.ivanti.com/support/product-documentation)最新リリースに該当するMTD関連の機能については、Ivantiの製品ドキュメンテーションページの [MOBILE THREAT DEFENSE] セクションから、お使いのプラットフォームに対応するMobile Threat Defenseソリューションガイドを参照してください。

MTDガイドの各バージョンには、サーバー環境とクライアント環境の両方で完全にテストされ、提供されているMobile Threat Defense機能がすべて含まれます。 サーバーリリースとクライアントリリースの差により、新しいバージョンのMTDガイドは、シリーズの最終リリースで機能が完全に動作するようになってから公開されます。

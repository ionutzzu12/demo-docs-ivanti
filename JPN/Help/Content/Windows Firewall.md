Windowsファイアウォール構成では、Windowsファイアウォールのプロファイル設定のほか、デバイスに適用したいカスタムルール群を構成できます。 この構成は、ドメイン外のデバイスの管理に使用し、企業ネットワークに接続する全システムにかかるネットワークセキュリティ脅威のリスクを軽減します。

## Windowsファイアウォールの構成

**手順**

1. **[構成] > [+追加]** を開きます。
2. ****[ファイアウォール] 構成を選択します。
3. **Windows**アイコンをクリックします。
4. 構成の名前を入力します。
5.  ファイアウォール構成の説明を入力します。 
6. [構成設定] セクションで、次の表に記載されているように残りの設定を指定します。


|  設定                   |  操作内容                                                                                                                                                      |
|-----------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
|**プロファイル**||
|有効化|プロファイルを有効化するにはスイッチをONにスライドさせます。|
|タイプ|プロファイルのタイプを表示します。<br />例：ドメイン|
|デフォルトのインバウンドアクション|インバウンドトラフィックに実行するデフォルトアクションを選択します。<br />**許可**：トラフィックを許可<br />**ブロック**：トラフィックをブロック|
|デフォルトのアウトバウンドアクション|アウトバウンドトラフィックに実行するデフォルトアクションを選択します。<br />**許可**：トラフィックを許可<br />**ブロック**：トラフィックをブロック|

7. **** ルールを追加するには、[+追加] をクリックして以下の設定を構成します。 


| 設定 | 対処方法 |
 | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | **ルール** | |
 | ON | スイッチをスライドしてプロファイルを有効にします。                                                                                                                                                                                   このルールを識別する名前を入力します。このルールの目的を明示する説明を入力します。ルールを適用するトラフィックの方向を選択します。<br />****イン：インバウンドトラフィック
* **アウト**：アウトバウンドトラフィック
* **両方**：両方向実行するアクションを選択します。<br />****許可：トラフィックを許可
* **ブロック**：トラフィックをブロックルールを適用するプロファイルを選択します。<br />****すべて
* **ドメイン**
* **非公開**
* ****パッケージファミリー名（PEN）または実行可能アプリへのフルパスを入力します。ルールを適用するプロトコルを以下から選択します。<br />****TCP
* **UDP**
* **ICMP** |
 | ローカル アドレス範囲 | ローカル IPv4/IPv6 アドレス範囲またはサブネット マスクを入力します。                                                                                                                                                                  ローカルポートまたはポート範囲をカンマ区切りで入力します。<br />例：20,50,100-120リモートIPv4/IPv6アドレス範囲またはサブネットマスクを入力します。ローカルポートまたはポート範囲をカンマ区切りで入力します。<br />例：20,50,100-120以下からインターフェイスのタイプを選択します。<br />****すべて
* **リモートアクセス**
* **ワイヤレス**
* **LAN**
* **モバイルブロードバンド**<br />**** インターフェイスのタイプが選択されていない場合、デフォルトオプションの [すべて] が適用されます。|


8. **[次へ]** をクリックします。
9. 以下の配布オプションから1つ選択します。
   - すべてのデバイス
   - デバイスなし（デフォルト）
   - カスタム

10. **[完了]** をクリックします。

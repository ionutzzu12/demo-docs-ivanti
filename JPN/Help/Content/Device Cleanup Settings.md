デバイスクリーンアップにより、使用されていないデバイスのデバイスライフサイクルを自動化します。 指定した日数の間に接触がなかったデバイスを撤去できます。 指定した日数の間、撤去されていたデバイスを、削除できます。 [監査証跡] ページには、[デバイスを撤去]、[デバイスを削除]、および [ワイプ済みデバイスを削除] の設定がキャプチャされます。

次の表に、デバイスクリーンアップ設定をサポートしているAndroidデバイスについての情報を示します。 

|  モード                                                         |  非準拠の場合は撤去                |  撤去済みデバイスの削除 |  強制撤去または撤去保留中デバイスの撤去         |  ワイプ済みデバイスの削除|  ワイプ保留中デバイスの削除    |
|--------------------------------------------------------------|---------------------------|--------------|------------------------------|------------|-------------------|
|Android仕事用プロファイル（プロファイル所有者）|**はい**|**はい**|**はい**|**いいえ**|**いいえ**|
| デバイス管理者/ 会社所有デバイス上の仕事用プロファイル|**はい**|**はい**|**はい**|**はい**|**はい**|
|仕事用マネージドデバイス / 仕事用プロファイルを持つAndroidマネージドデバイス |**いいえ**|**いいえ**|**いいえ**|**はい**|**はい**|

**前提条件**

この設定にアクセスするには、システム管理役割の権限が必要です。

## デバイスを撤去

**手順**

**[管理]** > **[システム]** > **[デバイスクリーンアップ]** を開きます。 [デバイスクリーンアップ] ページが開きます。

**[デバイスを撤去]** を選択します。 

「**デバイスを撤去**」表を使用して、詳細を指定します。

**[チェックインしていないデバイスのリストを表示]** をクリックします。 指定した日数の間にチェックインしていないデバイスのリストが表示されます。

**[デバイスを今すぐ撤去]** をクリックするか、またはデバイスの撤去をスケジュールできます。

指定したデバイスが管理ポータルによって撤去されます。

**[保存]** をクリックして設定を保存します。

（任意）値を更新した場合、**[リセット]** をクリックすると、設定を初期設定に戻せます。

### デバイスを撤去

| フィールド | 説明 |
 | ----------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | **(日数) を超えてチェックインされていないデバイスを廃止します** | 日数: デフォルトは 30 日で、最大許容日数は 365 日です。                                                                                                                                                                                                                                |
 | **各セッションで廃止するデバイスの最大数** | 100、500、または 1000 を選択します (デフォルト - 100)。                                                                                                                                                                                                                                                            |
 | **スケジュールに従ってデバイスを自動的に廃止する** | 事前に設定されたスケジュールに基づいてデバイスを廃止するには、チェック ボックスをオンにします。                                                                                                                                                                                                                                   |
 | **廃止スケジュールの構成** | 廃止の頻度を設定するには、次のいずれかのオプションを選択します。<br />****毎日 - デバイスを毎日撤去する場合に設定します。
* **毎週 - **撤去を予定する曜日を指定します。
* **毎月 - **毎月の第1日目にデバイスを撤去する場合に設定します。|


## 撤去済みデバイスの削除

**手順**

1. **[管理]** > **[システム]** > **[デバイスクリーンアップ]** を開きます。 [デバイスクリーンアップ] ページが開きます。
2. **[撤去済みデバイスを削除]** を選択します。
3. 「**撤去済みデバイスを削除**」表を使用して、詳細を指定します。
4. **[撤去済みデバイスのリストを表示]** をクリックします。 指定した日数の間に撤去されていたデバイスのリストが表示されます。
5. **[撤去済みデバイスを今すぐ削除]** をクリックするか、またはデバイスの削除をスケジュールできます。
6. 指定したデバイスが管理ポータルによって削除されます。
7. **[保存]** をクリックして設定を保存します。
8. （任意）値を更新した場合、**[リセット]** をクリックすると、設定を初期設定に戻せます。


### 撤去済みデバイスの削除

| フィールド | 説明 |
 | ---------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | **(日数) 以上使用されていないデバイスを削除します** | 日数: デフォルトは 30 日で、最大許容日数は 365 日です。                                                                                                                                                                                                                               |
 | **各セッションで削除する廃止デバイスの最大数** | 100、500、または 1000 を選択します (デフォルト - 100) |
 | **スケジュールに従って廃止されたデバイスを自動的に削除する** | 事前に設定されたスケジュールに基づいてデバイスを廃止するには、チェックボックスをオンにします。                                                                                                                                                                                                                                      |
 | **削除スケジュール設定** | 削除の頻度を設定するには、次のいずれかのオプションを選択します。<br />****毎日 - デバイスを毎日削除する場合に設定します。
* **毎週 - **削除を予定する曜日を指定します。
* **毎月 - **毎月の第1日目に撤去済みデバイスを削除する場合に設定します。|


### ワイプ済みデバイスの削除

**手順**

1. **[管理]** > **[システム]** > **[デバイスクリーンアップ]** を開きます。 [デバイスクリーンアップ] ページが開きます。
2. **[ワイプ済みデバイスを削除]** を選択します。
3. 「**ワイプ済みデバイスを削除**」表を使用して、詳細を指定します。
4. **[ワイプされたデバイスのリストを表示]** をクリックします。 指定した日数の間に撤去されていたデバイスのリストが表示されます。
5. **[ワイプ済みデバイスを今すぐ削除]** をクリックするか、またはワイプ済みデバイスの削除をスケジュールできます。
6. 指定したデバイスが管理ポータルによって削除されます。
7. **[保存]** をクリックして設定を保存します。
8. （任意）値を更新した場合、**[リセット]** をクリックすると、設定を初期設定に戻せます。


### ワイプ済みデバイスの削除

| フィールド | 説明 |
 | -------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | **(日数) 以上消去されたデバイスを削除します** | 日数: デフォルトは 30 日で、最大許容日数は 365 日です。                                                                                                                                                                                                                                   |
 | **各セッションで削除するワイプデバイスの最大数** | 100、500、または 1000 を選択します (デフォルト - 100) |
 | **スケジュールに従ってワイプされたデバイスを自動的に削除する** | 事前に設定されたスケジュールに基づいてワイプされたデバイスを削除するには、チェックボックスをオンにします。                                                                                                                                                                                                                                    |
 | **削除済みデータの削除スケジュール設定** | 削除の頻度を設定するには、次のいずれかのオプションを選択します。<br />****毎日 - ワイプ済みデバイスを毎日削除する場合に設定します。
* **毎週 - **削除を予定する曜日を指定します。
* **毎月 - **毎月の第1日目にワイプ済みデバイスを削除する場合に設定します。|

3. 「**ワイプ保留中デバイスの削除**」表を使用して、詳細を指定します。
4. **[ワイプ保留中デバイスのリストを表示]** をクリックします。 指定した日数の間にワイプされる予定であるデバイスのリストが表示されます。
5. **[ワイプ保留中デバイスを今すぐ削除]** をクリックするか、またはワイプ保留中デバイスの削除をスケジュールできます。

### ワイプ保留中デバイスの削除

| フィールド | 説明 |
 | ------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | **ワイプ保留中の日数が (日数) を超えるデバイスを削除します** | 日数: デフォルトは 30 日で、最大許容日数は 365 日です。                                                                                                                                                                                                                                                 |
 | **各セッションで削除するワイプ保留中のデバイスの最大数** | 100、500、または 1000 を選択します (デフォルト - 100) |
 | **ワイプ保留中のデバイスをスケジュールに従って自動的に削除します** | 事前設定されたスケジュールに基づいてワイプされたデバイスを削除するには、チェックボックスをオンにします。                                                                                                                                                                                                                                                  |
 | **削除ワイプ保留スケジュール構成** | 削除の頻度を設定するには、次のいずれかのオプションを選択します。<br />****毎日 - ワイプ保留中デバイスを毎日削除する場合に設定します。
* **毎週 - **削除を予定する曜日を指定します。
* **毎月 - **毎月の第1日目にワイプ保留中デバイスを削除する場合に設定します。|


## 撤去保留中のデバイスの撤去

2. **[撤去保留中のデバイスを撤去]** を選択します。
3. 「**撤去保留中のデバイスを撤去**」表を使用して、詳細を指定します。
4. **[撤去保留中デバイスのリストを表示]** をクリックします。 指定した日数の間に撤去される予定であるデバイスのリストが表示されます。
5. **[撤去保留中のデバイスを今すぐ強制撤去]** をクリックするか、または撤去保留中デバイスの撤去をスケジュールできます。
6. 指定したデバイスが管理ポータルによって撤去されます。
7. **[保存]** をクリックして設定を保存します。
8. （任意）値を更新した場合、**[リセット]** をクリックすると、設定を初期設定に戻せます。

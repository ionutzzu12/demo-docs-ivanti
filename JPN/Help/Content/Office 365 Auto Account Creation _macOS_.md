**対象：**

- サポートされるmacOSデバイス。 
- 推奨されるMicrosoft Office 365アプリのバージョンは、16.13x以降です。


ユーザ情報とオプションを構成し、すべてのMicrosoft Office 365アプリについて初期構成を設定します。

このセクションは以下のトピックを含みます。

## [****](#)Office 365自動アカウント作成構成の作成

**手順**

1. **[構成]** を選択します。
2. **[+追加]** をクリックします。
3. 検索フィールドに **[オフィス]** と入力し、**Office 365自動アカウント作成**構成をクリックします。
4. 構成の名前と説明を入力します。
5. 次のOffice 365自動アカウント作成設定表に記載されている設定を入力します。
6. **[次へ]** をクリックして配布設定を行います。
7. **[完了]** をクリックします。


## [****](#)Office 365自動アカウント作成設定

| **設定** | **対処方法** |
 | ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | Office アクティベーション メール アドレス | ユーザーのメール アドレスを入力します。                                                                                                                                                                    |

 | Office 自動サインイン | 初回実行時のウィンドウを抑制する場合に選択します。 O365認証などの必須情報のみユーザにプロンプト表示します。パネルの「開く」/「保存」を、「オンラインロケーション」ではなく「Mac上」に強制する場合に選択します。|

 | 起動時に新機能を表示 | 起動時に新機能の情報を表示する場合に選択します。                                                                                                                                         |

 | Visual Basic マクロ実行状態 | 次のいずれかのオプションを選択します。<br />警告付きで無効
* 警告なしで無効
* 警告なしで有効化 |

 | Visual Basic 外部 Dylibs を無効にする | Visual Basic の外部依存関係を無効にする場合に選択します。                                                                                                                                            |

 | Visual Basic がシステムをバインドできるようにする | マクロが DECLARE を使用して system() OS API にバインドできるようにする場合に選択します。 このAPIにより、マクロが任意の外部プロセスを実行し、それらにコマンドライン上で任意のデータを渡すことができます。|

 | Visual Basic を無効にして Popen にバインドする | マクロが DECLARE を使用して popen() OS API にバインドできるようにする場合に選択します。 このAPIにより、マクロが任意の外部プロセスを実行し、それらにコマンドライン上で任意のデータを渡すことができます。|

 | Visual Basic Mac スクリプトを無効にする | マクロが Apple Script Visual Basic API を呼び出すことを許可する場合に選択します。                                                                                                                          |


[****](./Configurations.md)詳細は構成を作成するにはを参照してください。

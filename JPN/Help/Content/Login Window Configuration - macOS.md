**手順**

1. にログインします。
2. **[構成]** > **[構成を追加]** を開きます。
3. **[ログインウィンドウ]** をクリック（または選択）します。
4. **[名前]** フィールドに、ログインウィンドウ構成の名前を入力します。
5. **[+ 説明を追加]** をクリックして詳細を追加します。
6. **[構成設定]** セクションの下で、次のフィールドに入力します。
   - **管理ホスト情報**：ホスト名またはIPアドレス、またはシステムバージョンを入力します。
   - **許可リスト**：システムがログインを許可するユーザのユーザGUIDまたはグループGUIDのリスト。 アスタリスク（「*」）文字列は、すべてのユーザまたはグループを指定します。 
   - **自動ログインパスワード**：自動ログインのパスワードを入力します。
   - **拒否リスト**：システムがログインを拒否するユーザのユーザGUIDまたはグループGUIDのリスト。 このリストは、'AllowList' キー内のリストよりも優先されます。
   - **ログイン ウィンドウのテキスト**：ログインウィンドウのテキストを入力します。
   - 次のような追加オプションのチェックボックスを選択します。
     - **コンソールアクセスを無効にする**
     - **FDE自動ログインを無効にする（FileVaultディスク暗号化）**
     - **即時画面ロック機能を無効にする**
     - **ユーザリスト表示時に管理者ユーザを非表示にする**
     - **ネットワーク ユーザとシステム ユーザのみを表示**
     - **モバイルアカウントを非表示にする**
     - **ネットワークユーザを表示**
     - **「ログアウト」メニュー項目を無効にする**
     - **ユーザのログイン時に電源オフを無効にする**
     - **ユーザのログイン時に再起動メニュー項目を無効にする**
     - **再起動項目を無効にする**
     - **名前とパスワードのダイアログを表示する**
     - 他のユーザを表示する - 管理対象
     - 入力メニューを表示する
     - シャットダウンボタンを無効にする
     - スリープボタンを無効にする
     - シャットダウンメニューボタンを無効にする


7. **[完了]** をクリックします。

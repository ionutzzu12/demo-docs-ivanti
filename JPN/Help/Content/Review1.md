## 新機能セクション

- **アプリ インベントリからのアプリ詳細情報へのアクセス:** 管理者は、**[デバイス] > [アプリ インベントリ]** のアプリケーション リンクをクリックして、アプリ カタログの対応するアプリケーション詳細ページに移動できます。 詳細については、アプリ インベントリをご参照ください。
- **永続的なアプリ インベントリ検索結果:** アプリ インベントリ検索結果でデバイス番号リンクをクリックした後に表示される [デバイス] ページでブラウザの戻るボタンをクリックすると、アプリ インベントリ ページが開き、元の検索結果が読み込まれます。 
- **特定のアプリのすべてのバージョンおよびインベントリ詳細情報が表示された [アプリ詳細] ページからのワンクリック ビュー:** 管理者が **[アプリ] > [アプリ カタログ]** > *[選択したアプリ]* から **[アプリ インベントリの表示 (すべてのバージョン)]** をクリックすると、**[デバイス] > [アプリ インベントリ]** にそのアプリのバンドル ID でフィルタリングされた一覧を表示できます。詳細については、アプリ詳細ページからアプリ インベントリ情報を表示するをご参照ください。
- **バンドル/パッケージ ID でアプリ インベントリを検索:** 管理者は、名前またはバンドル/パッケージ ID で **[デバイス] > [アプリ インベントリ]** を検索できます。 詳細については、アプリ インベントリをご参照ください。


## 機能ドキュメント

### アプリ詳細ページからアプリ インベントリ情報を表示する

アプリ インベントリ情報を表示するには、**[アプリ インベントリの表示 (すべてのバージョン)]** をクリックすると、**[デバイス] > [アプリ インベントリ]** にそのアプリのバンドル ID でフィルタリングされた一覧が表示されます。

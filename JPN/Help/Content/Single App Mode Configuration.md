## Single Appモード構成

| **設定** | 対処方法 |
 | --------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | 名前 | この構成を識別する名前を入力します。                                                                                                                                                                                                                                                                                                                                                                                                                                                           この構成の目的を明示する説明を入力します。アプリの選択方法を選びます。 <br />****アプリカタログおよびシステムアプリから： Ivanti Neurons for MDM アプリカタログとシステムアプリ（Appleデバイスのデフォルトでインストール済み）から検索する場合に選択します。
*  アプリの名前を入力し、アプリリストに表示されたら選択します。 
*   **バンドルIDを入力：**選択したいシステムアプリの固有識別子を入力する場合に選択します。 このオプションは、**[アプリカタログおよびシステムアプリから]** オプションでシステムアプリが見つからない場合に使用してください。|

 | タッチを無効にする | 選択するとタッチ スクリーンが無効になります。                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

 | デバイスの回転を無効にする | デバイスの回転検知を無効にする場合に選択します。                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |

 | 音量ボタンを無効にする | デバイスの音量ボタンを無効にする場合に選択します。                                                                                                                                                                                                                                                                                                                                                                                                                                                             |

 | 着信音スイッチを無効にする | デバイスの着信音スイッチを無効にする場合に選択します。                                                                                                                                                                                                                                                                                                                                                                                                                                                              デバイスのスリープ/スリープ解除ボタン（デバイスの枠の右上）を無効にする場合に選択します。操作しないままの状態の後でもスリープにならないようにする場合に選択します。VoiceOver画面リーダーを有効にする場合に選択します（アクセシビリティ機能）。ズーム機能を有効にする場合に選択します（アクセシビリティ機能）。色の反転調整を有効にする場合に選択します（アクセシビリティ機能）。Assistive Touchを有効にする場合に選択します（アクセシビリティ機能）。選択項目の読み上げを有効にする場合に選択します（アクセシビリティ機能）。ステレオとモノラルのオーディオ切り替えを有効にする場合に選択します（アクセシビリティ機能）。デバイスがVoiceOverの調整を行えるようにする場合に選択します。デバイスユーザがズーム機能の調整を行えるようにする場合に選択します。|

 | 色調整を反転 | 選択すると、デバイス ユーザーが色を反転できるようになります。                                                                                                                                                                                                                                                                                                                                                                                                                                                             |

 | Assistive Touch 調整 | 選択すると、ユーザーが AssistiveTouch 調整を行えるようになります。                                                                                                                                                                                                                                                                                                                                                                                                                                                  |


[****](./Configurations.md)詳細は構成を作成するにはを参照してください。

Android APN設定の構成では、パブリックネットワーク上のデバイスに必要なアクセスポイント名（APN）を設定できます。 この構成は、Android Enterprise仕事用マネージドデバイス、および会社所有デバイス上の仕事用プロファイルを持つマネージドデバイス（Androidバージョン9.0またはサポートされる以降のバージョン）に適用されます。

**手順**

1. **[構成]**** > [+追加]** を開きます。
2. **[Android APN設定]** 構成を選択します。
3. 構成の名前を入力します。
4. 説明を入力します。
5. [構成設定] セクションで、以下のオプションを構成します。


| 設定 | 説明 |
 | ----------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | **エントリ名** | アクセス ポイント設定の名前を入力します。                                                                                                                                                                                                                                                                               |
 | **アクセス ポイント名** | アクセス ポイントの名前を入力します。                                                                                                                                                                                                                                                                                            ****アクセスポイントのタイプを以下から選択：<br />****デフォルト
* **ダン**
* **IMSI **
* **緊急**
* MMS
* **HIPRI**
* **CBS**
* **MCX**
* **スープル**
* **フォタ**
* ********仮想移動体通信事業者（MVNO）のタイプを以下から選択：<br />****なし
* **SPN**
* **IMSI **
* **性同一性障害**
* ******** データ送信に使用されるベアラサービスのタイプを以下から選択：<br />* **1xRTT**
* **CDMA**
* **角**
* **EHRPO**
* **EVDO**
* **EVDO A**
* **EVDO B**
* **GPRS**
* **GSM**
* **HSDPA**
* **ハスプ**
* **HSPAP**
* **HSUPA**
* **アイデン**
* **無線LAN**
* **LTE**
* **NR**
* **TD\\SCDMA**
* **UMTS** |
 | **APN プロトコル** | APN に必要な APN プロトコルを選択します。 以下から選択：<br />****なし
* **IPV4**
* **IPV6**
* **IPV4/IPV6**
* **非IP**
* **PPP**（ポイントツーポイントプロトコル）
* **非構造化** |
 | **APN ローミング プロトコル** | APN に必要な APN ローミング プロトコルを選択します。 以下から選択：<br />****なし
* **IPV4**
* **IPV6**
* **IPV4/IPV6**
* **非IP**
* **PPP**（ポイントツーポイントプロトコル）
* **非構造化** |
 | **APN を有効/無効にする** | APN 設定をオンにします。                                                                                                                                                                                                                                                                                            ****通信事業者IDの数値を入力。****認証プロトコルのタイプを以下から選択：<br />****なし
* **PAP**（Password Authentication Protocol）
* **CHAP**（Challenge-Handshake Authentication Protocol）
* **PAP または CHAP** |
 | **ユーザー名** | ログインユーザー名を入力します。                                                                                                                                                                                                                                                                                                 |
 | **パスワード** | ログイン パスワードを入力します。                                                                                                                                                                                                                                                                                                 |
 | **パスワードの確認** | 確認のためにパスワードをもう一度入力します。                                                                                                                                                                                                                                                                                   ****ポート番号を入力（1～65535の数値）。|
 | **プロキシ アドレス** | プロキシ アドレスを入力します。                                                                                                                                                                                                                                                                                                   |
 | **携帯電話の国コード** | 携帯電話の国コードを入力します。                                                                                                                                                                                                                                                                                            |
 | **モバイル ネットワーク コード** | モバイル ネットワーク コードを入力します。                                                                                                                                                                                                                                                                                            |
 | **MMS プロキシ アドレス** | MMS プロキシ アドレスを入力します。                                                                                                                                                                                                                                                                                               ****MMSポート番号を入力（1～65535の数値）。|
 | **MMS サーバー アドレス (mmsc)** | MMS サーバー アドレスを入力します。                                                                                                                                                                                                                                                                                              |


6. **[次へ]** をクリックします。
7. 以下の配布オプションから1つ選択します。
   - **すべてのデバイス**
   - デバイスなし（デフォルト）
   - **カスタム**

8. **[完了]** をクリックします。


デバイスが以下の値を持つ既存のAPN構成がある場合、以下のフィールドに同じ値を持つAPN構成を追加することはできません。
- Mobile Country Code
- Mobile Network Code
- アクセスポイント名
- プロキシアドレス
- ポート番号
- MMSプロキシアドレス
- MMSポート番号
- MMSサービスアドレス
- APNを有効化/無効化
- MVNOのタイプ
- APNプロトコル
- APNローミングプロトコル


Android APN設定の構成は、手動で、またはオペレータによってデバイス内ですでに構成されている他のAPN設定より優先されます。

La configuration Écran de veille vous permet d'ajouter des options, notamment Mot de passe, Délai d'inactivité, Chemin du module et Nom de module.

**Procédure**

1. Connectez-vous au portail d'administration .
2. Cliquez sur **Configurations**.
3. Cliquez sur **+ Ajouter**.
4. Tapez « écran » dans le champ de recherche, puis cliquez sur **Écran de veille**:
 La page de détails de configuration de l'économiseur d'écran s'affiche.
5. Configurez les paramètres disponibles sur cette page. Reportez-vous au tableau de la section **Paramètres de configuration Écran de veille** pour obtenir des conseils concernant les valeurs.
6. Cliquez sur **Suivant** pour configurer les paramètres de distribution, puis sur **Terminé**.


## Paramètres de configuration Écran de veille

|  **Paramètre**                                |  Action                                                                                                                                       |
|-----------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
|**Nom** (obligatoire)|Saisissez un nom qui identifie cette configuration.|
|**Description**|Saisissez une description qui précise l'objectif de cette configuration.|
|Case à cocher Demander le mot de passe|Cochez cette case pour inviter l'utilisateur du périphérique à entrer un mot de passe pour déverrouiller ou arrêter l'écran de veille. (Disponible pour macOS 10.1.13 et supérieur.)|
|**Retard de l'invite de mot de passe**|Spécifiez une durée de retard en secondes.|
|**Délai d'inactivité de la fenêtre de connexion**|Spécifiez le délai d'inactivité (en secondes) à l'issue duquel l'écran de veille doit s'afficher.|
|**Chemin du module d'écran de veille**|Spécifiez le chemin du module d'écran de veille.|
|**Nom du module d'écran de veille** (obligatoire)|Entrez le nom de l'écran de veille.|

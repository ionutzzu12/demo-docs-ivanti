Cette configuration permet aux entreprises de restreindre, modifier ou configurer les paramètres d'accessibilité des périphériques Apple.

**Procédure**

1. Accédez à **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Cliquez sur **Configurations Paramètres d'accessibilité**.
4. Configurez les paramètres indiqués dans le tableau suivant :

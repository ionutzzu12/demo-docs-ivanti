 Cette section traite des sujets suivants :
- Modifier la configuration du code d'accès attribuée
- Affecter une autre configuration du code d'accès


Utilisez le [**Configuration du code d'accès**](\<./Passcode Configuration.md>) attribué à un appareil pour modifier les paramètres du code d'accès. Vous pouvez :
- Modifier les paramètres de la configuration attribuée


OU

- Attribuer une autre configuration du code d'accès


**Les modifications que vous apportez à la configuration sont appliquées à tous les périphériques auxquels la configuration est attribuée**. 

## [****](#)Modifier la configuration du code d'accès attribuée

**Procédure**

1. Accédez à **Périphériques**.
2. Recherchez le nom du périphérique dans la liste.
3. Cliquez sur le lien dans la colonne **Nom**.
4. Si une configuration de code d'accès a été attribuée, elle s'affichera dans l'onglet Configurations.
 Dans l'onglet Configurations, cliquez sur le lien **Configuration du code d'accès** .
5. Cliquez sur **Modifier** (en haut à droite).
6. Effectuez les modifications.


## [****](#)Affecter une autre configuration du code d'accès

**Procédure**

1. Assurez-vous que la configuration dont vous êtes besoin existe.
2. Accédez à **Périphériques**.
3. Recherchez le nom du périphérique dans la liste.
4. Cliquez sur le lien dans la colonne **Nom**.

Ivanti Tunnel active les fonctionnalités VPN sur les appareils iOS, Android et Windows. Il interagit avec la plateforme de gestion unifiée des terminaux (UEM), Standalone Sentry et Access pour sécuriser l'accès aux ressources de l'entreprise depuis l'extérieur du réseau. Ces ressources peuvent être hébergées sur site ou dans le cloud. Les plateformes UEM sont Ivanti EPMM et Ivanti Neurons pour la gestion des appareils mobiles (MDM).

La configuration par application est obsolète pour les appareils Android Enterprise. Vous devez utiliser la configuration gérée pour Ivanti Tunnel depuis le catalogue d'applications.

## À propos de la configuration d'Ivanti Tunnel

Les configurations d'Ivanti Tunnel sont créées sur une plateforme de gestion unifiée des terminaux (UEM). Ivanti Tunnel reçoit ensuite cette configuration du client UEM. Le client d'Ivanti EPMM est Mobile@Work, et celui d'Ivanti Neurons pour la gestion des appareils mobiles (MDM) est Go.

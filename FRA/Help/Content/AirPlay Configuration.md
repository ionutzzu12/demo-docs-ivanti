**Licence :**

La configuration AirPlay permet de définir un accès à différents périphériques pour consulter des fichiers multimédias. Le tableau suivant répertorie les paramètres Airplay :

|  **Paramètre**|  Action                                                                                                                                                                                          |
|---------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Liste d'autorisations|Saisissez l'identifiant du périphérique pour chaque destination AirPlay autorisée. Si vous n'indiquez pas d'identifiant, les destinations AirPlay ne sont pas limitées.<br />**S'applique à** : iOS 7.0+ et macOS 10.10+ (supervisé).|
|Paramètres du périphérique|Saisissez l'ID du périphérique (macOS) ou le nom du périphérique (iOS) ainsi que le mot de passe correspondant pour chaque destination AirPlay connue.|

[****](./Configurations.md)Pour en savoir plus, consultez la rubrique Création d'une configuration.

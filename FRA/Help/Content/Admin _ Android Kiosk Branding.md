Vous pouvez intégrer des éléments de votre marque dans la page du kiosque Android pour que vos utilisateurs accèdent à une interface qui leur est plus familière. Vous pouvez personnaliser les éléments suivants :
- logo (PNG, 840x114) ou texte de la bannière
- couleur de bordure de la bannière
- couleur d'arrière-plan de la bannière
- couleur d'arrière-plan de l'écran
- image d'arrière-plan de l'écran
- format d'arrière-plan de l'écran


## Intégrer des éléments de la marque dans la fenêtre du kiosque Android

Procédure

1. Accédez à **Admin > Kiosque Android**.
2. Sur la page **Éléments de la marque dans le mode kiosque**, cliquez sur **Créer un élément de marque**.
3. Dans le champ **Nom**, saisissez le nom de l'éléments de la marque dans le mode kiosque.
4. Pour désactiver la bannière, désélectionnez la case à cocher **Activer la bannière supérieure**.
5. Cliquez sur le champ **Couleur d'arrière-plan de la bannière** pour ouvrir une palette de couleurs et sélectionner celle que vous préférez, ou entrer son code hexadécimal.
6. Cliquez sur le champ **Couleur de bordure de la bannière** pour ouvrir une palette de couleurs et sélectionner celle que vous préférez, ou entrer son code hexadécimal.
7. Sélectionnez **Image/Logo** ou **Texte** pour définir le contenu de la bannière.
8. Si vous avez sélectionné **Image/Logo**, faites glisser le fichier d'image ou cliquez sur **Choisir un fichier** pour en sélectionner un.
9. Si vous avez sélectionné **Texte**, tapez le texte à afficher dans la bannière.
10. Cliquez sur l'onglet **Arrière-plan**.
11. Cliquez sur le champ **Couleur d'arrière-plan** pour ouvrir une palette de couleurs et sélectionner celle que vous préférez, ou entrer son code hexadécimal.
12. Pour changer l'image d'arrière-plan :
13.  a. Supprimez l'image par défaut.Faites glisser l'image voulue ou cliquez sur **Choisir un fichier** pour en sélectionner un.Sélectionnez la mise en page souhaitée.
 Cliquez sur **Enregistrer les modifications**.

    L'élément créé est indiqué sur la page **Éléments de la marque dans le mode kiosque**. Pour apporter d'autres modifications aux éléments de marque, cliquez sur l'icône Modifier dans la colonne **Actions**. Pour supprimer les éléments de marque personnalisés, cliquez sur l'icône Supprimer. Toute configuration reposant sur ces éléments de marque personnalisés réutilisera les éléments de marque par défaut.

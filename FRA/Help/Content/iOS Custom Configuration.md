La configuration personnalisée d'iOS vous permet d'importer et de distribuer un profil de configuration d'iOS créé par une autre appli, comme l'Utilitaire de configuration iPhone d'Apple.

## Paramètres personnalisés d'iOS

|  **Paramètre**|  Action                                                                                         |
|-----------|-------------------------------------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Données du fichier|Faites glisser le fichier de configuration et déposez-le dans la zone pointillée, ou cliquez sur **Choisir un fichier** pour sélectionner un fichier dans votre ordinateur.|

[****](./Configurations.md)Pour en savoir plus, consultez la rubrique Création d'une configuration.

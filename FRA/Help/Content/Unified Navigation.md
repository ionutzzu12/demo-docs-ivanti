Nous avons créé Neurons Unified Navigation pour offrir aux utilisateurs de Neurons une expérience cohérente et rationalisée dans l'ensemble des applications **Ivanti Neurons for MDM**, **Neurons for Endpoint Management** et **Neurons for IT Service Management**. 

La nouvelle zone de navigation supérieure vous permet désormais d'accéder à différentes sections depuis l'angle supérieur droit de la fenêtre, avec les icônes suivantes :

**Icônes de barre d'outils**

| Icône | Description |
 | ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | ![](Resources/Images/bell_icon.png) | **Notifications**: Cliquez pour afficher la liste des événements ou messages importants. |
 | ![](Resources/Images/Global_Announcements.JPG) | **Annonces mondiales**: Cliquez pour afficher la liste des nouveautés récemment publiées. |
 | ![](Resources/Images/help_icon.png) | **Aide**: Cliquez pour consulter la documentation **Ivanti Neurons pour MDM** . |
 | ![](Resources/Images/app_switcher.png) | **Sélecteur d'applications Neurons**: Cliquez pour afficher les applications auxquelles vous pouvez accéder à partir d'ici.<br />************ Utilisez le sélecteur d'appli Neurons pour accéder à Neurons for Endpoint Management ou à Neurons for IT Service Management.
* ************Le sélecteur d'appli Neurons vous permet de basculer entre les applications , Neurons for Endpoint Management et Neurons for IT Service Management si vous disposez des droits d'accès voulus sur ces applications.
* Si vous n'avez pas accès à **Neurons for Endpoint Management** ou à **Neurons for IT Service Management**, seule cette option sera visible. |
 | ![](Resources/Images/profile_icon.png) | **Compte**: Cliquez pour afficher les détails de votre compte. Cette section propose les options suivantes :<br />****Mise à niveau

### Marque globale

********

Prérequis pour l'accès au paramètre Marque globale :

****Outre , vous devez avoir accès à Neurons for Endpoint Management pour utiliser ce paramètre. 

Par défaut, ce paramètre est INACTIF, mais il peut être activé sur demande pour les locataires configurés avec **Neurons for Endpoint Management**. 

Pour accéder au paramètre de marque, accédez à **Paramètres globaux** > **Branding (Marque)**.

**Rétablir les valeurs par défaut** – Utilisez cette option pour rétablir les valeurs par défaut des paramètres modifiés.

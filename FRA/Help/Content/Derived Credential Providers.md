Sur la page Fournisseurs d'identifiants dérivés, vous pouvez consulter la liste des fournisseurs d'identifiants dérivés utilisés pour la distribution des certificats. Vous pouvez spécifier les fournisseurs d'identifiants dérivés par défaut et ajouter d'autres fournisseurs personnalisés.

Pour définir un fournisseur de références d'authentification dérivées par défaut :

1. Accédez à **Administration > Fournisseurs de références d'authentification dérivées**. Les fournisseurs ci-dessous sont répertoriés dans la page.
   - **Entrust**
   - **Intercede**
   - **Purebred**

2. ************


Pour ajouter un fournisseur de références d'authentification dérivées personnalisé :

1. Accédez à **Administration > Fournisseurs d'identifiants dérivés**.
2. Cliquez sur **+ Ajouter**.
3. Saisissez le nom du fournisseur de références d'authentification dérivées dans la zone de texte de la colonne **Nom**.
4. Cliquez sur **Enregistrer**.

   ********Une fois ajouté, ce fournisseur de références d'authentification dérivées personnalisé peut être sélectionné dans le champ Marque lors de la configuration de la distribution de références d'authentification dérivées avec certificat d'identité.


Cliquez sur **Supprimer** dans la colonne **Actions** pour supprimer un fournisseur de références d'authentification dérivées.

Il n'est pas possible de supprimer un fournisseur de références d'authentification dérivées par défaut.

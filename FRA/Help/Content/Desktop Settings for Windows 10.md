La configuration des paramètres du bureau pour Windows 10 vous permet de personnaliser les paramètres du bureau et de les appliquer aux appareils Windows 10. Grâce à cette configuration, vous pouvez paramétrer les éléments suivants :

- Image d'arrière-plan du bureau
- Image de l'écran de verrouillage
- Télécharger un économiseur d'écran personnalisé
- Raccourcis du bureau


Cette fonctionnalité nécessite Bridge. Consultez Bridge pour plus de détails.

**Procédure**

1. ******** Sous Configuration, cliquez sur +Ajouter. 
2. ********
3. **** Dans le champ Nom, attribuez un nom adapté aux paramètres. 
4. (Facultatif) Cliquez sur le lien +**Ajouter une description** pour ajouter une description de la configuration.
5. Dans la section **Installation de la configuration**, configurez les paramètres suivants :


| Paramètres | Description |

 | ---------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

 | Livraison de fichiers | Sélectionnez l'une des options de livraison de fichiers suivantes pour les paramètres de bureau :<br />****Télécharger un fichier - Télécharger les paramètres vers Ivanti Neurons for MDM. 
* **URL de remplacement** - Fournissez des URL de remplacement pour les fichiers de paramètres à télécharger. |
 | Paramètres du fond d'écran | Cliquez sur **Choisir un fichier** pour sélectionner et télécharger une image de fond d'écran : les formats de fichiers pris en charge sont BMP, JPG, JPEG et PNG. |
 | Paramètres du fond d'écran de verrouillage<br />********<br />Les formats de fichiers pris en charge sont BMP, JPG, JPEG et PNG.
 | Paramètres de l'économiseur d'écran | Cliquez sur **Choisir un fichier** pour localiser et télécharger un fichier d'économiseur d'écran.<br />Téléchargez uniquement des fichiers .SCR compatibles avec Windows.<br />Sélectionnez **Protection par mot de passe pour l'économiseur d'écran** si vous souhaitez imposer un mot de passe pour sortir du mode Économiseur d'écran.<br />Sélectionnez une durée de délai d'inactivité de l'économiseur d'écran (en minutes). |
 | Raccourcis de bureau | Cliquez sur **Ajouter un raccourci** pour configurer des raccourcis à ajouter au bureau de votre appareil. La fenêtre **Ajouter un raccourci** s'affiche. Remplissez le tableau en utilisant les options suivantes : ****<br />****Location (Emplacement) : saisissez l'emplacement où le raccourci apparaîtra sur les périphériques Windows.
* **Target Path** (Chemin d'accès de la cible) : entrez le chemin d'accès de la cible, un chemin UNC ou la lettre du lecteur  vers lequel le raccourci pointe. 
* **Arguments** : saisissez les éventuels arguments que vous souhaitez invoquer lors de l'ouverture du fichier cible.
* **Working Directory** (Répertoire de travail) : saisissez le chemin d'accès du dossier contenant les fichiers nécessaires pour la cible.
* **Icon File** (Fichier icône) : téléchargez des fichiers .ico Windows valides.<br />Après avoir configuré les options, cliquez sur **Ajouter un raccourci**.


6. Cliquez sur **Suivant**.
7. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée

8. Cliquez sur **Terminé**.

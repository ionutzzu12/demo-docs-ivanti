**S'applique à** : macOS 10.13.2 ou versions plus récentes compatibles.

Contrôle les restrictions et les paramètres de chargement des extensions du noyau approuvées par l'utilisateur.

## Création d'une configuration Stratégie d'extensions de noyau macOS

**Procédure**

1. Sélectionnez **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Saisissez **noyau** dans le champ de recherche, puis cliquez sur la configuration **Stratégie d'extensions de noyau macOS**.
4. Nommez et décrivez la configuration.
5. Sélectionnez l'option **Autoriser les remplacements par l'utilisateur** pour autoriser les utilisateurs à approuver des extensions de noyau supplémentaires qui ne sont pas explicitement autorisées par la configuration suivante.
6. ****
7. Cliquez sur **Ajouter**.
8. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
9. Cliquez sur **Terminé**.


[****](./Configurations.md)Pour en savoir plus, consultez la rubrique Création d'une configuration.

Les administrateurs peuvent autoriser les utilisateurs à afficher et à mettre à jour les périphériques vers la version la plus récente (nombre le plus élevé) ou la plus ancienne (nombre le plus faible), ou les deux.

S'applique à : iOS et iPadOS 14.5+ (supervisé)

1. Accédez à Configurations > +Ajouter.
2. Saisissez recommandations de mise à jour logicielle dans le champ de recherche, puis cliquez sur la configuration Fréquence des recommandations de mise à jour logicielle.
3. Saisissez le nom et la description de la configuration.
4. Dans le menu déroulant, sélectionnez l'installation de la configuration requise : 
   - Proposer les deux versions de mise à jour logicielle
   - Proposer la version de mise à jour logicielle la plus ancienne
   - Proposer uniquement la version de mise à jour logicielle la plus récente

5. Cliquez sur Suivant.
6. Sélectionnez l'option Activer cette configuration.
7. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée.

8. Cliquez sur Terminé.

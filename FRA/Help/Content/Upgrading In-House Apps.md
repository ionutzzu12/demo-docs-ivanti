Appliquez la procédure suivante pour mettre à jour une appli interne :

1. Accédez à **Applis > Catalogue d'applis**.
2. Sélectionnez l'appli à mettre à niveau.
3. Ouvrez **Actions > Ajouter une nouvelle version**.
4. Faites glisser l'appli vers la zone **Télécharger l'appli** ou cliquez sur **Choisir un fichier** pour sélectionner un fichier dans votre ordinateur.
5. Sélectionnez une des options suivantes en fonction de ce que vous souhaitez faire avec la version précédente de l'appli : 
   - **Conserver à l'identique la description, les captures d'écran, la distribution, ainsi que les prérequis et les configurations des applis** : Remplace la version précédente dans le catalogue d'applis.
   - **Modifier la description, les captures d'écran, la distribution, les prérequis ou les configurations des applis** : Inclut les deux versions dans le catalogue d'applis.

6. Sous **Nouveautés**, indiquez à l'intention des utilisateurs en quoi la nouvelle version est différente.
7. Ce texte s'affichera sur l'appareil lorsque l'utilisateur sélectionnera l'application pour installation.

 Si vous avez choisi de modifier les descriptions, les captures d'écran ou les options de distribution, veuillez finaliser ces modifications.
8. Cliquez sur **Terminé**.


****

## Afficher une liste des versions d'appli

Les administrateurs sont autorisés à télécharger des applis ayant la même version, mais des architectures différentes.

**Procédure**

1. Cliquez sur le lien de l'appli dans **Applis > Catalogue d'applis.**
2. Cliquez sur l'onglet **Version**.

Affiche un message et les informations d'identification de l'appareil sur les écrans de connexion et de verrouillage. Ceci concerne les appareils supervisés utilisant iOS 9.3 ou des versions ultérieures compatibles.

## Configurer le message de l'écran de verrouillage

Procédure

1. Sélectionnez **Configurations**.
2. Cliquez sur **+ Ajouter**.


::Image[]{src="r34001.png" size="100" caption alt isUploading="false" sha initialPath="r34001.png" githubPath="Content/r34001.png" indent="2"}

3. Saisissez **verrouillage** dans le champ de recherche, puis cliquez sur la configuration **Message de l'écran de verrouillage** :


::Image[]{src="r34002.png" size="100" caption alt isUploading="false" sha initialPath="r34002.png" githubPath="Content/r34002.png" indent="2"}

4. La page de détails de configuration des messages de l'écran de verrouillage s'affiche.
 Configurez les paramètres sur cette page. Consultez le tableau de la section [**Paramètres de configuration des messages de l'écran de verrouillage**](#Lock_Screen_Message_Configuration_settings) pour obtenir des indications sur les valeurs.
5. Cliquez sur **Suivant** pour configurer les paramètres de distribution, puis sur **Terminé**.


## [****](#)Paramètres de configuration du message de l'écran de verrouillage

|  **Paramètre**      |  Action                                                             |
|---------------------|---------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Note de bas d'écran de verrouillage|Ce texte s'affiche dans la fenêtre de connexion et sur l'écran de verrouillage.|
|Informations de suivi de l'actif|Ce texte s'affiche en bas de la fenêtre de connexion et de l'écran de verrouillage.|

[****](./Configurations.md)Pour en savoir plus, consultez la rubrique Création d'une configuration.

**Pour les applis publiques** disponibles sur Google Play Store :

1. Utilisez un navigateur Web pour localiser l'appli dans Google Play Store.
2. Sélectionnez l'appli.
3. Regardez l'URL affichée dans le navigateur.


Le nom de paquet est indiqué dans l'URL après l'élément « id= » tel qu'indiqué ci-dessous :

https://play.google.com/store/apps/details?id=\<package name>

****[****](https://play.google.com/store/apps/details?id=com.csdroid.pkg)Pour les applis internes et celles non disponibles dans Play Store, essayez de télécharger Package Name Viewer ou une appli similaire dans Google Play Store.

La configuration Écran de veille de l'utilisateur vous permet d'ajouter des options, notamment Mot de passe, Délai d'inactivité, Chemin du module et Nom de module.

**Procédure**

1. Connectez-vous au portail d'administration .
2. Cliquez sur **Configurations**.
3. Cliquez sur **+ Ajouter**.
4. Tapez « écran » dans le champ de recherche, puis cliquez sur **Écran de veille de l'utilisateur**:
 La page de détails de configuration de l'économiseur d'écran de l'utilisateur s'affiche.
5. Configurez les paramètres disponibles sur cette page. Reportez-vous au tableau de la section « **Paramètres de configuration Écran de veille de l'utilisateur** » pour obtenir des conseils sur ces valeurs.
6. Cliquez sur **Suivant** pour configurer les paramètres de distribution, puis sur **Terminé**.


## Paramètres de configuration Écran de veille de l'utilisateur

|  **Paramètre**                                |  Action                                                                  |
|-----------------------------------------------|--------------------------------------------------------------------------|
|**Nom** (obligatoire)|Saisissez un nom qui identifie cette configuration.|
|**Description**|Saisissez une description qui précise l'objectif de cette configuration.|
|**Délai d'inactivité**|Spécifiez le délai d'inactivité (en secondes) à l'issue duquel l'écran de veille doit s'afficher.|
|**Chemin du module d'écran de veille**|Spécifiez le chemin du module d'écran de veille.|
|**Nom du module d'écran de veille** (obligatoire)|Entrez le nom de l'écran de veille.|

[****](./Configurations.md)Pour en savoir plus, consultez la rubrique Création d'une configuration.

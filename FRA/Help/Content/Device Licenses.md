Les licences par appareil définissent le nombre d'appareils que vous pouvez enregistrer, la quantité de contenu que vous pouvez configurer pour la distribution sur ces appareils et les fonctionnalités disponibles. Si vous atteignez votre limite d'appareils, un triangle rouge s'affiche sur la page d'administration. Si vous atteignez votre limite de contenu, le service vous empêchera d'en ajouter et affichera un message vous informant que vous avez atteint votre limite.

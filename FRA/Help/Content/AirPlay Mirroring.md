La recopie vidéo AirPlay est une fonctionnalité qui vous permet d'afficher l'écran d'un périphérique iOS sur un moniteur avec Apple TV. Apple TV et le périphérique iOS doivent être connectés au même réseau Wi-Fi. Cette fonctionnalité requiert les périphériques suivants :
- Périphérique iOS 7 ou version ultérieure (version supervisée)
- Périphériques macOS 10.10 ou version ultérieure (version supervisée)
- d'Apple TV -
- AirPlay supervisé


L'inclusion de la gestion des périphériques non-iOS est une action irréversible.

Cette section traite des sujets suivants :

## [****](#)Configurer Apple AirPlay

****Pour en savoir plus sur les paramètres de configuration d'AirPlay, consultez la page Configuration d'AirPlay.

Procédure

1. Accédez à **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Cliquez sur **AirPlay**.
4. Saisissez le nom et la description de la configuration dans les champs prévus à cet effet.
5. Pour toutes les versions prises en charge d'iOS, saisissez le nom du périphérique et le mot de passe.
6. Cliquez sur + **Ajouter** pour ajouter un autre périphérique, si nécessaire.
7. (Facultatif) Vous pouvez ajouter les ID des périphériques supervisés iOS 7+ ou macOS 10.10+ à une liste d'autorisations.
8. Cliquez sur **Suivant**.
9. Sélectionnez un niveau de distribution.
10. Cliquez sur **Terminé**.


## [****](#)Configurer AirPlay sur le périphérique mobile

Procédure

1. Configuration [**Apple Configurator**](\<./Apple Configurator.md>).
2. Accédez à ** Périphériques > Périphériques**.
3. Cliquez sur le nom d'un périphérique iOS pour afficher la page Détails de ce périphérique.
4. Cliquez sur l'icône ![](More_icon.png).
5. Sélectionnez **Recopier vidéo AirPlay** pour afficher la boîte de dialogue du même nom.
6. Sélectionnez un périphérique Apple TV dans le menu déroulant.
7. Indiquez la durée de balayage en secondes durant laquelle le périphérique que vous avez sélectionné doit être recherché.
8. Saisissez le mot de passe du périphérique Apple TV.
9. Cliquez sur **Envoyer une demande**.


## [****](#)Configurer un moniteur pour Apple TV

Procédure

1. Sur un moniteur connecté à Apple TV, ouvrez **Paramètres > Profil**.
2. Sélectionnez **Ivanti Neurons for MDM Apple Configurator**.
3. Cliquez sur **Ajouter un profil**.
4. Cliquez sur l'icône ![](More_icon.png).
5. Sélectionnez **Recopier vidéo AirPlay** pour afficher la boîte de dialogue du même nom.
6. Sélectionnez un périphérique Apple TV dans le menu déroulant.
7. Indiquez la durée de balayage en secondes durant laquelle le périphérique que vous avez sélectionné doit être recherché.
8. Saisissez le mot de passe du périphérique Apple TV.
9. Cliquez sur **Envoyer une demande**.


## [****](#)Connecter votre périphérique iOS à Apple TV

Procédure

1. Connectez le périphérique Apple TV à un moniteur.
2. Avec la télécommande Apple TV, ouvrez **Réglages > Comptes > Partage à domicile** pour activer le partage à domicile.
3. **Connectez le périphérique iOS** au même réseau Wi-Fi que votre **périphérique Apple TV**.
4. Ouvrez l'appli Remote sur votre périphérique **iOS**.
5. Activez le **Partage à domicile** dans l'écran **Réglages Remote**.

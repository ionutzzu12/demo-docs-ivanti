**S'appliquent à :**

- iOS 11,3 et versions ultérieures
- macOS 10.13.4 et versions ultérieures


Vous pouvez activer ou désactiver le Bluetooth sur un périphérique. 

**Procédure**

1. Accédez au périphérique dans la page [**Périphériques**](./Devices-.md).
2. Effectuez l'une des actions suivantes :
   - Sélectionnez les périphériques dans la liste.
   - Cliquez sur le nom du périphérique pour afficher la page des détails de ce dernier.

3. Dans le menu **Actions**, cliquez sur **Activer/Désactiver le Bluetooth**.
4. Cliquez sur **OK**.


Les modifications seront appliquées sur le périphérique ou les périphériques lors du prochain enregistrement.

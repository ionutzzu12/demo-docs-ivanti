La configuration d'Apple TV permet de définir la langue et les paramètres régionaux pour Apple TV. 

Les paramètres d'Apple TV sont les suivants :

|  **Paramètre**|  Action                                                                   |
|-----------|---------------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Langage|Saisissez le code de langue à deux lettres pour préciser la langue de l'interface.|
|Paramètres régionaux|Saisissez l'identifiant des paramètres régionaux afin de préciser l'association langue/pays valable dans l'interface.|

[****](./Configurations.md)Pour en savoir plus, consultez la rubrique Création d'une configuration.

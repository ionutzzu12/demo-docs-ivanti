Les restrictions AppStore macOS définissent les fonctions activées dans AppStore macOS.

Vous pouvez définir les options suivantes :

| Paramètres | Que faire |
 | ----------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
 | Nom | Saisissez un nom qui identifie cette configuration. |
 | Description | Saisissez une description qui précise l'objectif de cette configuration. |
 | **Configuration** | |
 | **Version macOS** | **Fonctionnalités** |
 | 10.9 | Limiter l'installation des applications aux utilisateurs administrateurs. |
 | 10.10 | - Limiter les installations d'applications aux seules mises à jour logicielles.
- Désactiver l'adoption d'appli par les utilisateurs.
- Désactiver les notifications de mise à jour logicielle. |

 | 10.11 | Limiter l'installation d'applications aux applications installées via MDM et aux mises à jour logicielles. |


## Distribution de la configuration

**Procédure**

1. Définissez les options à l'aide du tableau précédent.
2. Cliquez sur **Suivant**.
3. Sélectionnez l'option **Activer cette configuration**.
4. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée

5. Cliquez sur **Terminé**.

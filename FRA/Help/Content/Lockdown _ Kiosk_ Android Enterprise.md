Une configuration Verrouillage et mode Kiosque : Android Enterprise désactive certaines fonctions des périphériques Android Enterprise et crée une liste d'autorisations indiquant les applis auxquelles les utilisateurs ont accès en mode Kiosque.

Cette section traite des sujets suivants :

## [****](#)Paramètres de verrouillage

| Paramètres | Description |

 | -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

 | Nom | Saisissez un nom qui identifie cette configuration. |

 | Description | Saisissez une description qui précise l'objectif de cette configuration. |

 | Choisir le type de confinement | Sélectionnez le type de paramètres de confinement que vous souhaitez configurer :<br />Profil professionnel
* Périphériques professionnels gérés (Propriétaire du périphérique avec mode kiosque)
* Paramètres de verrouillage des appareils gérés avec profil professionnel/profil professionnel sur les appareils appartenant à l'entreprise
 L'option « Profil professionnel sur les appareils appartenant à l'entreprise » (paramètres de verrouillage) est applicable uniquement aux appareils Android 11.<br />Un seul type est autorisé par configuration. Les options affichées dépendent du type sélectionné.<br />Si des configurations Périphérique géré (Propriétaire du périphérique) et Périphérique géré avec profil professionnel sur un périphérique détenu par l'entreprise sont attribuées à un même périphérique, la configuration Périphérique géré avec profil professionnel prévaut.

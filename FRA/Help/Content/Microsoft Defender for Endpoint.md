La configuration Microsoft Defender pour point de terminaison (MDE), anciennement appelée Protection avancée contre les menaces Windows, permet à nos clients d'effectuer l'onboarding et l'offboarding de périphériques dans le service MDE.

**Procédure**

1. Accédez à **Configuration > + Ajouter**.
2. Sélectionnez la configuration **Microsoft Defender pour point de terminaison**.
3. Saisissez un nom pour la configuration.
4. Entrez  une description.
5. Dans la section Installation de la configuration, définissez les paramètres restants comme décrit dans le tableau suivant.


|  Paramètre                       |  Description                                                                                         |
|----------------------------------|------------------------------------------------------------------------------------------------------|
|**Intégration ou exclusion de Blob**|Collez le Blob d'onboarding ou d'offboarding depuis le site de Centre de sécurité Microsoft Defender pour point de terminaison.|

6. Cliquez sur **Suivant**.
7. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée

8. Cliquez sur **Terminé**.

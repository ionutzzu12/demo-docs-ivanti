Après avoir créé des attributs personnalisés, vous pouvez les affecter à une ou plusieurs applications. Chaque attribut est associé à une valeur que vous pouvez utiliser pour accomplir des tâches, par exemple créer des groupes d'applications. Pour en savoir plus sur la gestion des attributs, reportez-vous à « Attributs ».

## Création et affectation d'un attribut personnalisé pour une seule application

Vous pouvez affecter un attribut personnalisé à une seule application.

Procédure

1. Connectez-vous au portail d'administration.
2. Accédez à **Applis** > **Catalogue d'applis**.
3. Sélectionnez une application et cliquez sur **Attributs**.
4. Cliquez sur **+Ajouter nouveau**, puis sélectionnez une valeur dans le menu déroulant **Nom d'attribut**.
5. Spécifiez la valeur de l'attribut dans le champ **Valeur**.
6. Cliquez sur **Enregistrer**. L'attribut personnalisé est ajouté à l'application.


## Affectation d'un attribut personnalisé à plusieurs applications

Vous pouvez affecter des attributs personnalisés à une ou plusieurs applications. Lorsque vous choisissez plusieurs applications, l'attribut personnalisé est appliqué à toutes les versions de l'application concernée. Vous pouvez choisir une application spécifique, puis accéder à l'onglet Attributs et modifier les détails d'attribut personnalisé d'une version spécifique de cette application.

Procédure

1. Connectez-vous au portail d'administration.
2. Accédez à **Applis** > **Catalogue d'applis**.
3. Cochez les cases d'une ou plusieurs applications.
4. Cliquez sur **Actions**.
5. Sélectionnez **Affecter des attributs personnalisés**. L'assistant Affecter des attributs personnalisés aux applis apparaît.
6. Sélectionnez *l'une* des options suivantes :
   - Forcer l'affectation (remplacement) de tous les attributs même si des valeurs existantes sont trouvées. 
   - Remplacer uniquement si la valeur est vide et ignorer les attributs déjà définis.

7. Cochez les cases d'un ou plusieurs attributs.
8. Spécifiez leur valeur dans les champs Valeur (aucune valeur vide n'est admise).
9. Cliquez sur **Affecter**. L'attribut personnalisé est affecté à toutes les versions des applications sélectionnées.
10. (Facultatif) Pour modifier l'attribut personnalisé pour une seule version d'application, sélectionnez cette version dans la liste déroulante et cliquez sur Modifier. 


Vous pouvez utiliser les **attributs d'appli personnalisés** et leurs valeurs pour créer des rapports, et pour l'exportation au format CSV, dans la page **Détails du périphérique**.

**S'applique à** : tvOS 10.2 et les versions plus récentes compatibles.

Cette configuration activera le mode Affichage Salle de conférence sur l'Apple TV. Le mode Affichage Salle de conférence verrouille Apple TV dans ce mode afin d'empêcher d'autres types d'utilisation.

À partir de tvOS 10.2, vous pouvez configurer des périphériques Apple TV supervisés sur le mode Affichage Salle de conférence. Le mode Affichage Salle de conférence verrouille les périphériques Apple TV concernés sur un écran avec papier peint noir et télécharge un économiseur d'écran par défaut, sauf si une image d'arrière-plan et un économiseur d'écran sont définis manuellement dans les paramètres de périphérique Apple TV. Ce mode peut également afficher un message configuré dans la configuration Affichage Salle de conférence.

La configuration Affichage Salle de conférence peut être déployée automatiquement et les applis peuvent être installées lorsque les périphériques sont configurés dans ce mode. Lorsqu'un périphérique configuré sur le mode Affichage Salle de conférence est réinitialisé il redémarre automatiquement avec l'écran verrouillé sans afficher d'abord l'écran d'accueil.

Si un périphérique Apple TV fonctionne en mode Single App ou si le mode Single App est déployé avec la configuration Affichage Salle de conférence, celle-ci prévaut sur le mode Single App. De plus, si un périphérique Apple TV est connecté à un réseau via Ethernet, le mode Affichage Salle de conférence n'affiche pas automatiquement un réseau Wi-Fi pour le partage AirPlay. Cette instruction peut être affichée sur l'écran à l'aide du champ Message personnalisé de la configuration Affichage Salle de conférence.

## Création d'une configuration Affichage Salle de conférence

**Procédure**

1. Sélectionnez **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Saisissez **conférence** pour le champ de recherche, puis cliquez sur la configuration **Affichage Salle de conférence**.
4. Saisissez un nom et une description pour la configuration.
5. Indiquez le **Message personnalisé**. Il s'agit du message personnalisé affiché sur l'écran dans le mode Affichage Salle de conférence.
6. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
7. Cliquez sur **Terminé**.

Vous pouvez appliquer une configuration ChromeOS sur des périphériques spécifiques. Cependant, le périphérique doit faire partie d'un groupe d'utilisateurs LDAP. Lorsque vous tentez d'appliquer la configuration ChromeOS Blueprint, seule la section Groupes d'utilisateur est disponible. Elle répertorie tous les groupes d'utilisateurs LDAP. Vous pouvez choisir une ou plusieurs entrées dans la liste des groupes, et appliquer la configuration.

Les administrateurs peuvent configurer la stratégie de mise à jour logicielle d'un périphérique en définissant les règles de mise à jour logicielle macOS.

Applicable à : macOS 10.7+

1. Accédez à Configurations > +Ajouter.
2. Saisissez macOS dans le champ de recherche, puis cliquez sur la configuration Paramètres de mise à jour logicielle de macOS.
3. Saisissez le nom et la description de la configuration.
4. Sélectionnez les configurations requises dans les règles de mise à jour logicielle de macOS.
5. Cliquez sur Suivant.
6. Sélectionnez l'option Activer cette configuration.
7. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée.

8. Cliquez sur Terminé.


## [****](#)Règles de mise à jour logicielle de macOS

Les administrateurs peuvent procéder ainsi pour sélectionner des règles dans la liste :

Les utilisateurs ne peuvent pas modifier ces paramètres lorsque ces règles s'appliquent au périphérique.
- Autoriser l'installation de versions préliminaires.
- Effectuer automatiquement les actions suivantes : 
  - Rechercher les mises à jour
  - Télécharger les nouvelles mises à jour disponibles
  - Installer les mises à jour macOS
  - Installer les mises à jour d'applis à partir de l'app Store
  - Installer les fichiers de données système et les mises à jour de sécurité

- Restreindre l'installation des applis aux administrateurs.
- Option permettant d'ajouter l'URL du catalogue de mises à jour logicielles (non prise en charge sur macOS 11+).

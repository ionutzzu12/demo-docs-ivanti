Vous pouvez envoyer une requête aux appareils iOS, macOS et Android gérés par l'entreprise afin de récupérer les journaux de débogage et résoudre les problèmes rencontrés. La commande « Demander les journaux de débogage » de la page Appareils permet d'enregistrer les actions effectuées et le résultat (succès ou échec) d'un événement dans les journaux d'audit.

Cette fonctionnalité requiert les clients suivants :
- Les appareils iOS nécessitent Go 5.3.0 pour iOS ou une version ultérieure compatible. Pour les appareils migrés d'Ivanti EPMM vers Mobile\\@Work, cette fonctionnalité requiert Mobile\\@Work 12.2.0 pour iOS ou une version ultérieure compatible.
- les périphériques macOS requièrent Mobile@Work 1.5 pour macOS ou versions plus récentes compatibles.
- les périphériques professionnels gérés Android requièrent Go 65 pour Android ou une version plus récente prise en charge.


**Procédure**

1. Accédez à **Périphériques > Périphériques**.
2. Sélectionnez le périphérique, puis cliquez sur le lien du nom du périphérique pour accéder à la page Informations sur le périphérique.
3. Cliquez sur l'icône ![](More_icon.png).
4. Sélectionnez **Demander les journaux de débogage**, puis cliquez sur **OK**.


Une fois la requête envoyée et les journaux disponibles sur l'appareil, une notification est transmise à l'administrateur et affichée dans les journaux de l'appareil. Ces journaux peuvent également être téléchargés en cliquant sur le lien.

Les changements apportés dans cette page affectent l'icône et le texte qui s'affichent dans l'écran Accueil d'un utilisateur.

**Applicable à** : macOS

1. Dans l'écran **Éléments de marque pour l'accueil d'un utilisateur**, cliquez sur **Personnaliser** (en haut à droite).
2. Pour changer l'icône d'accueil d'un utilisateur, faites glisser le fichier d'icône vers la zone en pointillés, ou cliquez sur **Choisir un fichier** pour le sélectionner dans votre système de fichiers.
3. Modifiez le texte de marque dans le champ **Modifier le nom**.
4. Cliquez sur le champ **Modifier la couleur du texte** pour afficher une palette de couleurs où faire votre choix, ou entrez le numéro hexadécimal de la couleur de votre choix.
5. Cliquez sur **Enregistrer les modifications**.

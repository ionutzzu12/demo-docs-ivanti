Par défaut, les paramètres régionaux utilisateur sont ceux du locataire. Si nécessaire, vous pouvez modifier les paramètres régionaux d'un seul utilisateur.

**Procédure**

1. Accédez à **Utilisateurs**.
2. Cliquez sur le nom de l'utilisateur concerné.
3. Cliquez sur **Modifier**. La fenêtre **Authentification requise** s'affiche. 
4. **** Lorsque vous vous trompez plusieurs fois lors de la saisie du mot de passe et si la valeur définie pour « Seuil des tentatives de connexion » dans la section « Paramètre de complexité du mot de passe » est atteinte, le compte est verrouillé et vous êtes déconnecté de la session en cours.
5. Sous le champ Paramètres régionaux, cliquez sur **Modifier**.
6. Dans la fenêtre **Modifier les paramètres régionaux utilisateur**, sélectionnez la valeur voulue dans la liste déroulante **Définir les paramètres régionaux utilisateurs sur**.
7. Cliquez sur **Terminé**.
8. Cliquez sur **Enregistrer**.

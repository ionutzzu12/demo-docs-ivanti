 Cette section traite des sujets suivants :
- Pour un ou plusieurs utilisateurs depuis la page d'accueil Utilisateurs
- Pour un seul utilisateur depuis Détails de l'utilisateur


Vous pouvez générer des PIN d'inscription de périphériques pour les utilisateurs sans envoyer d'e-mail d'invitation, ce qui rationalise le processus d'inscription.

## [****](#)Pour un ou plusieurs utilisateurs depuis la page d'accueil Utilisateurs

1. Accédez à **Utilisateurs** > **Utilisateurs**.
2. Sélectionnez le ou les utilisateurs pour lesquels vous souhaitez générer un code PIN d'enregistrement. Vous pouvez sélectionner un ou plusieurs utilisateurs.
3. Cliquez sur le bouton **Actions**.
4. Sélectionnez **Générer un PIN** dans le menu déroulant.
5. Les PIN générés s'affichent dans la colonne **PIN D'INSCRIPTION**.
6. Pour afficher les PIN, cliquez sur **Afficher le PIN**.


## [****](#)Pour un seul utilisateur depuis Détails de l'utilisateur

1. Accédez à **Utilisateurs** > **Utilisateurs**.
2. Cliquez sur le nom de l'utilisateur pour lequel vous voulez générer un PIN d'inscription pour ouvrir la page Détails de l'utilisateur.
3. Dans la section Informations utilisateur, cliquez sur **Générer un PIN**.
4. Cliquez sur **Afficher le code PIN** pour afficher le code PIN généré.
 Assurez-vous de disposer des autorisations appropriées pour générer des codes PIN d'enregistrement pour les utilisateurs. L'option « Générer un code PIN » est activée uniquement pour les utilisateurs dont le statut d'enregistrement de l'appareil est « Code PIN par mot de passe ».

Pour déverrouiller un périphérique :

Vous pouvez déverrouiller l'écran de votre appareil. La procédure de déverrouillage varie légèrement d'un appareil à l'autre.

**Procédure**

1. Accédez à **Périphériques > Périphériques**.
2. Sélectionnez les périphériques concernés.
3. Cliquez sur **Actions**.
4. Sélectionnez **Déverrouiller**.
5. Vous pouvez aussi cliquer sur le lien portant le nom du périphérique pour accéder à la page Détails du périphérique, cliquer sur l'icône **Déverrouiller** ![](Unlock_device_icon.png), puis cliquer sur **OK**.


## [****](#)Déverrouiller les périphériques Android

Lorsqu'une commande de déverrouillage est reçue, l'application Android tente de réinitialiser le code d'accès. En modes Administrateur et Propriétaire de l'appareil, le code de déverrouillage est réinitialisé, tandis qu'en modes Propriétaire du profil et Personnel d'entreprise activé, c'est le code de déverrouillage du profil professionnel qui est réinitialisé.

****************
- Par défaut
- (Android uniquement) Entrer un code de déverrouillage - Vous pouvez indiquer un code de déverrouillage dans cette section et cliquer sur **Déverrouiller**. 
  - Ignorer l'invite de réinitialisation du code d'accès sur le périphérique - Sélectionnez cette option pour ignorer la réinitialisation du code d'accès si la configuration de code d'accès est transmise en mode Push à un périphérique. 



Après avoir défini le code de déverrouillage, l'administrateur le trouve dans la page **Historique d'audit** et dans la page **Journaux du périphérique**. 

Le déverrouillage Directboot fonctionne uniquement avec l'option de déverrouillage par défaut, le code de déverrouillage « 0000 ». Les codes de déverrouillage personnalisés pour le démarrage direct ne sont pas pris en charge.

## [****](#)Déverrouiller AppConnect pour les applis Android

Pour les applis AppConnect, la commande **Déverrouiller AppConnect** permet de déverrouiller des conteneurs verrouillés suite à plusieurs tentatives de connexion d'utilisateurs à l'aide de codes d'accès erronés.

Une configuration Exchange permet de définir une messagerie ActiveSync sur les périphériques Android, iOS et visionOS, ainsi qu'une messagerie Exchange Web Services (EWS) sur les périphériques MacOS.

La configuration Exchange est obsolète chez Samsung sous Android 9. Sur les appareils Samsung fonctionnant sous Android 9 et versions ultérieures, la configuration Exchange n'est pas prise en charge en mode d'administration de l'appareil.

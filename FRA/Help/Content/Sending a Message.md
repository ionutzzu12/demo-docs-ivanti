 Cette section traite des sujets suivants :
- Envoyer un message à des utilisateurs
- Envoyer un message à des périphériques


Vous pouvez envoyer un message à tout utilisateur connu. Les messages peuvent être des e-mails ou des notifications push. Seuls les utilisateurs dont les appareils sont enregistrés peuvent recevoir des notifications push.

**Prérequis**

- Pour les périphériques iOS, vérifiez que le client Go est bien installé.
- Pour les périphériques macOS, veillez à installer le client Mobile@Work.


## [****](#)Envoyer un message à des utilisateurs

1. Accédez à **Utilisateurs > Utilisateurs**.
2. Sélectionnez les utilisateurs auxquels vous voulez envoyer un message.
3. Cliquez sur **Actions** (en haut à droite).
4. Cliquez sur **Envoyer le message**.
5. Si vous ne souhaitez pas envoyer un e-mail, désélectionnez la case **Envoyer un message par e-mail**.
6. Si vous envoyez un e-mail, saisissez un objet et un message.
7. Si vous envoyez une notification push, cochez la case **Envoyer une notification push** et saisissez le message.
8. Cliquez sur **Envoyer**.


## [****](#)Envoyer un message à des périphériques

1. Accédez à **Périphériques > Périphériques**.
2. Sélectionnez les périphériques auxquels vous voulez envoyer un message.
3. Cliquez sur **Actions** (en haut à droite).
4. Cliquez sur **Envoyer le message**.
5. Vous pouvez également cliquer sur le lien comportant le nom du périphérique pour accéder à la page des détails de ce dernier, puis sur l'icône **Envoyer le message**![](Send_msg_icon.png).
6. Si vous ne souhaitez pas envoyer un e-mail, désélectionnez la case **Envoyer un message par e-mail**.
7. Si vous envoyez un e-mail, saisissez un objet et un message.
8. Si vous souhaitez envoyer une notification push, cochez la case **Envoyer une notification push** et saisissez le texte du message.
 Un message de notification push peut également inclure des URL auxquelles les utilisateurs peuvent accéder.
9. Cliquez sur **Envoyer**.

   Lorsqu'une notification push est envoyée à l'utilisateur, une icône de cloche apparaît dans la barre d'outils de son appareil. En appuyant sur cette icône, l'utilisateur peut consulter l'historique des notifications reçues et effectuer une action ou supprimer une notification.

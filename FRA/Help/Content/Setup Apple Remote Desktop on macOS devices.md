 Cette section traite des sujets suivants :
- Activation du bureau à distance Apple sur les périphériques macOS
- Désactivation du bureau à distance Apple sur les périphériques macOS


## [****](#)Activation du bureau à distance Apple sur les périphériques macOS

La fonctionnalité Bureau à distance d'Apple active le partage d'écran et permet de gérer les appareils à distance. Elle est disponible pour les appareils macOS 10.14.4 supervisés.

**Procédure**

1. Sous **Périphériques**, sélectionnez un ou plusieurs périphériques macOS supervisés. 
2. Cliquez sur **Actions** > **Activer le bureau à distance** pour les périphériques.
3. Cliquez sur **Activer le bureau à distance** pour confirmer.


## [****](#)Désactivation du bureau à distance Apple sur les périphériques macOS

Procédure

1. Sous **Périphériques**, sélectionnez un ou plusieurs périphériques macOS supervisés. 
2. Cliquez sur **Actions** >  **Désactiver le Bureau à distance** pour les appareils. Le partage d'écran est désactivé et vous ne pouvez plus gérer les appareils à distance.

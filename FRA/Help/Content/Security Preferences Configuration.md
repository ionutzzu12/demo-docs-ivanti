Les administrateurs peuvent gérer et contrôler les modifications apportées par les utilisateurs aux paramètres du pare-feu, aux messages de verrouillage et aux mots de passe sur le périphérique sur lequel il est possible de configurer les préférences de sécurité.

Applicable à : macOS 10.10+

1. Accédez à Configurations > +Ajouter.
2. Dans le champ de recherche, saisissez Sécurité, puis cliquez sur la configuration Security Preferences (Préférences relatives à la sécurité).
3. Saisissez le nom et la description de la configuration.
4. Sélectionnez les configurations requises : 
   - Désactiver les modifications apportées aux paramètres du pare-feu
   - Désactiver les modifications apportées au message de verrouillage
   - Désactiver les modifications apportées au mot de passe

5. Cliquez sur Suivant.
6. Sélectionnez l'option Activer cette configuration.
7. Sélectionnez l'une des options de canal suivantes pour appliquer la configuration :
   - Canal de périphérique (option la plus courante)
   - Canal d'utilisateur (utilisateur déjà enregistré)

8. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée.

9. Cliquez sur Terminé.

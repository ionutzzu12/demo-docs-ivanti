Une configuration APN détermine le nom du point d'accès cellulaire pour le périphérique. [****](./Cellular.md)Sous iOS 7, suivez les indications de l'article Configuration mobile.

## Paramètres APN

|  **Paramètre**       |  Action                                                                                                                 |
|----------------------|-------------------------------------------------------------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Nom du point d'accès|Saisissez le nom du point d'accès correspondant. Il est généralement défini par l'opérateur qui fournit le service.|
|Nom d'utilisateur du point d'accès|Entrez le nom d'un utilisateur doté d'autorisations pour ce point d'accès.*|
|Mot de passe du point d'accès|Saisissez le mot de passe correspondant au nom d'utilisateur indiqué.|
|Serveur Proxy et port|Saisissez l'adresse IP ou l'URL et le numéro de port du serveur proxy APN.|
|EnableXLAT464|Cochez la case pour activer l'APN (Nom de point d'accès). Cette option fournit des services IPv4 sur un réseau IPv6 uniquement.|

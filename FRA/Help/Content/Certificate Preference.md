**S'applique à** : macOS 10.12 ou versions plus récentes compatibles.

Identifiez un élément de Préférence de certificat dans le trousseau de l'utilisateur qui référence une charge de traitement Certificat incluse dans le même profil.

Cette configuration sert à lier un certificat à une adresse e-mail ou une URL. Une fois que vous avez lié un certificat à une adresse e-mail, l'appli de messagerie l'utilise pour ce compte. Si un certificat SSL d'un site Web n'est pas marqué De confiance, l'ajout d'une préférence de certificat évite l'affichage d'un avertissement par le navigateur lorsque vous essayez d'accéder au site Web.

## Création d'une configuration de préférence de certificat

**Procédure**

1. Sélectionnez **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Dans le champ de recherche, saisissez **préférence**, puis cliquez sur la configuration **Préférence de certificat**.
4. Nommez et décrivez la configuration.
5. Dans la section Installation de la configuration, dans le champ **Nom**, saisissez un identifiant d'e-mail ou un nom faisant l'objet d'une demande de préférence de certificat.
6. Dans le champ **UUID du certificat**, sélectionnez un certificat.
7. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
8. Cliquez sur **Terminé**.

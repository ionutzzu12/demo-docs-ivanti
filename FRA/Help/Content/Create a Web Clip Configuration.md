Un clip Web est un raccourci vers un site ou une page Web sur un périphérique iOS. Utilisez une configuration de clip Web pour créer des clips Web standard sur les périphériques. Vous pouvez ajouter une icône de clip Web à votre périphérique iOS pour lancer un site Web spécifique. Les clips Web vous aident à trouver rapidement des signets et à les utiliser sur l'écran d'accueil de vos périphériques. Vous pouvez également contrôler certains des paramètres de l'expérience d'affichage Mobile Safari pour le site concerné.

Une option de délégation avec distribution personnalisée est disponible pour cette configuration. **Pour en savoir plus, reportez-vous à la rubrique « Distribution de la configuration » à la section « Utilisation de configuration ».

**Procédure**

1. Connectez-vous au portail d'administration .
2. Cliquez sur **Configurations**.
3. Cliquez sur **+ Ajouter**.
4. Recherchez et sélectionnez la configuration Clip Web.
5. Configurez les paramètres disponibles sur cette page. Reportez-vous au tableau de la section **Paramètres de configuration Clip Web** pour obtenir des conseils concernant les valeurs.
6. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
7. Sélectionnez **Personnalisé**, puis **Périphériques/Groupes de périphériques**.
8. Cliquez sur **Terminé**.


## Paramètres de configuration Clip Web

Le tableau suivant répertorie les paramètres de configuration Clip Web :

|  **Paramètre**                         |  Action                                                                                                                                            |
|----------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
|**Nom**|Saisissez un nom qui identifie cette configuration.|
|**Description**|Saisissez une description qui précise l'objectif de cette configuration.|
|**Étiquette**|Entrez le texte à afficher sous le raccourci sur l'écran du périphérique.*|
|**URL**|Entrez l'URL à laquelle le clip Web accède.*|
|**Supprimable**|Cochez la case pour autoriser l'utilisateur du périphérique à supprimer le clip Web.|
|**Icône**|Faites glisser le fichier de l'icône vers la zone pointillée, ou cliquez sur **Choisir un fichier** pour sélectionner un fichier dans votre ordinateur.|
|**Icône précomposée**|Sélectionnez ce paramètre pour supprimer les effets spéciaux associés aux versions récentes de Safari.|
|**Plein écran**|Sélectionnez ce paramètre pour afficher le clip Web en mode plein écran plutôt que comme un contenu dans un navigateur.|
|**Ignorer la portée Manifest**|Sélectionnez cette option pour autoriser la navigation vers un site Web externe sans afficher le navigateur Safari. Cette option n'a aucun effet si vous ne sélectionnez pas Plein écran.|
|**Identifiant de lot de l'application cible**|Identifiant de lot qui spécifie l'application ouvrant l'URL. **Exemple** : com.google.chrome.ios|

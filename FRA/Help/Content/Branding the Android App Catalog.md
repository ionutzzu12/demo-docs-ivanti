Vous pouvez intégrer des éléments de votre marque dans la page du catalogue d'applis d'Android pour que vos utilisateurs accèdent à une interface qui leur est plus familière. Vous pouvez personnaliser les éléments suivants du catalogue d'applis Android :
- Logo du catalogue (PNG, 360 × 64)
- Nom du catalogue
- Couleur de la barre d'action
- Icône de raccourci
- Nom du raccourci 


## Intégrer des éléments de marque dans le catalogue d'applis Android

Procédure

1. Dans la fenêtre **Éléments de marque dans le catalogue d'applis Android**, cliquez sur **Personnaliser** (en haut à droite).
2. Pour changer le logo du catalogue d'applis, faites glisser le fichier comportant votre logo vers la zone pointillée, ou cliquez sur **Choisir un fichier** pour sélectionner un fichier dans votre ordinateur.
3. Cliquez sur le champ **Couleur de la barre d'action** pour ouvrir une palette de couleurs et sélectionner celle que vous préférez, ou entrer son code hexadécimal.
4. Saisissez un nouveau texte dans le champ **Nom du catalogue d'applis** pour modifier le libellé du catalogue.
5. Le nom du catalogue d'applications que vous saisissez s'applique à Android, iOS et macOS.
 Pour modifier l'icône du raccourci, faites glisser le fichier icône vers la zone pointillée, ou cliquez sur **Choisir un fichier** pour le sélectionner dans votre système de fichiers.
6. Saisissez un nouveau texte dans le champ **Nom du raccourci** pour modifier le libellé du raccourci de l'appli.
7. Cliquez sur **Enregistrer les modifications**.

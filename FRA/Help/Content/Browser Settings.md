Grâce à Paramètres du navigateur, vous pouvez configurer les paramètres et les restrictions pour Google Chrome, Mozilla Firefox, Microsoft Edge et Internet Explorer sur les périphériques Windows 10.

Cette fonctionnalité requiert Bridge. Consultez Bridge pour obtenir plus d'informations.

 Avant d'appliquer les paramètres du navigateur, vérifiez que les navigateurs sont installés sur le périphérique.

Pour configurer les paramètres du navigateur :

1. Accédez à **Configuration > + Ajouter**.
2. Sélectionnez la configuration **Paramètres du navigateur**.
3. Saisissez un nom pour la configuration.
4. Entrez  une description.
5. Dans la section Installation de la configuration, définissez les paramètres restants comme décrit dans le tableau suivant.

7. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée

8. Cliquez sur **Terminé**.

### E-mail entrant

| **Paramètres** | **Que faire** |
 | -------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
 | Serveur de messagerie et port | Votre fournisseur d'accès Internet (FAI) peut vous fournir cette adresse. |
 | Nom d'utilisateur | Saisissez le nom d'utilisateur permettant d'accéder au serveur de messagerie entrant. Il s'agit souvent de la même chose que votre adresse e-mail. Votre fournisseur d'accès Internet peut vous fournir le format. |
 | Type d'authentification | Sélectionnez le type d'authentification défini par le FAI. |
 | Mot de passe | Saisissez le mot de passe pour accéder au serveur de messagerie entrant. |
 | Utiliser SSL | Sélectionnez cette option pour utiliser uniquement le protocole SSL (Secure Socket Layer) pour les communications entre l'appareil et le serveur. |

### E-mail sortant

| **Paramètres** | **Que faire** |
 | ---------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | Serveur de messagerie et port | Votre fournisseur d'accès Internet (FAI) peut vous fournir cette adresse. |
 | Nom d'utilisateur | Saisissez le nom d'utilisateur permettant d'accéder au serveur de messagerie sortante. Il s'agit souvent de la même chose que votre adresse e-mail. Votre fournisseur d'accès Internet peut vous fournir le format. |
 | Type d'authentification | Sélectionnez le type d'authentification défini par le FAI. |
 | Mot de passe | Saisissez le mot de passe pour accéder au serveur de messagerie sortante. |
 | Mot de passe sortant identique au mot de passe entrant | Cochez cette case si l'authentification SMTP utilise le même mot de passe que POP/IMAP. |
 | Utilisation uniquement dans la messagerie | Cochez cette case si vous souhaitez que cette configuration soit utilisée uniquement par le client de messagerie. Les autres applications qui envoient des e-mails, y compris celles qui envoient du contenu via le client de messagerie natif, ne peuvent pas utiliser cette configuration.
 | Utiliser SSL | Sélectionnez cette option pour utiliser uniquement le protocole SSL (Secure Socket Layer) pour les communications entre l'appareil et le serveur. |

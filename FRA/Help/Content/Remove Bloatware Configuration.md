La fonction de suppression des logiciels préinstallés vous permet de sélectionner la liste des applications installées sur les appareils qui doivent être désinstallées de force. L'installation d'un pont est requise pour cette fonction. Consultez [**Bridge**](\<./Ivanti Bridge.md>) pour plus de détails.

Pour exécuter ou désinstaller des applis :

1. Dans l'onglet **Configuration**, cliquez sur  **+Ajouter**.
2. ********
3. Dans le champ **Nom**, saisissez un nom pour la configuration.
4. Cliquez sur le lien **+Ajouter une description** pour ajouter une description à la configuration. Ce champ est facultatif.
5. ****************
6. Dans les options avancées, configurez les options suivantes :


|  Option                                  |  Description                                                                   |
|------------------------------------------|--------------------------------------------------------------------------------|
|**Exécuter cette configuration toutes les**|Définissez l'intervalle (en minutes) auquel la configuration doit être exécutée.|
|**Exécuter lors de la connexion**|Cochez cette case pour exécuter la configuration au moment de la connexion.|
|**Supprimer le redémarrage forcé après la désinstallation**|Cochez cette case pour empêcher le redémarrage forcé après la désinstallation de l'appli.|

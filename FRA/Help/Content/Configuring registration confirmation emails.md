## Configuration et utilisation des e-mails de confirmation de l'enregistrement

 Les administrateurs peuvent configurer et envoyer automatiquement un e-mail aux utilisateurs qui ont finalisé le processus d'enregistrement. Cet e-mail peut contenir, par exemple, des instructions supplémentaires destinées aux utilisateurs qui se sont bien enregistrés. Les administrateurs peuvent activer l'envoi de cet e-mail lors de l'invitation de l'utilisateur.

Processus :
- **Configuration de la fonctionnalité :**
  - Configurez le modèle d'e-mail : Le modèle en anglais par défaut ressemble à celui-ci, mais vous pouvez l'adapter pour mieux répondre à vos besoins, en suivant les instructions « Personnalisation d'un modèle d'e-mail », sous « Intégration d'éléments de marque dans les modèles d'e-mail » :





    ![](Resources/Images/cloud67_emailtemplate001.png)
      - Turn on the registration confirmation email. See Configuring User Registration Confirmation Emails in User Settings.
    - <strong cat-type="astérisque fort">Using the feature:</strong>
    
      - Send the user the invitation to register, as described in Inviting Users. When the user successfully registers, Ivanti Neurons for MDM will send that user the registration confirmation email.

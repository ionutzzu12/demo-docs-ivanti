Vous pouvez activer le verrouillage de l'écran sur un appareil. Le fonctionnement du verrouillage varie légèrement d'un appareil à l'autre.

**Procédure**

1. Accédez à **Périphériques > Périphériques**.
2. Sélectionnez le périphérique concerné.
3. Cliquez sur **Actions**.
4. Sélectionnez **Verrouiller**.
5. Vous pouvez aussi cliquer sur le lien portant le nom du périphérique pour accéder à la page Détails du périphérique, puis cliquer sur l'icône **Verrouiller** ![](Lock_device_icon.png), puis sur **OK**.
6. Pour les applications Android AppConnect, la commande « Verrouiller » empêche l'utilisateur d'accéder au conteneur et verrouille également l'appareil. L'utilisateur peut se reconnecter à l'appareil et à l'application AppConnect en utilisant respectivement le code d'accès de l'appareil et le code d'accès AppConnect.
7. Sur les périphériques iOS 7, vous pouvez entrer un message d'accueil et un numéro de téléphone (facultatif).

    De la sorte, les utilisateurs savent pourquoi le périphérique a été verrouillé et quel numéro contacter pour le déverrouiller.
8. Sur les appareils macOS, l'utilisateur est invité à saisir un code PIN à 6 chiffres pour accéder à l'appareil. Pour verrouiller l'écran, l'utilisateur doit :
   1. Entrer le code PIN
   2. Sélectionner la case à cocher de confirmation de verrouillage du périphérique.
   3. Cliquer sur **Oui, envoyer la commande de verrouillage**.

9. Dans macOS, l'utilisateur a la possibilité d'ajouter un message et un numéro de téléphone sur l'écran de verrouillage lors de la configuration du code d'accès de verrouillage du périphérique.
10. ********


Autres méthodes de verrouillage d'un périphérique : 
- L'utilisateur d'un périphérique peut effectuer l'action de verrouillage à partir du portail en self-service. 
- Un administrateur peut effectuer l'action de verrouillage à partir du portail des administrateurs.

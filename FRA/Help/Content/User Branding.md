La personnalisation de l'interface utilisateur vous permet de modifier le processus d'enregistrement des appareils avec des noms et des logos que vos utilisateurs reconnaîtront. Vous pouvez personnaliser l'interface utilisateur de différentes manières :

- Définir un nom d'hôte personnalisé pour l'URL d'enregistrement
- Faire apparaître votre logo dans l'e-mail et sur l'écran d'enregistrement
- Afficher un favicon personnalisé pendant les étapes d'enregistrement


**Licence** : Gold

**Prérequis** :
- Choisissez le nom d'hôte que vous souhaitez utiliser dans votre URL personnalisée. Il doit respecter les exigences suivantes :
  - Ne contenir aucune espace
  - Ne contenir aucun caractère spécial

- Le fichier du logo doit répondre aux critères suivants :
  - Format PNG
  - 580 x 80 pixels

- Le fichier du favicon doit répondre aux critères suivants :
  - Format PNG
  - 64 x 64 pixels



**Procédure** :

1. Accédez à **Utilisateurs > Éléments de marque de l'utilisateur**.
2. Cliquez sur **Personnaliser** (en haut à droite).
3. Dans le champ **Nom d'hôte**, saisissez un mot court à utiliser comme nom d'hôte dans l'URL.
4. Cliquez sur **Vérifier la disponibilité** pour vérifier que le nom d'hôte saisi n'est pas déjà utilisé.
5. Si le nom d'hôte n'est pas disponible, saisissez-en un autre.
6. L'URL d'enregistrement ainsi créée apparaît sous **Aperçu de l'URL**.
7. Cliquez sur **Suivant**.
8. Sous **Logo**, cliquez sur **Choisir un fichier** pour télécharger le logo à utiliser dans l'e-mail et l'écran d'enregistrement.
9. Cliquez sur **Suivant**.
10. Sous **Favicon**, cliquez sur **Choisir un fichier** pour télécharger le favicon à afficher à la place du favicon Ivanti Neurons for MDM pendant les étapes d'inscription.
11. Cliquez sur **Terminé**.

Les paramètres de configuration Gestion des disques  définissent des stratégies pour la gestion efficace des données des périphériques de stockage réseau et externes.

Applicable à : macOS 15.0 supervisé, jusqu'à la version la plus récente prise en charge par .

**Procédure**

1. Accédez à **Configurations** > **+Ajouter**.
2. Recherchez et sélectionnez la configuration **Gestion des disques**.
3. Configurez les paramètres **Gestion des disques** en vous basant sur le tableau suivant :


| **Paramètres** | **Description** |
 | ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | Nom | Saisissez un nom qui identifie cette configuration. |
 | Description | Saisissez une description qui précise l'objectif de cette configuration. |
 | **Gestion des disques** | |
 | Stockage externe | Sélectionnez une des valeurs suivantes dans la liste déroulante :<br />****Autorisé : Autorise la lecture et l'écriture dans le stockage.
* **Lecture seule** : Autorise la lecture dans le stockage.
* **Interdit**: L'accès au stockage est interdit. |
 | Stockage réseau | Sélectionnez l'une des valeurs suivantes dans la liste déroulante :<br />****Autorisé : Autorise la lecture et l'écriture dans le stockage.
* **Lecture seule** : Autorise la lecture dans le stockage.
* **Interdit** : L'accès au stockage est interdit.


4. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
5. ****
6. Cliquez sur **Terminé**.

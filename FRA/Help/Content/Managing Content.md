 Cette section traite des sujets suivants :
- Distribuer du contenu hébergé
- Supprimer du contenu


Le contenu hébergé permet la distribution de contenu téléchargeable via des URL externes. Ces URL doivent mener exclusivement à des fichiers PDF, EPUB ou iBook téléchargeables et doivent impérativement posséder l'une de ces extensions.

La distribution des licences VPP pour les livres n'est pas prise en charge. Par conséquent, la distribution de livres Apple basée sur les ID de l'iTunes Store n'est pas prise en charge. 

Utilisez l'application Livres ou Pages sur votre appareil pour accéder au contenu diffusé par Ivanti Neurons pour MDM. Vous le trouverez dans la section Bibliothèque.

****

 L'aperçu du contenu n'est pas disponible pour ces formats.

Pour le contenu **PDF**, vous pouvez publier le document à l'aide de l'application iBook sur les périphériques avec iOS 8 et versions ultérieures.

## [****](#)Distribuer du contenu hébergé

Vous ne pouvez pas télécharger de nouveaux documents dans , mais vous pouvez fournir un chemin (une URL) indiquant l'emplacement où le contenu est hébergé et le distribuer aux groupes de périphériques. 

**Procédure**

1. Accédez à **Contenu > Contenu hébergé**.
2. Cliquez sur **+ Ajouter**.
3. Saisissez les informations suivantes :
   - Titre
   - Auteur
   - Catégorie
   - (Facultatif) Description

4. ****Dans le champ Hosted Content Path (Chemin du contenu hébergé), saisissez l'URL du fichier que vous souhaitez télécharger.
5. Cliquez sur **Suivant**.
6. Apportez les modifications nécessaires à la distribution.
7. Cliquez sur **Terminé**.


Pour modifier le contenu hébergé, vous devez supprimer le contenu précédent, puis ajouter et distribuer le nouveau contenu hébergé.

Pour modifier d'autres paramètres que l'URL : 

1. Accédez à **Contenu > Contenu hébergé**.
2. Cliquez sur le lien du document dans la colonne **Nom**.
3. Cliquez sur l'icône Modifier.
4. Effectuez les modifications nécessaires.
5. Cliquez sur **Suivant**.
6. Apportez les modifications nécessaires à la distribution.
7. Cliquez sur **Terminé**.


## [****](#)Supprimer du contenu

1. Cliquez sur le lien du document dans la colonne **Nom**.
2. Ouvrez **Actions > Supprimer ce document**.
3. Cochez la case pour confirmer.
4. ****Cliquez sur Supprimer document.


Lorsque vous supprimez un document :
- Il est supprimé du système.
- Il n'est plus disponible dans le catalogue de contenu.
- Il est supprimé des périphériques qui l'ont téléchargé.


Si vous ne pouvez pas effectuer de tâches sur la page **Contenu** , il se peut que vous ne disposiez pas des autorisations requises. Vous avez besoin du rôle suivant : [**rôle**](\<./User Roles.md>) :

- Gestion des applis et des contenus

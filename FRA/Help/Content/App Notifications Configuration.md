Indiquez comment les utilisateurs doivent recevoir les notifications des applis sélectionnées. 

**S'applique à :** Périphériques iOS 9.3 et versions ultérieures supervisés.

## Configurer des notifications d'appli

**Procédure**

1. Sélectionnez **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Saisissez notifications dans le champ de recherche, puis cliquez sur la configuration **Notifications d'appli**. La page Installation de la configuration des notifications d'appli s'affiche.
4. Nommez et décrivez la configuration.
5. Ajoutez les applis en recherchant dans l'App Store ou en saisissant l'ID de paquet.
6. Choisissez une appli à laquelle appliquer les paramètres de notification. 
7. Configurez les paramètres de notification. Voici les notifications disponibles :
   - Autoriser les notifications
     - Afficher le centre de notifications
     - Sons
     - Icône de la version à badge de l'appli
     - Afficher sur l'écran de verrouillage
     - (Périphériques iOS 12.0 et versions ultérieures supervisés) Afficher l'alerte critique quand CarPlay est utilisé
     - (Périphériques iOS 12.0 et versions ultérieures supervisés) Autoriser l'activation des alertes critiques (ignorer « Ne pas déranger »)

   - Style de l'alerte de déverrouillage
     - Bannières
     - Alerte modale
     - Aucun

   - (Périphériques iOS 12.0 et versions ultérieures supervisés) Type de groupement
     - Automatique
     - Par appli
     - Désactivé

   - (iOS 14.0+) Type d'aperçu de notification : sélectionnez un type d'aperçu à afficher dans les aperçus de messages de notification du périphérique.
     - Contrôlé par l'utilisateur : affiche les aperçus de message en fonction des paramètres utilisateur des applis figurant sur le périphérique.
     - Toujours : affiche les aperçus des messages.
     - Si déverrouillé : affiche les aperçus de messages uniquement lorsqu'un périphérique est déverrouillé.
     - Jamais : empêche les applis d'afficher des aperçus de messages dans Notifications.


8. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
9. Cliquez sur **Terminé**.

Le navigateur ADMX (GPO) permet de voir les paramètres GPO, organisés sur la base des objets ADMX des locataires. Vous pouvez rechercher et consulter les objets ADMX par défaut et ajouter (télécharger) les fichiers ADMX personnalisés qui fournissent une structure XML pour définir l'affichage des paramètres GPO.

Pour télécharger un objet ADMX personnalisé :

1. Sélectionnez **Administration > Navigateur ADMX (GPO)**. La page **Navigateur ADMX (GPO)** s'affiche.
2. Cliquez sur **Joindre**. La fenêtre **Joindre des objets ADMX (GPO) personnalisés** s'affiche.
3. Cliquez sur **Choisir un fichier** pour sélectionner le fichier ADMX à télécharger.
 Assurez-vous que le fichier ADMX est un fichier zip.
4. Cliquez sur **Joindre**. Un message de confirmation s'affiche une fois le téléchargement réussi.
 Assurez-vous que le dossier racine contienne le fichier avec l'extension *.admx* après avoir décompressé le fichier zip.


## Recherche de paramètres GPO

Dans le navigateur ADMX (GPO), pour chercher et sélectionner un objet de stratégie de groupe (GPO), cliquez sur le composant approprié dans l'arborescence des objets GPO dans le volet de gauche. L'arborescence GPO montre le chemin des paramètres de stratégie. Pour chercher un paramètre GPO spécifique, saisissez le nom de l'objet GPO ou du fichier ADMX dans le champ Rechercher. Les détails du paramètre GPO sélectionné s'affichent dans le volet à droite.

Avant d'inviter une personne à enregistrer des appareils mobiles, vous devez créer un compte utilisateur pour cette personne. Vous devez également créer un compte utilisateur pour toute personne qui utilisera Ivanti Neurons for MDM afin de gérer les appareils ou de publier du contenu (administrateurs).

 Cette section traite des sujets suivants :
- Groupes d'utilisateurs
- Paramètres utilisateur
- Éléments de marque de l'utilisateur
- Inscription des utilisateurs avec Apple Business Manager
- Inscription d'utilisateurs basée sur un compte
- Licences utilisateur
- Gérer les utilisateurs

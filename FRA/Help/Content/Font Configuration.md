La configuration des polices vous permet de fournir des fichiers de polices TrueType ou OpenType supplémentaires aux appareils iOS 7. Le tableau suivant décrit les paramètres de police :

|  **Paramètre**|  Action                                                                                                                                 |
|------------|-----------------------------------------------------------------------------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Télécharger les polices|Faites glisser le fichier de police dans la zone en pointillés, ou cliquez sur **Choisir un fichier** pour le sélectionner dans votre système. Les fichiers de police doivent être au format .otf ou .ttf.|

[****](./Configurations.md)Pour en savoir plus, consultez la rubrique Création d'une configuration.

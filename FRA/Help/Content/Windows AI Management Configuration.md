Permet aux administrateurs de gérer les paramètres et configurations de l'IA Windows. La gestion de l'IA Windows améliore la créativité et la productivité grâce à un assistant IA performant.

**S'applique à** : Windows 10+

**Procédure**

1. Accédez à **Configurations** > **+Ajouter**.
2. Recherchez et sélectionnez la configuration **Gestion IA Windows**.
3. Saisissez le **nom** et la **description** de la configuration.
4. Dans la section **Configuration** , sélectionnez la case à cocher **Désactiver Windows Copilot** pour désactiver Windows Copilot.
 Si vous ne cochez pas la case **Désactiver Windows Copilot** , la configuration remplacera la sélection de configuration précédente sur l'appareil pour activer la gestion de l'IA Windows.
5. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
6. Sélectionnez une option de distribution pour configurer **Gestion de l'IA Windows** . Pour plus d'informations sur la configuration des options de distribution, consultez la section Utilisation des configurations.
7. Cliquez sur **Terminé**.

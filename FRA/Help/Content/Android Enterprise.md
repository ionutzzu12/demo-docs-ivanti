****Une configuration Android Enterprise définit les options Android Enterprise activées pour les périphériques pris en charge. Vous pouvez créer des configurations spécifiques pour certains groupes de périphériques ou modifier la configuration par défaut. [****](https://www.google.com/work/android/features/devices.html)Pour obtenir la liste des périphériques prenant en charge Android Enterprise, consultez la page officielle Android.

## [****](#)Paramètres Android Enterprise

|  **Paramètre**                                    |  **Action**                                                                                                                                                                                                                                                         |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Désactiver la capture d'écran (Android 5.0+)|Sélectionnez cette option pour empêcher les périphériques d'utiliser la fonction de capture d'écran native.|
|Désactiver le contrôle des applis (Android 5.0+)|Sélectionnez cette option pour empêcher les utilisateurs de modifier les applis sous Paramètres ou Lanceurs.|
|Interdire la configuration d'identifiants (Android 5.0+)|Ce paramètre permet d'empêcher les utilisateurs de configurer des références d'authentification.|
|Interdire la copie d'un profil à l'autre (Android 5.0+)|Sélectionnez cette option pour empêcher les périphériques d'effectuer des copier-coller vers d'autres profils professionnels Android Enterprise.|
|Interdire la modification de comptes (Android 5.0+)|Sélectionnez cette option pour empêcher les utilisateurs d'ajouter et de supprimer des comptes.|
|Interdire l'envoi avec Beam (Android 5.0+)|Sélectionnez cette option pour empêcher les utilisateurs d'utiliser NFC pour transférer des données d'appli.|
|Interdire le partage de position (Android 5.0+)|Ce paramètre permet d'empêcher les sites Web et les applis d'inviter l'utilisateur à communiquer la position du périphérique.|
|Restreindre les méthodes de saisie (Android 5.0+)|Sélectionnez cette option pour restreindre les méthodes de saisie en indiquant la liste des noms de paquet autorisés. Si la liste d'autorisations ne contient aucun paquet, seules les méthodes de saisie système sont autorisées. Les méthodes de saisie ne sont pas restreintes uniquement pour les applis professionnelles, mais pour tout le périphérique.|
|Restreindre les services d'accessibilité (Android 5.0+)|Sélectionnez cette option pour restreindre les méthodes de saisie en indiquant la liste des noms de paquet autorisés. Si la liste d'autorisations ne contient aucun paquet, seuls les services d'accessibilité système sont autorisés. Les méthodes de saisie ne sont pas restreintes uniquement pour les applis professionnelles, mais pour tout le périphérique.|
|Interdire l'ID de l'appelant (Android 6.0 et versions ultérieures)|Détermine si l'ID de l'appelant provenant du profil professionnel s'affichera sur le périphérique lors des appels entrants.|

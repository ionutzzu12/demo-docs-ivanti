 autorise la charge de traitement de configuration Réseau privé cellulaire à fournir des informations de périphérique pour les déploiements sur réseau privé, notamment la situation géographique, les préférences de Wi-Fi et le type de déploiement réseau.

**Applicable à**

iOS 17.0 et supérieur, jusqu'à la dernière version publiée prise en charge par .

**Procédure**

1. Accédez à **Configurations** > **+Ajouter**.
2. Recherchez et sélectionnez la configuration **Réseau privé cellulaire**.
3. Configurez les paramètres **Réseau privé cellulaire** en vous basant sur le tableau suivant :


|  **Paramètre**            |  **Description**                                                                                                                                                                                   |
|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|**Nom**|Saisissez un nom qui identifie cette configuration.|
|**Description**|Saisissez une description qui précise l'objectif de cette configuration.|
|**Données cellulaires préférées**|Cochez la case pour activer les données cellulaires et les utiliser à la place du Wi-Fi.|
|**NR autonome**|Cochez cette case si votre réseau de données cellulaire est doté de la 5G NR autonome.|
|**Nom du jeu de données**|Spécifiez le nom de jeu de données qui identifie cette configuration.|
|**Numéro de version**|Spécifiez le numéro de version du jeu de données pour le suivi des mises à jour système.|
|**Zones GeoFence**|Spécifiez la latitude, la longitude et le rayon pour chaque ID géographique pour déployer un réseau privé à un emplacement géographique donné. <br /> Vous pouvez créer une liste contenant jusqu'à mille zones Geofence pour les réseaux privés.|

4. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
5. Sélectionnez l'une des options de distribution pour définir la configuration **Réseau privé cellulaire**. Pour en savoir plus la configuration des options de distribution, reportez-vous à « Utilisation de configurations ».
6. Cliquez sur **Terminé**.

Vous utilisez actuellement une version d'Ivanti Neurons for MDM optimisée pour les périphériques iOS. Cette section explique comment autoriser la gestion des périphériques non-iOS. Une fois cette procédure terminée, vous pourrez gérer les périphériques suivants :
- Android 5.0 ou versions plus récentes compatibles
- Windows 10+


L'inclusion de la gestion des périphériques non-iOS est une action irréversible.

Pour inclure les périphériques non-iOS :

1. Cliquez sur** Administration > Plateformes autorisées**.
2. Cliquez sur le bouton **Autoriser toutes les plateformes**.
3. Cochez **Je comprends que cette action est définitive.** pour indiquer que vous êtes conscient que cette action est irréversible.
4. Cliquez sur le bouton **Autoriser toutes les plateformes**.

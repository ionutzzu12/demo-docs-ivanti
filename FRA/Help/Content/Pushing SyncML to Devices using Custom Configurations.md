Vous pouvez créer vos propres fichiers de configuration SyncML (Synchronization Markup Language) ou les obtenir d'une source tierce, et y intégrer des fonctionnalités personnalisées en les ajoutant à une configuration personnalisée.

Plateformes prises en charge :
- Windows


**Périphériques pris en charge :**

- Windows 10+
- Microsoft HoloLens 2


**Procédure**

1. Accédez à **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Cliquez sur **Configuration personnalisée** pour afficher la page **Créer une configuration personnalisée**.
4. Saisissez un nom pour la configuration.
5. Cliquez sur l'icône Windows OS.
6. Glissez-déposez le fichier SyncML dans l'interface ou cliquez sur **Choisir un fichier** pour naviguer jusqu'au fichier à sélectionner pour le téléchargement sur l'appareil.
 Ivanti Neurons for MDM n'effectue aucun contrôle de validation sur le code du fichier.
7. Cliquez sur **Suivant**.

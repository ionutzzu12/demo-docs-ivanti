Le mode « Appareil géré avec profil professionnel sur appareil appartenant à l'entreprise » permet d'utiliser un appareil Android professionnel, dont les données personnelles sont séparées des autres. Ce mode prend en charge deux profils : le profil géré pour les applications professionnelles et le profil personnel pour l'utilisateur. Il est activé en distribuant une configuration « Appareil géré avec profil professionnel » à un appareil configuré en mode propriétaire.

Pour obtenir plus d'informations sur les paramètres de verrouillage de la configuration Périphérique géré avec Work Profile, consultez Verrouillage et mode kiosque : Android Enterprise.

le périphérique doit utiliser Android 8.0 ou versions ultérieures jusqu'à la dernière version publiée.

Les configurations des applis, le partage de widgets entre profils, les alias de certificat client et les certificats d'identification peuvent être appliqués dans le mode Périphérique géré avec profil professionnel. 

les périphériques Périphérique géré avec Work Profile acceptent les configurations suivantes :
- Code d'accès avancé
- Toujours sur VPN
- Certificat
- Certificat de clé publique
- Compte Google
- Code d'accès
- Restrictions des téléphones Samsung
- Threat Defense
- Wi-Fi
- Permissions d'exécution des applis par défaut
- Attestation SafetyNet
- Code d'accès
- Actions locales Threat Defense (Protection contre les menaces)

Vous pouvez accéder aux détails de chaque application depuis le catalogue. Sur la page de détails, vous trouverez des informations telles que la version affichée (par exemple, 1.5.0), la version du bundle (par exemple, 1.5.0.42) et la version minimale d'Android requise (par exemple, 5.0 pour Android).

[****](./Dashboard.md)

**Procédure**

1. Cliquez sur **Applis**.
2. Cliquez sur **Catalogue d'applis**.
3. Sélectionnez l'appli.


La fenêtre Détails de l'appli s'affiche. 

Pour les applis iOS internes, vous pouvez vérifier la **date d'expiration du profil de provisioning** dans la page des détails de l'appli.

Pour les périphériques Windows, si l'appli est configurée pour installation silencieuse sous Configurations d'appli, la zone Récapitulatif des périphériques indique le nombre de périphériques éligibles, installés, en attente ou en échec pour l'appli.

## [****](#)Affichage des informations d'inventaire des applis depuis la page Détails de l'appli

Pour afficher les informations d'inventaire des applis, cliquez sur **Afficher l'inventaire des applis (toutes les versions)** pour afficher sous **Périphériques > Inventaire des applis** une liste filtrée par ID de lot concernant l'appli voulue.

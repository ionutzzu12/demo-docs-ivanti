 Cette section traite des sujets suivants :
- Pour les périphériques iOS
- Pour les périphériques Android


Si la configuration de confidentialité appliquée autorise la collecte des données de localisation, l'utilisateur du périphérique peut remplacer cette configuration.

## [****](#)Pour les périphériques iOS

Les utilisateurs de périphériques iOS peuvent désactiver les services de localisation pour empêcher l'envoi des données correspondantes au système de gestion des périphériques, avec le paramètre suivant :

Paramètres > Confidentialité > Services de localisation

## [****](#)Pour les périphériques Android

Les utilisateurs d'appareils Android peuvent désactiver la géolocalisation pour empêcher la collecte de données de localisation. L'emplacement de ce paramètre varie selon le fabricant. Les utilisateurs d'appareils Android sont également invités à accepter la demande d'accès à leurs données de localisation.

Configuration servant à changer les paramètres des applications par défaut sur les périphériques iOS et visionOS.

**Prérequis**

- Applicable uniquement aux versions iOS (18.2+) et visionOS 2.2.
- Ces applis sont configurées sur les périphériques iOS sous **Paramètres** - **Applis** - **Applis par défaut**.


**Procédure**

1. Dans la page **Périphériques**, inscrivez un périphérique iOS (18.2 +) ou visionOS 2.2.
2. Naviguez jusqu'à **Configurations** > **Ajouter une configuration** > **Paramètres d'application par défaut** pour définir le navigateur par défaut des périphériques iOS.
3. Cliquez sur **Configurations** > **Créer des paramètres** > **Créer une configuration Paramètres d'application par défaut**
4. Dans la section Détails de configuration, entrez le **nom** et la **description**.
5. Dans Paramètres de configuration > Ajouter une application > Ajouter par recherche, recherchez le navigateur souhaité. La liste des navigateurs s'affiche.
6. Sélectionnez dans la liste le navigateur voulu.
7. Dans la page **Créer une configuration Paramètres d'application par défaut**, cliquez sur **Personnalisé** pour distribuer la configuration à un utilisateur, groupe d'utilisateurs, périphérique ou groupe de périphériques spécifique.
8. Sous **Choisissez une de ces options de distribution personnalisée**, sélectionnez l'option voulue pour demander la distribution de cette configuration à un utilisateur, un groupe d'utilisateurs, un périphérique ou un groupe d'utilisateurs.
9. Sélectionnez l'option voulue sous **Tous disponibles** pour distribuer cette configuration.
10. Cliquez sur **Terminé**.

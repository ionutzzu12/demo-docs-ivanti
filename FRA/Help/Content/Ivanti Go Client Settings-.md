Configurez la collecte de données anonymes des utilisateurs finaux, notamment les informations relatives aux appareils et à leur utilisation, afin d'identifier les problèmes liés aux produits et de garantir un service de qualité. Ce paramètre permet également de configurer les options de contact du support.

**S'appliquent à :**

- Mobile@Work pour macOS 1.67 ou versions plus récentes compatibles.
- Go pour iOS 3.5.0 ou versions plus récentes compatibles.


## Création d'une configuration **Paramètres de client Ivanti Go**

**Procédure**

1. Accédez à **Configurations** > **Ajouter une configuration** > **Paramètres du client Ivanti Go**. La page **Créer une configuration des paramètres du client Ivanti Go** s'affiche.
2. Entrez un nom de configuration dans le champ **Nom**.
3. Entrez la **description** de la configuration.
4. ********
5. Sous **Collecte de données via Analytics pour les clients**, sélectionnez l'option **Activer Analytics** si la fonction a été désactivée. 
6. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
7. Cliquez sur **Terminé**.


## Définition de la configuration de l'onglet Support

**Procédure**

1. Accédez à **Configurations** > **Ajouter une configuration** > **Paramètres du client Ivanti Go**. La page **Créer une configuration des paramètres du client Ivanti Go** s'affiche.
2. Entrez un nom de configuration dans le champ **Nom**.
3. Entrez la **description** de la configuration.
4. Dans la section **Définition de la configuration de l'onglet Support**, cliquez sur **Activer l'onglet Support**.
5. Assurez-vous que la version du client Ivanti Go est 118.0.0 ou supérieure pour activer l'onglet Support.

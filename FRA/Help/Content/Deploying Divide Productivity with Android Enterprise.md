Divide Productivity est une appli PIM que vous pouvez déployer sur des périphériques Android Enterprise.

1. Accédez à **Applis > Catalogue d'applis**.
2. Sous **Applis professionnelles**, cliquez sur **Divide Productivity**.
3. Saisissez des catégories supplémentaires ou une description.
4. Cliquez sur **Suivant**.
5. Acceptez les permissions affichées.
6. Cliquez sur **Suivant**.
7. Sélectionnez une option de distribution.
8. Développez **Options avancées et configuration de l'appli**.
9. Utilisez les instructions suivantes pour activer les options :


|  Paramètre                              |  Action                                                                                                                                                                                                                                            |
|-----------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|Empêche l'utilisateur de désinstaller l'appli.|Sélectionnez cette option pour empêcher l'utilisateur final de désinstaller l'appli lorsque celle-ci a été installée en mode silencieux.|
|Adresse e-mail|Utilisez des variables pour définir l'adresse e-mail à associer à l'appli.|
|Mot de passe|Utilisez cette variable pour définir le mot de passe du compte de messagerie. Si vous laissez ce champ vide, l'utilisateur sera invité à saisir son mot de passe.|
|Hôte|Saisissez le nom d'hôte du serveur de messagerie à utiliser. Saisissez le nom de domaine complet du serveur ActiveSync. Si vous utilisez un serveur Sentry autonome, saisissez son nom de domaine complet (FQDN).<br />Exemple :<br />monSentry.monentreprise.com|
|Type de serveur|Sélectionnez le type de serveur de messagerie.|
|Nom d'utilisateur|Utilisez des variables pour définir le nom d'utilisateur du compte de messagerie.|
|SSL obligatoire|Sélectionnez ce paramètre si vous voulez sécuriser les communications via https vers le serveur que vous avez indiqué dans le champ Hôte.|
|Faire confiance à tous les certificats|Sélectionnez cette option seulement si vous voulez que l'appli accepte automatiquement les certificats non marqués De confiance.<br />Généralement, vous sélectionnez cette option seulement lorsque vous travaillez dans un environnement de test.|
|Signature d'e-mail par défaut|Saisissez la signature par défaut de tous les e-mails. <br />L'utilisateur final peut modifier ce paramètre à tout moment. Une fois modifié par l'utilisateur de l'appareil, les modifications ultérieures apportées à ce champ seront sans effet.|
|Email Max Attachment Size (Taille maximale des pièces jointes)|Saisissez la taille maximale autorisée pour les fichiers joints.|
|Enable Task (Activer la tâche)|Sélectionnez cette option pour synchroniser les tâches.|
|Login Certificate Alias (Alias du certificat de connexion)|Saisissez l'alias du certificat de connexion.|
|Smime Signing Certificate Alas (Alias du certificat de signature Smime)|Non pris en charge actuellement.|
|Smime Encryption Certificate Alias (Alias du certificat de cryptage Smime)|Non pris en charge actuellement.|
|Options avancées||
|Installer sur le périphérique|Sélectionnez cette option pour inviter l'utilisateur à installer l'appli.|
|Installer en arrière-plan sur les périphériques Samsung Knox|Sélectionnez cette option pour installer automatiquement l'appli sur des périphériques Samsung Knox.|
|Ne pas afficher l'appli dans le catalogue d'applis de l'utilisateur|Sélectionnez cette option si vous ne voulez pas que l'utilisateur puisse voir l'appli dans le catalogue d'applis sur le périphérique.|

10. Sélectionnez une option de promotion.
11. Cliquez sur **Terminé**.

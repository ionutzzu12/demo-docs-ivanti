Effacer les données d'un périphérique consiste à supprimer toutes les données du périphérique et restaurer les paramètres d'usine.

**Procédure**

1. Accédez à **Périphériques > Périphériques**.
2. Sélectionnez le périphérique concerné.
3. Cliquez sur **Actions** (en haut à droite).
4. ****Sélectionnez Réinitialiser.
5. ![](More_icon.png)********Vous pouvez aussi cliquer sur le lien de nom de périphérique pour accéder à la page Informations sur le périphérique, puis sur l'icône . 
6. Par défaut, l'option **Conserver le forfait de données** est sélectionnée.
 (Facultatif, applicable aux appareils iOS 11.3) Sélectionnez l'option **Ignorer la configuration de proximité** .
7. ********
8. Pour les appareils macOS, vous pouvez envoyer un code PIN à 6 chiffres comme code d'accès. Sur l'appareil, l'utilisateur est invité à saisir ce code PIN pour y accéder. Pour procéder à l'effacement des données, l'utilisateur doit :
   1. Entrer le code PIN
   2. Sélectionner la case à cocher pour confirmer l'effacement des données du périphérique
   3. Cliquer sur **Oui, effacer les données de ce périphérique**

9. Pour les périphériques ChromeOS, lorsque vous exécutez l'opération Réinitialiser depuis la page **Détails du périphérique** ou **Liste des périphériques**, un popup s'affiche, indiquant « un périphérique effacé revient à ses paramètres d'usine, ce qui peut se traduire par la perte de données sur le périphérique.    1. Cochez la case « Je comprends que la réinitialisation est définitive » pour confirmer l'opération.
   2. Cliquez sur **Réinitialiser** pour effacer le contenu du périphérique.

10. L'état du périphérique devient **Réinitialiser envoyé** et cet état mis à jour est visible après la synchronisation périodique du périphérique.


Sur les périphériques Android Enterprise, vous pouvez réaliser l'action **Réinitialiser** même si le périphérique est toujours verrouillé après redémarrage.

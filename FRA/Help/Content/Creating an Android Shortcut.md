Les raccourcis sont uniquement disponibles en mode Kiosque avec un navigateur autorisé. Ce navigateur doit être autorisé dans les paramètres de verrouillage et de mode Kiosque.

Les raccourcis apparaissent dans l'outil de lancement du kiosque .

**Procédure**

1. ********Accédez à Configurations > +Ajouter.
2. ******** Cliquez sur Raccourci Android pour afficher la page Créer une configuration de raccourci Android. 
3. **** Attribuez un nom à la configuration dans le champ Nom. 
4. **** Entrez une description de cette configuration dans le champ Description. 
5. **** Entrez un libellé unique pour le raccourci dans le champ Libellé. 
6. **** Indiquez l'URL de la cible du raccourci dans le champ URL. 
7. **** (Facultatif) Faites glisser un fichier vers le champ d'icône ou cliquez sur Choisir un fichier pour naviguer et choisir le fichier à utiliser comme icône du raccourci. 
8. Cliquez sur **Suivant**.

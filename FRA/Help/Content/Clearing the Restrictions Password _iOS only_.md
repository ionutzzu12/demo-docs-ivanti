Vous pouvez effacer le mot de passe des restrictions défini par les utilisateurs sur les périphériques iOS 8 gérés. Cette action est disponible uniquement pour les périphériques actifs.

**Procédure**

1. Accédez à Périphériques > Périphériques.
2. Recherchez le périphérique concerné.
3. Sélectionnez Actions > Effacer le mot de passe des restrictions.
4. Confirmez l'action lorsque vous y êtes invité.

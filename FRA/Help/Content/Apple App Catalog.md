**S'applique à :** iOS et macOS

Cette configuration gère les accès au catalogue d'applis Apple via un clip Web. Depuis Ivanti Neurons for MDM version 83, vous pouvez passer à l'expérience Apps@Work native depuis l'application Go. Pour les locataires nouvellement créés, la configuration Apps@Work Webclip n'est pas distribuée par défaut pour les périphériques iOS installés via iReg ou le client. L'administrateur doit distribuer manuellement la configuration Webclip aux périphériques enregistrés via iReg ou le client. 

Les résultats de recherche de webclip Apps@Work affichent seulement 10 applications sur les périphériques iOS et iPad, même si le paramètre Lignes de la recherche est supérieur à 10.

**Procédure**

Les administrateurs peuvent modifier la distribution de cette configuration définie par le système comme suit :

1. Accédez à **Configurations**.
2. Cliquez sur **Catalogue d'applis Apple**.
3. Cliquez sur **Modifier la distribution**.
4. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques : cette configuration sera envoyée à tous les périphériques compatibles.
   - Aucun périphérique : désactiver l'accès au catalogue d'applis Apple ou préparer cette configuration pour une distribution ultérieure.
   - Personnalisé : définir des groupes de périphériques spécifiques qui recevront cette configuration.

5. Cliquez sur **Enregistrer**.

L'administrateur peut lier tous les appareils Windows à des locataires grâce à la fonctionnalité TenantLockdown CSP. Pour utiliser cette fonctionnalité, les appareils doivent être inscrits via l'option Autopilot. Cette configuration peut être effectuée au niveau de l'appareil.

********

L'administrateur peut activer le fournisseur de services de configuration TenantLockdown en créant une configuration des restrictions de Windows et en sélectionnant l'option **Exiger des utilisateurs qu'ils se connectent au réseau pendant la configuration du périphérique (profil Autopilot requis)** sous **Autres restrictions**.

Pour supprimer un périphérique du fournisseur de services de configuration TenantLockdown, l'administrateur doit supprimer manuellement le périphérique du groupe ou modifier les restrictions.

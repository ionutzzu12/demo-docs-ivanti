La configuration du mode Single App permet de limiter l'utilisation d'un périphérique iOS à une appli spécifiée. Par exemple, vous pouvez vouloir configurer des périphériques de sorte qu'ils n'utilisent qu'une appli personnalisée que votre entreprise a développée.

Procédure

1. Accédez à **Configurations > Ajouter > Mode Single App**.
2. Suivez les instructions suivantes pour définir l'appli et les paramètres associés.


| **Paramètres** | Que faire |
 | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
 | Nom | Saisissez un nom qui identifie cette configuration.                                                                                                                                                                                                                                                                                                                                                                                                                                                           Saisissez une description qui précise l'objectif de cette configuration.Choisissez le mode de sélection de l'appli : <br />****Depuis le catalogue d'applis et les applis système : Sélectionnez cette option pour faire des recherches dans le catalogue d'applis et les applis système Ivanti Neurons for MDM (préinstallées sur les périphériques Apple par défaut).
*  Entrez le nom de l'appli et sélectionnez-le lorsqu'il apparaît dans la liste des applis. 
*   **Entrer un ID de lot :** Sélectionnez cette option pour entrer l'identifiant unique de l'appli système à sélectionner. Utilisez cette option si vous ne trouvez pas l'appli système avec l'option **Depuis le catalogue d'applis et les applis système**.|

 | Désactiver la fonction tactile | Sélectionnez cette option pour désactiver l'écran tactile.                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

 | Désactiver la rotation de l'appareil | Sélectionnez cette option pour désactiver la détection de la rotation de l'appareil.                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |

 | Désactiver les boutons de volume | Sélectionnez cette option pour désactiver les boutons de volume de l'appareil.                                                                                                                                                                                                                                                                                                                                                                                                                                                             |

 | Désactiver le commutateur de sonnerie | Sélectionnez cette option pour désactiver le commutateur de sonnerie de l'appareil.                                                                                                                                                                                                                                                                                                                                                                                                                                                              Sélectionnez ce paramètre pour désactiver le bouton de mise en veille/réveil du périphérique (en haut à droite sur la tranche du périphérique).Sélectionnez ce paramètre pour empêcher le périphérique de se mettre en veille après une période d'inactivité.Sélectionnez ce paramètre pour activer la lecture de l'écran VoiceOver (fonctionnalité d'accessibilité).Sélectionnez ce paramètre pour activer le zoom (fonctionnalité d'accessibilité).Sélectionnez ce paramètre pour activer le réglage de l'inversion des couleurs (fonctionnalité d'accessibilité).Sélectionnez ce paramètre pour activer AssistiveTouch (fonctionnalité d'accessibilité).Sélectionnez ce paramètre pour activer Énoncer la sélection (fonctionnalité d'accessibilité).Sélectionnez ce paramètre pour basculer entre le son stéréo et le son mono (fonctionnalité d'accessibilité).Sélectionnez ce paramètre pour autoriser les utilisateurs du périphérique à réaliser des réglages sur VoiceOver.Sélectionnez ce paramètre pour autoriser les utilisateurs du périphérique à réaliser des réglages sur le zoom.|

 | Réglages d'inversion des couleurs | Sélectionnez cette option pour autoriser les utilisateurs de l'appareil à inverser les couleurs.                                                                                                                                                                                                                                                                                                                                                                                                                                                             |

 | Réglages AssistiveTouch | Sélectionnez cette option pour autoriser les utilisateurs à effectuer des réglages AssistiveTouch.                                                                                                                                                                                                                                                                                                                                                                                                                                                  |


3. Cliquez sur **Suivant**.
4. Dans l'écran **Distribution**, sélectionnez les groupes de périphériques destinataires de cette configuration.
5. Cliquez sur **Terminé**.


Si vous avez configuré le clavier de numérotation du téléphone comme appli à utiliser, le bouton Accueil ne fonctionne qu'une seule fois lorsque le périphérique passe en mode Single App.

Les paramètres du Finder macOS gèrent les paramètres de l'appli Finder sous macOS.

Vous pouvez définir les options suivantes :

| Paramètres | Que faire |
 | ----------------------------------- | --------------------------------------------------------------------- |
 | Nom | Saisissez un nom qui identifie cette configuration. |
 | Description | Saisissez une description qui précise l'objectif de cette configuration. |
 | **Configuration** | |
 | **Paramètres** | **Que faire** |
 | Désactiver la prise en charge de la gravure de disques dans le Finder | - Activer
- Désactiver


## Distribution de la configuration

Procédure

1. Définissez les options à l'aide du tableau précédent.
2. Cliquez sur **Suivant**.
3. Sélectionnez l'option **Activer cette configuration**.
4. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée

5. Cliquez sur **Terminé**.

**Paramètres de nom de périphérique par défaut (pour iOS)**

|  **Paramètre**|  Action                                                                                                                                                                                                                       |
|-----------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Nom du périphérique|Entrez le format du nom de périphérique par défaut, avec les attributs de périphérique et LDAP disponibles.*<br />Si le nom du périphérique contient plus de 63 caractères, il sera tronqué afin d'assurer un affichage correct sur le périphérique.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|

**Licence :**

**S'applique à :** iOS 9.3+ supervisé

Configure la charge utile Apple Éducation et l'application Classroom pour les responsables et les membres. Le tableau suivant répertorie les paramètres Éducation :

| **Paramètres** | Que faire |
 | ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | Nom | Saisissez un nom qui identifie cette configuration. |
 | Description | Saisissez une description qui précise l'objectif de cette configuration. |
 | Type de configuration | Sélectionnez l'un des types suivants :<br />Leader
* Membre |

 | Activer cette configuration | - Sélectionnez cette option pour appliquer cette configuration aux appareils sélectionnés.
- Désélectionnez cette option pour supprimer cette configuration de tous les appareils si elle a été appliquée précédemment.

 | Diffuser | Sélectionnez l'une des options de diffusion suivantes :<br />Tous les périphériques 
- Aucun périphérique
- Personnalisée

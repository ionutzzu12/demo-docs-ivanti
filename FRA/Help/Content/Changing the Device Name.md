 Les administrateurs peuvent modifier manuellement le nom d'un périphérique (sans passer par la configuration Modifier le nom du périphérique).

**S'appliquent à :**

- Périphériques supervisés iOS
- Périphériques macOS 10.10 et versions ultérieures


**Procédure**

1. Accédez à **Périphériques**.
2. Recherchez le nom du périphérique dans la liste.
3. Effectuez l'une des étapes suivantes :
   - Si ce n'est pas déjà fait, ajoutez la colonne **Nom du périphérique** en cliquant sur l'icône Paramètres représentant une roue dentée à droite, puis sélectionnez **Nom du périphérique**.
   - Cliquez sur le lien dans la colonne **Nom** pour afficher la page de détails sur le périphérique.

4. **** En regard du nom du périphérique, cliquez sur l'icône de crayon Modifier.
5. Saisissez le nouveau nom du périphérique, puis cliquez sur l'icône représentant une coche.
6. Dans la boîte de dialogue Remplacer le nom du périphérique, vérifiez les notes et cliquez sur **OK**.


Le nom modifié est transféré en mode Push sur le périphérique lors de sa prochaine prise de contact. Cette action est irréversible.

 Si une configuration Nom du périphérique par défaut a déjà été configurée, cette action remplacera le nom défini par la configuration.

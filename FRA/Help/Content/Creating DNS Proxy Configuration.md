En tant qu'administrateur, vous pouvez configurer les paramètres du proxy DNS pour les utilisateurs d'iPhone et d'iPad. Vous pouvez utiliser la configuration du proxy DNS pour spécifier l'application fournissant l'extension réseau du proxy DNS, ainsi que d'autres valeurs propres au fournisseur.

**Procédure**

1. Connectez-vous au portail d'administration .
2. Accédez à **Configurations**
3. Entrez **DNS** dans le champ de recherche, puis cliquez sur **Configuration** **Proxy DNS**.
4. Saisissez un nom et une description pour la configuration.
5. Entrez les paramètres de configuration Proxy DNS suivants :
   - Identifiant de lot d'applis (Requis).
   - ID de lot du fournisseur.
   - Configuration du fournisseur (Clé-valeur).

6. Cliquez sur **Suivant**.
7. Sélectionnez l'option **Activer cette configuration**.
8. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée

9. Cliquez sur **Terminé**.

**S'applique à :** macOS 11.5 et versions ultérieures 

L'administrateur peut définir ou modifier le verrouillage de récupération pour le redémarrage des appareils macOS fonctionnant sous Apple Silicon. Ce verrouillage empêche le démarrage de l'appareil en mode de récupération tant que le code d'accès n'est pas saisi.

**Procédure** :

1. Accédez à **Périphériques**.
2. Pour définir ou modifier le verrouillage de récupération après le redémarrage :   1. Cliquez sur le nom d'utilisateur associé à l'appareil pour afficher la page de détails de l'appareil. Procédez ensuite comme suit :
   2. Dans la section **Aperçu** , développez **Verrouillage de récupération** et cliquez sur **Définir le mot de passe** ou cliquez sur **Modifier le mot de passe**. Ou cliquez sur les points de suspension **Actions** et cliquez sur **Définir/Modifier le verrouillage de récupération**.
   3. Dans la boîte de dialogue **Définir/modifier le verrouillage de récupération**, effectuez les opérations suivantes :       1. **Mot de passe actuel**: Saisissez votre mot de passe actuel ici. Laissez ce champ vide si vous le définissez pour la première fois.
      2. **Mot de passe** : saisissez le mot de passe que vous souhaitez définir.
      3. **Confirmer le mot de passe** : saisissez à nouveau le mot de passe que vous voulez définir.


3. Cliquez sur **Définir/modifier le verrouillage de récupération**.


Dans Vue d'ensemble, **Verrouillage de récupération activé** indique l'état du code d'accès du verrouillage de récupération.

Les administrateurs peuvent également effacer le code d'accès en supprimant ce dernier et en cliquant sur **Définir/modifier le verrouillage de récupération**.

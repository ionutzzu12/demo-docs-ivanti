**Applicable à :** Périphériques mobiles Windows 8 et 10

L'administrateur peut réinitialiser le code PIN d'un appareil mobile Windows. Un nouveau code PIN sera généré pour l'appareil. Cette fonctionnalité peut s'avérer utile, par exemple, lorsqu'un utilisateur quitte l'entreprise sans réinitialiser le code PIN de son appareil professionnel.

**Procédure**

1. Accédez à **Périphériques**.
2. Cliquez sur le nom de l'utilisateur auquel le périphérique est associé pour afficher la page des caractéristiques du périphérique.
3. Dans la section Général, cliquez sur **Réinitialiser**.
4. Dans la fenêtre Réinitialiser le code PIN, cochez la case pour confirmer l'opération. 
5. Cliquez sur **Oui, continuer**.

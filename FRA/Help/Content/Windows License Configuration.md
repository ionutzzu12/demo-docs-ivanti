La configuration de la licence Windows permet de mettre à niveau le système d'exploitation de l'appareil, par exemple de Windows 10 Professionnel vers Windows 10 Entreprise. De plus, cette configuration offre la possibilité d'activer ou de modifier la clé de produit des ordinateurs de bureau Windows 10.

Pour mettre à niveau une licence Windows :

1. Accédez à **Configuration > + Ajouter**.
2. Sélectionnez la configuration **Licence Windows**.
3. Saisissez un nom pour la configuration.
4. Dans la section **Installation de la configuration**, entrez la **clé de produit** Windows.
5. Cliquez sur **Suivant**.
6. Sélectionnez l'une des options de distribution suivantes :
   - Tous les périphériques
   - Aucun périphérique (par défaut)
   - Personnalisée

7. Cliquez sur **Terminé**.

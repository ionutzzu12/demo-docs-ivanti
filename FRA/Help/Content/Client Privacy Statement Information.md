****S'applique aux périphériques Android, Android Enterprise et iOS ou versions plus récentes compatibles.

Configuration pour diffuser les informations concernant la déclaration de confidentialité aux utilisateurs dans les clients Go. Vous pouvez modifier cette configuration définie par le système afin de configurer uniquement les paramètres de distribution.

L'utilisateur pourra voir les détails configurés dans les configurations suivantes :
- Protection de la vie privée
  -  Collecte des données de localisation
  - Collecte de l'inventaire des applis
  - Android 7.0+ :
    - Nom d'entreprise
    - Couleur de l'entreprise
    - Message court
    - Message long


- Confidentialité du client
  - SLC (Significant Location Change) : modification d'emplacement significative permettant de réveiller périodiquement le périphérique
  - Intervalle minimum des réveils basés sur l'emplacement
  - Activer l'état MixPanel

- Gestion des périphériques mobiles - Droits d'accès MDM (non applicable aux périphériques inscrits par l'utilisateur)
  - Verrouillage du périphérique et suppression du code d'accès
  - Effacement des données du périphérique
  - Informations réseau (téléphone/numéros SIM, adresses MAC)

**Procédure**

1. Connectez-vous à .
2. Accédez à **Configurations** > **Ajouter une configuration**.
3. Cliquez sur (Sélectionnez) **Fenêtre de connexion**.
4. Dans le champ **Nom**, entrez un nom pour la configuration Fenêtre de connexion.
5. Cliquez sur **+ Ajouter une description** pour ajouter des détails.
6. Dans la section **Définition de la configuration**, remplissez les champs suivants :
   - **Infos de l'hôte Admin** : Entrez le nom d'hôte, l'adresse IP ou la version du système.
   - ****
   - **Mot de passe de connexion auto** : Entrez le mot de passe de connexion automatique.
   - ****
   - **Texte de la fenêtre de connexion** : Entrez le texte de la fenêtre de connexion.
   - Cochez la case des options supplémentaires voulues, comme :
     - **Désactiver l'accès à la console**
     - **Désactiver la connexion auto FDE (FileVault Disk Encryption)**
     - **Désactiver les fonctions Verrouillage d'écran immédiates**
     - **Masquer les utilisateurs Admin lors de l'affichage d'une liste d'utilisateurs**
     - **Afficher uniquement les utilisateurs réseau et système**
     - **Masquer les comptes mobiles**
     - **Afficher les utilisateurs réseau**
     - **Désactive l'option de menu Déconnexion**
     - **Désactive Éteindre lorsque l'utilisateur est connecté**
     - **Désactive l'option de menu Redémarrer lorsque l'utilisateur est connecté**
     - **Désactiver l'option Redémarrer**
     - **Afficher la boîte de dialogue Nom et Mot de passe**
     - Afficher les autres utilisateurs - Géré
     - Afficher le menu d'entrée
     - Désactiver le bouton Arrêter
     - Désactiver le bouton Veille
     - Désactiver le bouton de menu Arrêter


7. Cliquez sur **Terminé**.

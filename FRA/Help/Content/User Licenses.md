Les licences par utilisateur définissent le nombre d'utilisateurs que vous pouvez enregistrer, le nombre d'appareils autorisés par licence, la quantité de contenu que vous pouvez configurer pour la distribution sur les appareils et les fonctionnalités disponibles. Si vous atteignez votre limite d'utilisateurs, un triangle rouge s'affiche sur la page d'administration. Si vous atteignez votre limite de contenu, le service vous empêchera d'en ajouter et affichera un message vous informant que vous avez atteint votre limite.

Pour déterminer le nombre de licences utilisateur à prévoir, considérez les points suivants :
- Chaque licence utilisateur achetée dans le cadre de l'offre Secure UEM ou Secure UEM Premium permet l'enregistrement de cinq périphériques.
- Dès qu'un utilisateur enregistre un sixième périphérique, une autre licence utilisateur est réclamée.
- Il n'existe pas de limite quant au nombre de licences qu'un utilisateur peut réclamer.
- Les licences sont libérées dès que les périphériques sont exclus ou lorsque leurs données sont effacées.


Par exemple, lorsque l'utilisatrice 1 enregistre son téléphone professionnel le premier jour de travail, elle obtient une licence utilisateur. La semaine suivante, elle enregistre deux téléphones personnels et une tablette avec cette même licence. Lorsqu'elle enregistre une autre tablette, elle possède désormais cinq appareils et obtient donc une deuxième licence utilisateur. Si son téléphone personnel est volé, elle efface toutes les données de l'appareil, ce qui libère la deuxième licence utilisateur.

L'administrateur ne recevra aucune notification par courriel ni via l'interface utilisateur concernant les détails des licences, tels que leur date d'expiration, le nombre total de licences disponibles ou le nombre de licences ayant dépassé la limite d'utilisation. De plus, les informations spécifiques relatives aux licences seront masquées dans la section « Compte ».

## Affichage du nombre de périphériques/licences d'un utilisateur

**Procédure** :

1. Accédez à **Utilisateurs**.
2. Cliquez sur le lien de l'utilisateur.


Le volet de gauche répertorie les données de l'utilisateur, dont l'utilisation de licences.

## Paramètres des domaines gérés

|  Paramètre          |  Action                                                             |
|---------------------|---------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Domaines de messagerie gérés|Cliquez sur **+Ajouter** pour entrer un domaine (monentreprise.com, par exemple).|
|Domaines Web gérés|Cliquez sur **+Ajouter** pour entrer un domaine (monentreprise.com, par exemple).|

[****](./Configurations.md)Pour en savoir plus, consultez la rubrique Création d'une configuration.

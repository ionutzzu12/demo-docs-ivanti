## [****](#)Mettre à niveau une licence

Les fonctionnalités de base sont incluses dans le package. Vous pouvez l'enrichir en :

- ajoutant des périphériques,
- mettant à niveau vers l'offre 
- mettant à niveau vers l'offre 
- mettant à niveau vers Platinum


Ces compléments permettent à votre solution mobile de ne plus être limitée à une configuration de base.

## [****](#)Demander une mise à niveau

Procédure

1. Dans le menu déroulant d'administration, sélectionnez **Options de mise à niveau**.
2. Cliquez sur **Demande de mise à niveau/Ajouter d'autres périphériques** (en haut à droite).
3. Sélectionnez les éléments que vous voulez ajouter et saisissez votre numéro de téléphone.


Un représentant vous contactera dans les 24 heures pour vous fournir davantage d'informations.

## [****](#)Mise à niveau à partir d'une version précédente

****
- Si l'option **Ignorer la connexion à AppleID et iCloud** est activée avant la mise à niveau, l'option **Ignorer la configuration Apple Pay** sera sélectionnée une fois la mise à niveau terminée.
- Si l'option **Ignorer la saisie du code d'accès** est activée avant la mise à niveau, les options **Ignorer la configuration Touch ID** et **Ignorer la configuration Apple Pay** seront sélectionnées à l'issue du processus.


Procédure

1. Après la mise à niveau, accédez de nouveau à la page **Modifier le profil Inscription des périphériques** afin de restaurer les paramètres souhaités.
2. Cliquez sur **Enregistrer**.


Plusieurs paramètres de configuration sont modifiés suite à la mise à niveau, 
- Les options de promotion sont désactivées.
 Les paramètres d'installation sont** définis sur** No**n.
 L**'option « Ne pas afficher dans le **catalogue d'applications » n'est plus **sélectionn**ée. Installation silencieuse sur Android Samsung **Knoxest défini sur Faux.
  


Les indicateurs de gestion iOS sont définis sur :
- Sauvegarder sur iCloud
- Supprimer lors d'une désinscription


 Ces paramètres d'indicateurs de gestion iOS peuvent être sélectionnés individuellement pour chaque appli.

Paramètres d'appli :
- Les paramètres d'appli s'appellent désormais « Configurations ».
- Tous les autres paramètres d'appli sont identiques à la précédente version.

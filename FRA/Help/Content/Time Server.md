**S'applique à** : macOS 10.12.4 et versions plus récentes compatibles.

Créer une configuration de serveur de temps permet aux périphériques de se connecter à des serveurs de temps personnalisés. 

## Créer une configuration de serveur de temps

**Procédure**

1. Sélectionnez **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Dans le champ de recherche, saisissez **temps**, puis cliquez sur la configuration **Serveur de temps**.
4. Saisissez un nom et une description pour la configuration.
5. Indiquez un **Serveur NTP**.
6. ****``
7. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
8. Cliquez sur **Terminé**.

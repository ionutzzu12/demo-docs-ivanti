Une configuration CalDAV définit l'accès à un calendrier Web à l'aide de la norme Internet CalDAV.

## Paramètres CalDAV

|  **Paramètre**  |  Action                                                                                                                                                                                                                                      |
|-----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Nom de l'hôte et port|Saisissez le nom de l'hôte et un port pour le serveur de calendriers.|
|URL principale|Saisissez l'URL permettant d'accéder aux services de calendriers.|
|Utilisateur|Saisissez le nom d'utilisateur permettant d'accéder au calendrier.|
|Mot de passe|Saisissez le mot de passe permettant d'accéder au calendrier.|
|Utiliser SSL|Sélectionnez cette option afin de n'utiliser que le protocole SSL (secure socket layer) pour les communications entre le périphérique et le serveur.|
|VPN par appli|**Prérequis** : Configurez Tunnel ou le VPN par appli avant de configurer le VPN par appli dans la configuration CalDAV.<br />Dans le menu déroulant, sélectionnez la configuration VPN par appli préconfigurée.<br />**S'applique à** : iOS 14+|

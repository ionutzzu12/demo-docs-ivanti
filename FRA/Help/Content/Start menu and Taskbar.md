Vous pouvez personnaliser la disposition du menu Démarrer pour permettre à vos utilisateurs de définir les applications autorisées et de supprimer celles qui ne sont pas nécessaires. Les versions Windows 10 et 11 prennent en charge des fonctionnalités différentes, car leurs dispositions de menu Démarrer diffèrent.

Pour définir la configuration Menu Démarrer et barre des tâches :

1. Accédez à **Configuration > + Ajouter**.
2. Sélectionnez la configuration **Menu Démarrer et barre des tâches Windows**.
3. Saisissez  un nom pour la configuration.
4. Sélectionnez la version de Windows appropriée.
5. ******** Dans la section Installation de la configuration, configurez les paramètres suivants :


|  Paramètre             |  Action                                                                                 |
|------------------------|-----------------------------------------------------------------------------------------|
|**Nom**|Saisissez un nom qui identifie cette configuration.|
|**Description**|Saisissez une description qui précise l'objectif de cette configuration.|
|**Sélectionner le périphérique final**|Sélectionnez le périphérique Windows 10 dont le menu Démarrer et la barre des tâches contiennent les applis provisionnées. |

9. **** Cliquez sur Récupérer la disposition du menu Démarrer sur le périphérique.
10. Après la récupération des paramètres depuis le périphérique de référence, toutes les applications configurées dans la section ÉPINGLÉES du menu Démarrer seront affichées pour que l'administrateur puisse les examiner.
 Cliquez sur **Suivant** et distribuez la configuration à tous les groupes d'appareils et d'utilisateurs concernés.
 En raison d'un problème fournisseur, lors de la non-distribution de la configuration, les applications épinglées conservent leur configuration d'origine. Cependant, lors de la distribution d'une nouvelle configuration, la disposition précédente est écrasée et la nouvelle est appliquée.

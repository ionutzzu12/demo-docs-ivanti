La configuration d'un serveur macOS définit un compte serveur macOS avec les types de comptes et les paramètres configurés. Cette configuration permet à l'utilisateur d'activer le partage de fichiers sur le serveur.

**S'applique à :** iOS 10+

## Configurer le serveur macOS

**Procédure**

1. Accédez à **Configurations > +Ajouter**.
2. Sélectionnez la configuration du **serveur macOS** pour afficher la page **Créer une configuration du serveur macOS**.
3. Saisissez un nom pour la configuration.
4. Entrez  une description.
5. Saisissez le **nom d'hôte** pour spécifier l'adresse du serveur.
6. Saisissez un **nom d'utilisateur** pour indiquer le nom de connexion.
7. (Facultatif) Saisissez le **mot de passe** de l'utilisateur.
8. (Facultatif) Saisissez la **description** du compte.
9. ****
10. Cliquez sur **Suivant**.
11. Sélectionnez  une distribution pour cette configuration.

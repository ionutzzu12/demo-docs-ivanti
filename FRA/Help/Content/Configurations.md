Les configurations sont des ensembles de paramètres que vous définissez pour les périphériques. Par exemple, vous pouvez utiliser des configurations pour définir automatiquement les paramètres du VPN et les exigences relatives aux codes d'accès sur les périphériques. Les configurations système existantes sont répertoriées dans la page Configurations.

 Cette section traite des sujets suivants :
- Utilisation des configurations
- Créer une configuration de portail utilisateur en self-service
- Configuration personnalisée
- Transfert en mode Push de SyncML sur les périphériques avec des configurations personnalisées
- Configuration de la disposition de l'écran d'accueil
- Configuration App Control : Contrôle des applis installées sur chaque périphérique
- Configuration des notifications d'appli
- Exportation des configurations
- Priorité des configurations
- Gérer les configurations

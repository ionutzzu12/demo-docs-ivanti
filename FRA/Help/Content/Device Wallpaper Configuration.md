## Licence :

La configuration du fond d'écran du périphérique définit une image de fond d'écran par défaut pour l'écran d'accueil et l'écran de verrouillage sur les périphériques Android 7.0 en mode Propriétaire du périphérique ou sur les périphériques COPE (Company Owned Personal Enabled - périphériques personnels utilisés pour le travail), à l'exclusion des périphériques Android 11 en mode EPO. 

## Paramètres de fond d'écran Android

Pour définir une image de fond d'écran par défaut pour les périphériques Android :

1. Accédez à **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Cliquez sur **Fonds d'écran des périphériques**.
4. Cliquez sur l'icône Android pour afficher la section Installation de la configuration pour Android et configurez les paramètres suivants :


|  **Paramètre**                                   |  Action                                                             |
|--------------------------------------------------|---------------------------------------------------------------------|
|**Nom**|Saisissez un nom qui identifie cette configuration.|
|**Description**|Saisissez une description qui précise l'objectif de cette configuration.|
|**Télécharger un fond d'écran Android**||
|Utiliser la même image pour l'écran d'accueil et l'écran de verrouillage|Sélectionnez ce paramètre pour télécharger la même image pour les deux écrans.|
|**Écran d'accueil**|Faites glisser le fichier d'image ou cliquez sur **Choisir un fichier** pour le sélectionner.|
|**Écran de verrouillage**|Faites glisser le fichier d'image ou cliquez sur **Choisir un fichier** pour le sélectionner.|

5. Cliquez sur **Suivant**.
6. Sélectionnez l'une des options de distribution suivantes :
   - **Tous les périphériques**
   - **Aucun périphérique** (par défaut)
   - **Personnalisé**

7. **** L'image téléchargée doit être au format .jpg ou .png.


## Paramètres de fond d'écran iOS

Vous pouvez définir une image de fond d'écran par défaut pour les périphériques iOS. 

Ce paramètre s'applique uniquement aux périphériques supervisés.

1. Accédez à **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Cliquez sur **Fonds d'écran des périphériques**.
4. Cliquez sur l'icône iOS pour afficher la section Installation de la configuration pour iOS et configurez les paramètres suivants :


|  **Paramètre**                                   |  Action                                                             |
|--------------------------------------------------|---------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Télécharger un fond d'écran iPhone||
|Utiliser la même image pour l'écran d'accueil et l'écran de verrouillage|Sélectionnez ce paramètre pour télécharger une seule image pour votre iPhone.|
|Écran d'accueil|Faites glisser le fichier d'image ou cliquez sur **Choisir un fichier** pour le sélectionner.|
|Écran de verrouillage|Faites glisser le fichier d'image ou cliquez sur **Choisir un fichier** pour le sélectionner.|
|Télécharger un fond d'écran iPad||
|Utiliser la même image pour l'écran d'accueil et l'écran de verrouillage|Sélectionnez ce paramètre pour télécharger une seule image pour votre iPad.|
|Écran d'accueil|Faites glisser le fichier d'image ou cliquez sur **Choisir un fichier** pour le sélectionner.|
|Écran de verrouillage|Faites glisser le fichier d'image ou cliquez sur **Choisir un fichier** pour le sélectionner.|

5. Cliquez sur **Suivant**.
6. Sélectionnez l'une des options suivantes :
   - **Tous les périphériques**
   - **Aucun périphérique** (par défaut)
   - **Personnalisé**

7. **** Les images téléchargées doivent être de 1 164 × 640 (H × L) et au format .jpg ou .png.


## Paramètres de fond d'écran macOS

Pour définir une image de fond d'écran par défaut pour les périphériques macOS :

1. Accédez à **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Cliquez sur **Fonds d'écran des périphériques**.
4. Cliquez sur l'icône macOS pour afficher la section Installation de la configuration pour macOS.
5. Entrez le chemin d'accès à l'image de bureau.
6. Cliquez sur **Suivant**.
7. Sélectionnez l'une des options suivantes :
   - **Tous les périphériques**
   - **Aucun périphérique** (par défaut)
   - **Personnalisé**

8. Cliquez sur **Terminé**.

Les administrateurs peuvent empêcher les utilisateurs de gérer les mises à jour système sur les appareils Android 6.0 ou versions ultérieures compatibles. Cette fonctionnalité est réservée aux appareils Android Enterprise.

 Pour procéder à la configuration :

1. Accédez à **Configuration ****> + Ajouter**.
2. Sélectionnez la configuration **Mise à jour du système**.
3. Saisissez un nom pour la configuration.
4. Entrez  une description.
5. Dans la section Installation de la configuration, configurez les options suivantes :

## Configuration Samsung e-FOTA (Obsolète)

La configuration E-FOTA de Samsung sera désactivée en juillet 2022. Par conséquent, elle ne sera plus disponible pour les nouveaux appareils. Les appareils disposant déjà de cette configuration pourront uniquement la désactiver.

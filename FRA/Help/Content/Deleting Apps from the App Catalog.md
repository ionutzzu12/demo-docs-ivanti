Vous pouvez supprimer les applications publiques et internes du catalogue d'applications. Les applications prérequises ne peuvent pas être supprimées. Vous devez modifier les applications pour supprimer les dépendances avant de les supprimer. Si l'application est installée sur des appareils, elle sera désinstallée lors de leur prochaine connexion. L'installation et la désinstallation silencieuses d'applications sont prises en charge sur les appareils Samsung et Zebra en mode Administrateur, et sur tous les appareils en mode Propriétaire.

****

Les conditions suivantes s'appliquent lorsque vous essayez de supprimer une ou plusieurs applis dans le catalogue d'applis :  
- Si l'une des versions de l'appli interne ne peut pas être supprimée, alors aucune version ne peut l'être.
- Si l'une des versions de l'appli interne est une appli prérequise, cette appli ne peut pas être supprimée, ni aucune des versions. 
- Quand vous sélectionnez certaines ou toutes les applis internes pour les supprimer du catalogue d'applis, toutes les versions des applis sélectionnées sont supprimées.
- Une appli interne déléguée depuis un espace ne peut pas être supprimée. 


**Procédure**

1. Accédez à **Applis > Catalogue d'applis**.
2. Cliquez sur le lien de l'appli.
3. Sélectionnez **Actions > Supprimer du catalogue**.
4. Prenez connaissance de l'avertissement expliquant ce qui se passe lorsque vous supprimez une appli.
5. L'avertissement explique que les licences d'applications et de livres (iOS) et les avis sur les applications (tous systèmes d'exploitation) sont également supprimés.

 Cochez la case « Je comprends les conséquences de la suppression d'une application » pour procéder à la suppression.
6. Cliquez sur **Supprimer l'appli**.

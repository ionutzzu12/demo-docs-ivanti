Permet la gestion des réglages Safari sur les appareils iOS et macOS supervisés. Les administrateurs peuvent ainsi gérer efficacement les extensions Safari pour la navigation standard et privée.

Applicable à :
- iOS 26.0+, iPad 26.0+, macOS 26.0+ et visionOS 26.0+, des versions bêta jusqu'à la dernière version publiée prise en charge par .


**Procédure**

1. Accédez à **Configurations** > **+Ajouter**.
2. Recherchez et sélectionnez la configuration **Extension et paramètres Safari**.
3. Cliquez sur **Paramètres Safari**. 
4. Configurez les **paramètres Safari** en vous basant sur le tableau suivant :


| **Paramètres** | **Description** |
 | ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | Nom | Saisissez un nom qui identifie cette configuration. |
 | Description | Saisissez une description qui précise l'objectif de cette configuration. |
 | Paramètres de configuration | Sélectionnez l'une des options suivantes pour définir le type de page de démarrage :<br />* **Démarrer**: Pour utiliser la page de démarrage par défaut. Safari désactive la page d'accueil existante.
* Page d'ac</strong>cueil **: Pour utiliser l'URL de la page d'accueil spécifiée. Safari la définit comme page d'accueil.
* <strong cat-type="strongasterisk">Extension**: Pour utiliser la page spécifiée par l'extension Safari dont l'identifiant est ExtensionIdentifier. Safari désactive la page d'accueil existante. |


5. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
6. ****
7. Cliquez sur **Terminé**.

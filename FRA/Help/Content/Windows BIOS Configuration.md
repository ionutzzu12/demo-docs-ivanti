Les administrateurs peuvent configurer les paramètres du BIOS Windows sur les appareils Lenovo. Au moins un appareil Lenovo compatible avec les paramètres du BIOS doit être enregistré pour effectuer cette configuration.

Pour procéder à la configuration :

1. Accédez à **Configuration ****> + Ajouter**.
2. Sélectionnez la configuration **BIOS Windows**.
3. Saisissez un nom pour la configuration.
4. Entrez  une description.
5. Sélectionnez le modèle de périphérique dans la liste déroulante prévue à cet effet.
6. Dans la section Configuration, configurez les options suivantes :

 La liste des paramètres varie en fonction des options disponibles pour le modèle d'appareil spécifique enregistré.

**S'appliquent à :**

- Périphériques macOS pris en charge. 
- Version 16.13.x ou versions ultérieures des applis Microsoft Office 365 recommandées.


permet de paramétrer les informations utilisateur et les options afin de définir la configuration initiale pour toutes les applications Microsoft Office 365.

Cette section traite des sujets suivants :

## [****](#)Créer une configuration Création d'un compte automatique Office 365

**Procédure**

1. Sélectionnez **Configurations**.
2. Cliquez sur **+ Ajouter**.
3. Dans le champ de recherche, saisissez **office**, puis cliquez sur la configuration **Création d'un compte automatique Office 365**.
4. Nommez et décrivez la configuration.
5. Saisissez les paramètres en suivant les indications du tableau des paramètres Création d'un compte automatique Office 365 ci-dessous.
6. Cliquez sur **Suivant** pour configurer les paramètres de distribution.
7. Cliquez sur **Terminé**.


## [****](#)Paramètres Création d'un compte automatique Office 365

| **Paramètres** | **Que faire** |
 | ------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
 | Adresse e-mail d'activation Office | Saisissez l'adresse e-mail de l'utilisateur. |
 | Connexion automatique Office | Sélectionnez cette option pour masquer les fenêtres de premier lancement. Seules les informations requises, telles que l'authentification O365, seront demandées à l'utilisateur.
 | Par défaut, ouvrir/enregistrer en local | Cocher cette option pour forcer l'affichage du panneau d'ouverture/enregistrement sur « Sur mon Mac » au lieu de « Emplacements en ligne ». |
 | Afficher les nouveautés au lancement | Sélectionnez cette option pour afficher les informations sur les nouvelles fonctionnalités au lancement. |
 | État d'exécution de la macro Visual Basic | Sélectionnez l'une des options suivantes :<br />Désactivé avec avertissements
* Désactivé sans avertissements
* Activé sans avertissement |

 | Désactiver les bibliothèques dynamiques externes Visual Basic | Sélectionnez cette option pour désactiver les dépendances externes de Visual Basic. |

 | Autoriser Visual Basic à se lier au système | Cochez cette option pour autoriser les macros à utiliser une déclaration DECLARE afin de se lier à l'API système system(). Cette API permet aux macros d'exécuter des processus externes arbitraires et de leur transmettre des données arbitraires sur la ligne de commande.

 | Désactiver Visual Basic pour la liaison à Popen | Cochez cette case pour autoriser les macros à utiliser une déclaration DECLARE pour se lier à l'API système popen(). Cette API permet aux macros d'exécuter des processus externes arbitraires et de leur transmettre des données arbitraires sur la ligne de commande.

 | Désactiver Visual Basic Mac Script | Sélectionnez cette option pour autoriser les macros à appeler l'API Visual Basic d'Apple Script. |

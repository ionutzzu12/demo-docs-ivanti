La configuration de la Protection des informations Windows (WIP) définit les paramètres WIP permettant de protéger les données de l'entreprise. Cette configuration peut être appliquée aux appareils inscrits sous gestion. Vous pouvez également consulter les détails WIP d'un appareil configuré sur sa page de présentation.

## Configurer la protection des informations Windows pour Windows

**Procédure**

1. Accédez à **Configuration > + Ajouter**.
2. **** Sélectionnez la configuration Protection des informations Windows. 
3. Saisissez  un nom pour la configuration.
4. Entrez  une description.
5. Dans la section Installation de la configuration, définissez les paramètres restants comme décrit dans le tableau suivant.
6. Cliquez sur **Suivant**.
7. Sélectionnez  une distribution pour cette configuration.

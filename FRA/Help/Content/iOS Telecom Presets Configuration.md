La configuration des réglages télécom prédéfinis pour iOS définit les valeurs par défaut pour les restrictions en matière d'itinérance et de zones d'accès.

## Paramètres des réglages télécom prédéfinis pour iOS

|  **Paramètre**                                 |  Action                                                                                            |
|------------------------------------------------|----------------------------------------------------------------------------------------------------|
|Nom|Saisissez un nom qui identifie cette configuration.|
|Description|Saisissez une description qui précise l'objectif de cette configuration.|
|Autoriser les périphériques à utiliser le service de transmission de la voix pendant l'itinérance|Sélectionnez cette option pour activer l'itinérance vocale. La disponibilité de l'itinérance vocale dépend de l'opérateur.|
|Autoriser les périphériques à utiliser le service de transmission des données pendant l'itinérance|Sélectionnez cette option pour activer l'itinérance des données. <br /> L'activation de l'itinérance des données active également l'itinérance voix sur le périphérique.|
|Autoriser les utilisateurs à activer les zones d'accès personnelles|Sélectionnez cette option pour activer la fonction de point d'accès personnel. La disponibilité de cette fonction dépend de l'opérateur.|

[****](./Configurations.md)Pour en savoir plus, consultez la rubrique Création d'une configuration.

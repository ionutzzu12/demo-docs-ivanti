**S'appliquent à :**

- Périphériques iOS 11.3 et versions ultérieures. 


Réinstallez les applis système iOS supprimées sur les périphériques iOS.

**Procédure**

1. [****](./Devices-.md)Vous pouvez également cliquer sur le nom du périphérique et réaliser cette procédure depuis la page Informations sur le périphérique.
2. Sélectionnez un ou plusieurs périphériques iOS.
3. Dans le menu **Actions**, cliquez sur **Réinstaller les applis système iOS**.
4. Dans la fenêtre Réinstaller les applis système iOS, sélectionnez une ou plusieurs applis système disponibles pour les installer sur les périphériques.
5. Cliquez sur **Réinstaller les applis**.


Les applications seront installées sur les appareils iOS compatibles sélectionnés lors de leur enregistrement. Les applications système installées de cette manière ne seront pas considérées comme des applications gérées. Si aucun appareil compatible n'est sélectionné, un message vous informera que les applications système ne seront pas installées sur ces appareils.

Pour plus d'informations, voir [**Software\\Updates**](\<./Software Updates.md>).

**S'appliquent à :**

- Périphériques iOS 9.0 et versions ultérieures supervisés enregistrés via Inscription des périphériques 
- Périphériques supervisés iOS 10.3 et versions ultérieures 


************

**Procédure**

1. Accédez au périphérique dans la page [**Périphériques**](./Devices-.md).
2. Cliquez sur le nom du périphérique pour afficher la page des détails de ce dernier.
3. Dans le menu **Actions**, cliquez sur **Mettre à jour la version de l'OS**.
4. Dans l'assistant **Mise à jour de la version du système d'exploitation** , vérifiez la version d'iOS et sélectionnez la version du système d'exploitation dans la liste déroulante **Mise à jour vers la version** .
 Si vous saisissez une version identique ou antérieure, un message d'erreur s'affiche indiquant que la version iOS cible doit être supérieure à la version actuelle.
5. Cliquez sur **Mettre à jour**.


L'appareil iOS sera mis à jour vers la dernière version d'iOS disponible lors de sa connexion. Si l'appareil est protégé par un code d'accès, après l'envoi de la mise à jour par le système MDM, l'appareil la met en file d'attente et l'utilisateur est invité à saisir son code d'accès pour lancer l'installation. Pour plus d'informations, consultez [**Software\\Updates**](\<./Software Updates.md>).

Disk Management configuration settings define the policies for effectively managing data in external and network storage devices.

Applicable to: macOS 15.0 Supervised through the most recently released version as supported by .

**Procedure**

1. Go to **Configurations** > **+Add**.
2. Search and select the **Disk Management** configuration.
3. Configure the **Disk Management** settings as per the following table:

| **Setting**         | **Description**                                                                                                                                                                                                                   |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name                | Enter a name that identifies this configuration.                                                                                                                                                                                  |
| Description         | Enter a description that clarifies the purpose of this configuration.                                                                                                                                                             |
| **Disk Management** |                                                                                                                                                                                                                                   |
| External Storage    | Select one of the following values from the drop-down list:<br />* **Allowed**: Allows reading and writing of the storage.
* **Read Only**: Allows to read the storage.
* **Disallowed**: It doesn’t allow access to the storage. |
| Network Storage     | Select one of the following values from the drop-down list:<br />* **Allowed**: Allows reading and writing of the storage.
* **Read Only**: Allows to read the storage.
* **Disallowed**: It doesn’t allow access to the storage. |

4. Click **Next** to configure the distribution settings.
5. Select one of the distribution options to set up the **Disk Management** configuration. For more information about configuring distribution options, see Working with Configurations.
6. Click **Done**.

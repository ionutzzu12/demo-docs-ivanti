Configuration to change the settings of default applications on iOS and visionOS devices.

**Prerequisites**

- Applicable only for iOS (18.2+) and visionOS 2.2 versions.
- These apps are set on iOS devices in **Settings** - **Apps** - **Default Apps**.

**Procedure**

1. On the **Devices** page, register an iOS (18.2 +) device or a visionOS 2.2 device.
2. Navigate to **Configurations** > **Add Configuration** > **Default Application Settings** to set default browser on iOS devices.
3. Click **Configurations** > **Create Settings** > **Create Default Application Settings Configurations**
4. Under the Configuration Details section, provide the **Name** and **Description**.
5. Under Configuration Settings > Add App > Add by Lookup, search for the desired browser. The list of browsers is displayed.
6. Select the desired browser from the list.
7. In the **Create Default Application Settings Configuration** page, click **Custom** to distribute the configuration to a specific device or device groups or users or user groups.
8. Under **Choose one of these Custom Distribution option**, select the desired option to specify distribution of this configuration to either User / User Groups or Device / Device Groups.
9. Select the desired option under **All Available** to distribute this configuration.
10. Click **Done**.

The changes made in this page affects the icon and text that appear on User On-boarding screen.

**Applicable to**: macOS

1. In the **User Onboarding Branding** screen, click **Customize** (upper right).
2. To change the user on-boarding icon, drag the icon file to the dotted box, or click **Choose File** to select it from your file system.
3. Change the branding name text from the **Edit Name** field.
4. Click the **Edit Text Colour** field to display a color palette to select from or enter the hex number for the color you prefer.
5. Click **Save Changes**.

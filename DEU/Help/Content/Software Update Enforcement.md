Applicable to:

- iOS 17.0 through the most recently released version as supported by .
- macOS 14.0 through the most recently released version as supported by .

**Procedure**

1. Go to **Configurations** > **+Add**.
2. Search and select the **Software Update Enforcement** configuration.
3. Configure the **Software Update Enforcement** settings as per the following table:

| **Setting**                      | **Description**                                                                                                                                                                                                                                  |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Choose to upgrade the OS version | Select one of the following options:<br />* **Latest Version**: Automatically updates to the latest version within a specified time frame.
* **Specific Version**: Updates to the specified OS and build versions within a specified time frame. |

4. Click **Next** to configure the distribution settings.
5. Select one of the distribution options to set up the **Software Update Enforcement** configuration. For more information about configuring distribution options, see Working with Configurations.
6. Click **Done**.

### Ivanti Neurons for MDM Left Navigation

The left navigation menu is enhanced to maintain consistent user experience across , **Neurons for Endpoint Management**, and **Neurons for IT Service Management**. Previously, when you mouse over on any of the menu items, you could view the submenu levels of the section (if a submenu existed) right over the content area. With the left navigation menu enhancements, you can click on any of the menu items, and the submenu items will appear below the main menu which gives a compact navigation experience. For example, when you click Dashboard, you will see the submenu items available on the Dashboard menu as depicted in the following image:

### Global Branding

Ivanti Neurons branding is updated to enhance the user experience across , **Neurons for Endpoint Management**, and **Neurons for IT Service Management**. You can customize the logo, change the color, and add different icons using the Branding section.

Prerequisite to access the Global Branding setting:

In addition to , you must have access to **Neurons for Endpoint Management** to use this setting.

By default, this setting is OFF, but can be enabled on request for tenants configured with **Neurons for Endpoint Management**.

To access the Branding setting, go to **Global Settings** > **Branding**.

**Reset to Default** – Use this option to reset the changed settings to default.

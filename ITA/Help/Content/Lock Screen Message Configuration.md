Displays a message and asset tag info on the login and lock screens. This is for supervised devices using Â iOS 9.3 or supported newer versions.

## Creating a Lock Screen Message configuration

Procedure

1. Select **Configurations**.
2. Click **+ Add**.

::Image[]{src="r34001.png" size="100" caption alt isUploading="false" sha initialPath="r34001.png" githubPath="Content/r34001.png" indent="2"}

3. Type **lock** in the search field, and then click the **Lock Screen Message** configuration:

::Image[]{src="r34002.png" size="100" caption alt isUploading="false" sha initialPath="r34002.png" githubPath="Content/r34002.png" indent="2"}

4. The Lock Screen Message Configuration details page appears.
   Configure the settings on this page. Refer to the table in the section [**Lock Screen Message Configuration Settings**](#Lock_Screen_Message_Configuration_settings) for guidance on the values.
5. Click **Next** to configure the distribution settings, and then click **Done**.

## [****](#)Lock Screen Message Configuration settings

| **Setting**           | What To Do                                                            |
| --------------------- | --------------------------------------------------------------------- |
| Name                  | Enter a name that identifies this configuration.                      |
| Description           | Enter a description that clarifies the purpose of this configuration. |
| Lock Screen Footnote  | This text appears on the login window and lock screen.                |
| Asset Tag Information | This text appears at the bottom of the login window and lock screen.  |

For more information, see [**How to create a configuration**](./Configurations.md).

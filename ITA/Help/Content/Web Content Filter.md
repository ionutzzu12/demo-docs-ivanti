A web content filter configuration limits web access for iOS 7+, macOS 10.15+, and visionOS 1.1+ devices.

## Web content filter settings

| **Setting**           | **What To Do**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name                  | Enter a name that identifies this configuration.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Description           | Enter a description that clarifies the purpose of this configuration.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Allowed websites      | **Limited Adult Content:** Select this option if you want to block access to web sites based on iOS automatic filters. These filters attempt, with a high degree of accuracy, to block websites with inappropriate content.<br />**Specific Web Sites Only:** Select this option if you want to manually list the accessible web sites.<br />**Plug-in (iOS8 Supervised and macOS 10.15+ Only):** Select this option to use a third-party plug-in.                                                                                                                                                             |
| Permitted URLs        | This option is available only if you selected Limit Adult Content.<br />Enter the permitted URLs. Each URL must begin with either:<br />* http\://
* https\://<br />If you want to permit both http\:// and https\:// for the same site, include two separate URLs.<br />All URLs for which the initial characters match the given permitted URL are accessible.<br />Example: http\://www\.someCompanySite.com permits access to the following:<br />* http\://www\.someCompanySite.com
* http\://www\.someCompanySite.com/jobs<br />These URLs are accessible even if the iOS automatic filters block them.  |
| Use Deny List URLs    | This option is available only if you selected Limit Adult Content.<br />Enter the Blockedlisted URLs. Each URL must begin with either:<br />* http\://
* https\://<br />If you want to block both http\:// and https\:// for the same site, include a row for each URL.<br />All URLs for which the initial characters match the given Blockedlisted URL are blocked.<br />Example: http\://www\.someCompanySite.com blocks access to the following:<br />* http\://www\.someCompanySite.com
* http\://www\.someCompanySite.com/jobs<br />These URLs are blocked even if the iOS automatic filters allow them. |
| Allowlisted bookmarks | This option is available only if you selected Specific Websites Only.<br />Optionally enter the folder into which the bookmark should be added in Safari.<br />Example:<br />/Sales/Products/<br />If absent, the bookmark is added to the default bookmarks directory.                                                                                                                                                                                                                                                                                                                                        |
| Filter Name           | This option is a available only if you selected Plug-in.<br /> <br />Enter text that will be displayed to identify this filter.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| Identifier            | This option is available only if you selected Plug-in.<br /> <br />Enter the bundle ID of the plug-in providing the filtering service.                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Service Address       | This option is available only if you selected Plug-in.<br /> <br />Optional: Enter any server address necessary for use by the plug-in. Consult the documentation for the plug-in to determine if this value is necessary.                                                                                                                                                                                                                                                                                                                                                                                     |
| Organization          | This option is available only if you selected Plug-in.<br /> <br />Optional: Enter any organization string required by the plug-in.  Consult the documentation for the plug-in to determine if this value is necessary.                                                                                                                                                                                                                                                                                                                                                                                        |
| Username              | This option is available only if you selected Plug-in.<br /> <br />Optional: Enter any username required by the plug-in service. Consult the documentation for the plug-in to determine if this value is necessary.                                                                                                                                                                                                                                                                                                                                                                                            |
| Password              | This option is available only if you selected Plug-in.<br /> <br />Optional: Enter any password required by the plug-in service. Consult the documentation for the plug-in to determine if this value is necessary.                                                                                                                                                                                                                                                                                                                                                                                            |
| Certificate           | This option is available only if you selected Plug-in.<br /> <br />Optional: Enter any certificate required by the plug-in service to authenticate the user. Consult the documentation for the plug-in to determine if this value is necessary.                                                                                                                                                                                                                                                                                                                                                                |
| Filter Webkit Traffic | This option is available only if you selected Plug-in.<br /> <br />Select to include Webkit traffic in the filter.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Filter Socket Traffic | This option is available only if you selected Plug-in.<br /> <br />Select to include socket traffic in the filter.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Custom Data           | This option is available only if you selected Plug-in.<br /> <br />Optional: Add any key/value pairs required by the plug-in service. Consult the documentation for the plug-in to determine if this value is necessary.                                                                                                                                                                                                                                                                                                                                                                                       |

For more information, see [**How to create a configuration**](./Configurations.md).

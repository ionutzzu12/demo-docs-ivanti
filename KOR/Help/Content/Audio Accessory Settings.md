The **Audio Accessory Settings** configuration allows you to configure audio accessory temporary pairing behavior on iOS and iPadOS devices.

**Applicable to**

- iOS and iPadOSÂ 26 through the most recently released version as supported by .

**Procedure**

1. Go to **Configurations** > **+Add**.
2. Search and select the **Audio Accessory Settings** configuration.
3. Configure the **Audio Accessory Settings** settings as per the following table:

| **Setting**                                        | **Description**                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| -------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Name**                                           | Enter a name that identifies this configuration.                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **Description**                                    | Enter a description that clarifies the purpose of this configuration.                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Disable temporary pairing of audio accessories** | Select the checkbox to disable the audio accessories from pairing to the device.<br />If thse checkbox is unselected, you can specify the unpairing policy of the device by selecting one of the following options from the drop-down list of the Specify the device's unpairing policy field:<br />* **None**
* **Hour**: Enter the number of hours to automatically unpair a temporarily paired audio accessory.<br />By default, the checkbox will be in unselected state. |

4. Click **Next** to configure the distribution settings.
5. Select one of the distribution options to set up the **Audio Accessory Settings** configuration. For more information about configuring distribution options, see Working with Configurations.
6. Click **Done**.

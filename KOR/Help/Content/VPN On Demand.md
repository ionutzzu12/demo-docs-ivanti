**Applicable to:** iOS devices

A VPN On Demand configuration sets up access to a VPN server based on domains, host names, etc.

## VPN On Demand settings

| **Setting**                                                            | What To Do                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| ---------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name                                                                   | Enter a name that identifies this configuration.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Description                                                            | Enter a description that clarifies the purpose of this configuration.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| Connection Type                                                        | Select the type of VPN to configure.<br />The remaining settings depend on this selection.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Enable VPN On Demand                                                   | Select to use this configuration for domains and host names that establish a VPN on demand.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| Enable iOS Rules<br />(Applicable if Enable VPN On Demand is selected) | For iOS and macOS, you can set up:<br />* Network rules that allow or disallow connections to, and allow or ignore,  the networks that evaluate as true.
* Connection rules allow when needed, or never allow, connections to the networks that evaluate as true.<br />For network rules, you can specify the following types of parameters:<br />* DNS Domain Match
* DNS Server Address Match
* SSID Match
* URL String Probe
* Interface Type Match<br />For connection rules, you can specify the following types of parameters:<br />* DNS Domain Match
* DNS Server Address Match
* SSID Match
* URL String Probe
* Interface Type Match
* Domain Name
* DNS Server
* URL Probe |
| Provider Type (iOS 9+)                                                 | Select one of the following tunnel provider:<br />* app-proxy - tunnels traffic at the app layer. See [**Apple documentation**](https://developer.apple.com/documentation/networkextension/app_proxy_provider) for an overview of App Proxy Provider.
* packet-tunnel - tunnels traffic at the IP layer. See [**Apple documentation**](https://developer.apple.com/documentation/networkextension/packet_tunnel_provider) for an overview of Packet Tunnel Provider.                                                                                                                                                                                                                  |
| EnforceRoutes                                                          | When enabled, the VPN's non-default routes overrides any locally defined routes.<br />* If IncludeAllNetworks is enabled, the system disregards the EnforceRoutes setting
* Available in iOS 14.2 and later.                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| ExcludeLocalNetworks                                                   | When enabled along with IncludeAllNetworks, all local network traffic routes outside of the VPN.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| IncludeAllNetworks                                                     | When enabled, it routes all traffic through the VPN.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |

The protocols and their settings are listed as follows:

### [****](#)IPsec (Cisco)

| **Setting**            | What To Do                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Server                 | Enter the IP address or host name for the VPN server.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Account                | Enter the user account to be used for authenticating the connection.\*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| Machine Authentication | Only Certificate authentication is supported.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Credential             | Select the identity certificate to use.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Include User PIN       | Select to prompt the user for a PIN.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Proxy Setup<br />      | Select **Manual** or **Automatic** to configure a proxy.<br />**If you select Manual,** then the following additional fields are available:<br />* **Server and Port**: Enter the network address and port number for the proxy server.\*
* **Authentication**: Enter a valid user name if one is required for connecting to the proxy.\*
* **Password**: Enter a valid password if one is required for connecting to the proxy.<br />**If you select Automatic,** then the following additional fields are available:<br />* **Proxy Server URL**: Enter the fully-qualified URL for the proxy. |

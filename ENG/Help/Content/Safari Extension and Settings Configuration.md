**Procedure**

1. Go to **Configurations** > **+Add**.
2. Search and select the **Safari Extension and Settings** configuration.
3. Click **Safari Settings**.
4. Configure the **Safari Settings** settings as per the following table:

| **Setting**            | **Description**                                                                                                                                                                                                                                                                                                                                                                                 |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name                   | Enter a name that identifies this configuration.                                                                                                                                                                                                                                                                                                                                                |
| Description            | Enter a description that clarifies the purpose of this configuration.                                                                                                                                                                                                                                                                                                                           |
| Configuration Settings | Select one of the following options to set the start page type:<br />* **Start**: To use the default start page. Safari disables the existing homepage.
* **Homepage**: To use the specified HomePage URL. Safari sets this as homepage.
* **Extension**: To use the page specified by Safari Extension for which the identifier is ExtensionIdentifier. Safari disables the existing homepage. |

5. Click **Next** to configure the distribution settings.
6. Select one of the distribution options to set up the **Safari Settings** configuration. For more information about configuring distribution options, see Working with Configurations.
7. Click **Done**.
